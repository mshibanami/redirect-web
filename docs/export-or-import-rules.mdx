# Export/Import Rules

## Export Rules {#export-rules}

### Export on macOS {#export-on-macos}

Click either the ellipsis button or the Share button as shown in this screenshot:

<Image src={ShareOnMacOS} alt="Share button on macOS" width="460" hasShadow hasRoundedCorners/>

Then, select "Export Rule" or "Share Rule..." from the context menu:

You can also multi-select your rules and export them by secondary clicking (right-clicking) them on the rule-listing sidebar:

<Image src={MultiSelectOnMacOS} alt="Share multiple rules on macOS" height="400" hasShadow hasRoundedCorners/>


In addition, you can export all the rules from the menubar > File > Export All Rules:

<Image src={MenubarFile} alt="Menubar > File" width="400" hasShadow hasRoundedCorners/>

> [!NOTE]
> If you want to export your rules in the format compatible with [Einar Egilsson's Redirector](https://einaregilsson.com/redirector/), press the Option key (`âŒ¥`) on the context menu or the menubar, then you'll see these:

> <br/>
> <Image src={ContextMenuRedirector} alt="Context Menu (Redirector)" width="400" hasShadow hasRoundedCorners/>
>
> :::warning
> Redirect Web is not fully compatible with Redirector, and vice versa. Some settings, such as "Replace Occurrences" for Capturing Group Processing, aren't exported with the Redirector's format.

## Import Rules {#import-rules}

### Import on macOS {#import-on-macos}

- Simply drag and drop the exported rule files to Redirect Web's window.
- Or, you can also import them from the menubar > File > Import Rules

### Import on iOS {#import-on-ios}

Suppose your rule file is in the Files app. Open it on Files, tap the Share button, and then select Redirect Web:

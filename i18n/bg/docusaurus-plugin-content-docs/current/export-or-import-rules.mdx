# Експорт/Импорт на правила

## Експорт на правила

### Експорт на macOS

Кликнете върху бутона с многоточие или бутона "Споделяне", както е показано на тази екранна снимка:

![Share on macOS](/img/share-on-macos.png)

Можете също така да изберете няколко правила и да ги експортирате, като кликнете с десния бутон (secondary clicking) върху тях в страничната лента със списъка с правила:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

В допълнение, можете да експортирате всички правила от лентата с менюта (menubar) > File (Файл) > Export All Rules (Експортиране на всички правила):

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Ако искате да експортирате правилата си във формат, съвместим с [Redirector на Einar Egilsson](https://einaregilsson.com/redirector/), натиснете клавиша Option (`⌥`) в контекстното меню или лентата с менюта, след което ще видите следното:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web не е напълно съвместим с Redirector и обратното. Някои настройки, като "Replace Occurrences" (Замяна на срещания) за "Capturing Group Processing" (Обработка на заснемащи групи), не се експортират във формата на Redirector.
> :::

## Импорт на правила

### Импорт на macOS

- Просто плъзнете и пуснете експортираните файлове с правила в прозореца на Redirect Web.
- Или можете също да ги импортирате от лентата с менюта (menubar) > File (Файл) > Import Rules (Импортиране на правила)

### Импорт на iOS

Да предположим, че файлът с правилата ви е в приложението Files (Файлове). Отворете го във Files, докоснете бутона Share (Споделяне) и след това изберете Redirect Web:

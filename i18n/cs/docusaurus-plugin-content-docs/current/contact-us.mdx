# Kontaktujte nás

Jsme tu, abychom vám pomohli, pokud máte jakékoli dotazy, zpětnou vazbu nebo problémy s Redirect Web. Zde jsou nejlepší způsoby, jak se s námi spojit:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Pokud narazíte na chyby nebo chcete požádat o nové funkce, navštivte stránku GitHub Issues. Můžete procházet stávající problémy, odeslat nový problém a sledovat průběh vašeho problému.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Pokud máte jakékoli dotazy nebo chcete získat pomoc od vývojáře nebo jiných uživatelů, navštivte stránku GitHub Discussions. Můžete zahájit novou diskuzi, připojit se ke stávajícím diskuzím a získat pomoc od komunity.
- **Formulář zpětné vazby**: Zpětnou vazbu nám můžete poslat prostřednictvím formuláře zpětné vazby v aplikaci Redirect Web. Ten odešle e-mail přímo vývojáři, takže vaše zpětná vazba zůstane soukromá před ostatními uživateli.

Děkujeme, že používáte Redirect Web! Vážíme si vaší podpory a zpětné vazby.

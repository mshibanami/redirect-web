# Kontaktujte nás

Jsme tu, abychom vám pomohli, pokud máte jakékoli dotazy, zpětnou vazbu nebo problémy s Redirect Web. Zde jsou nejlepší způsoby, jak se s námi spojit:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Pokud narazíte na nějaké chyby nebo chcete požádat o nové funkce, navštivte stránku GitHub Issues. Můžete procházet existující problémy, odeslat nový problém a sledovat průběh vašeho problému.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Pokud máte jakékoli dotazy nebo chcete získat pomoc od vývojáře či jiných uživatelů, navštivte stránku GitHub Discussions. Můžete zahájit novou diskusi, připojit se k existujícím diskusím a získat pomoc od komunity.
- **Formulář zpětné vazby**: Zpětnou vazbu nám můžete poslat prostřednictvím formuláře zpětné vazby v aplikaci Redirect Web. Tímto se odešle e-mail přímo vývojáři, takže vaše zpětná vazba zůstane soukromá a nebude viditelná pro ostatní uživatele.

Děkujeme, že používáte Redirect Web! Velmi si vážíme vaší podpory a zpětné vazby.

# Knihovna

Vítejte v naší knihovně, kde najdete užitečná, hotová pravidla zdarma! Můžete k nim přistupovat prostřednictvím aplikace Redirect Web.

![Snímek obrazovky knihovny](/img/library-screenshot.webp)

Chcete-li zkopírovat pravidla do Mých pravidel, stačí vybrat tlačítko Získat:

![Tlačítko Získat - iOS](/img/library-get-button-ios.gif)

## Přispějte svým pravidlem do knihovny

Současná knihovna má jen několik pravidel, takže si vážíme vašeho příspěvku!

Existují 2 možnosti, jak přispět svými pravidly do knihovny:

1. **Zveřejněte své pravidlo na fóru**: Sdílejte své pravidlo přesměrování s ostatními uživateli na [fóru](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Poté vás můžeme požádat, zda bychom vaše pravidlo mohli zařadit i do knihovny. To může být snazší než možnost 2, protože přidání můžete nechat na nás.
2. **Pošlete nám pull request**: Můžete nám poslat pull request. Další informace naleznete v [pokynech](https://github.com/mshibanami/redirect-web/tree/main/library).

## Další informace

- Veškerý obsah naleznete v [úložišti na GitHubu](https://github.com/mshibanami/redirect-web/tree/main/library).
- Pro obsah knihovny hostujeme náš soukromý API server na Cloudflare, což nám umožňuje poskytovat filtrování a další služby. Aplikace načítá obsah knihovny z tohoto serveru. Jak je uvedeno v [našich zásadách ochrany osobních údajů](/privacy-policy.mdx), tento server neshromažďuje vaše osobní údaje.

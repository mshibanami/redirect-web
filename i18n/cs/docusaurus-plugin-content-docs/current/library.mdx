# Knihovna

Vítejte v naší knihovně, kde zdarma naleznete užitečná pravidla připravená k použití! Můžete k nim přistupovat prostřednictvím aplikace Redirect Web.

![Snímek obrazovky knihovny](/img/library-screenshot.webp)

Chcete-li pravidla zkopírovat do Mých pravidel, stačí jednoduše vybrat tlačítko Získat:

![Tlačítko Získat - iOS](/img/library-get-button-ios.gif)

## Přispějte svým pravidlem do knihovny

Současná knihovna má jen několik pravidel, takže si vážíme vašeho příspěvku!

Existují 2 možnosti, jak přispět svými pravidly do knihovny:

1.  **Zveřejněte své pravidlo na fóru**: Sdílejte své pravidlo přesměrování s ostatními uživateli na [fóru](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Pak můžeme požádat, zda bychom vaše pravidlo mohli zařadit i do knihovny. To může být snazší než možnost 2, protože přidání můžete nechat na nás.
2.  **Pošlete nám pull request**: Můžete nám poslat pull request. Další informace naleznete v [pokynech](https://github.com/mshibanami/redirect-web/tree/main/library).

## Další informace

- Veškerý obsah si můžete prohlédnout v [úložišti na GitHubu](https://github.com/mshibanami/redirect-web/tree/main/library).
- Náš soukromý API server pro obsah knihovny hostujeme na Cloudflare, což nám umožňuje poskytovat filtrování a další služby. Aplikace načítá obsah knihovny z tohoto serveru. Jak je uvedeno v [našich zásadách ochrany osobních údajů](privacy-policy.mdx), tento server neshromažďuje vaše osobní údaje.

# Knihovna

Vítejte v naší knihovně, kde najdete užitečná hotová pravidla zdarma! Můžete k nim přistupovat prostřednictvím aplikace Redirect Web.

![Library Screenshot](/img/library-screenshot.webp)

Chcete-li pravidla zkopírovat do části Moje pravidla, jednoduše stiskněte tlačítko Získat:

![Get Button - iOS](/img/library-get-button-ios.gif)

## Přispějte svými pravidly do knihovny

Současná knihovna obsahuje jen několik pravidel, proto si velmi vážíme vašeho příspěvku!

Existují 2 možnosti, jak přispět svými pravidly do knihovny:

1. **Zveřejněte své pravidlo na fóru**: Podělte se o své pravidlo pro přesměrování s ostatními uživateli na [fóru](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Pak vás můžeme požádat, zda bychom vaše pravidlo mohli zařadit i do knihovny. To může být snazší než možnost 2, protože přidání můžete nechat na nás.
2. **Pošlete nám pull request**: Můžete nám poslat pull request. Další informace naleznete v [pokynech](https://github.com/mshibanami/redirect-web/tree/main/library).

## Další informace

- Veškerý obsah naleznete v [úložišti na GitHubu](https://github.com/mshibanami/redirect-web/tree/main/library).
- Soubory knihovny hostujeme na našem soukromém API serveru na Cloudflare, což nám umožňuje poskytovat filtrování a další služby. Aplikace načítá obsah knihovny z tohoto serveru. Jak je uvedeno v [našich zásadách ochrany osobních údajů](/privacy-policy), tento server neshromažďuje vaše osobní údaje.

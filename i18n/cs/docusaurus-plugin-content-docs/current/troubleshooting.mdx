## Pravidla přesměrování nefungují

Pokud vaše pravidla nefungují podle očekávání, zkontrolujte následující:

- **Rozšíření prohlížeče je nainstalováno a povoleno pro správnou stránku:** Ujistěte se, že máte nainstalované rozšíření prohlížeče Redirect Web a že je povoleno ve vašem prohlížeči pro webovou stránku, ze které chcete přesměrovat. Pokud můžete otevřít vyskakovací okno rozšíření na zdrojové webové stránce takto, je nainstalováno správně:
    <Image src={ExtensionPopup} alt="Vyskakovací okno rozšíření Redirect Web" width="260" hasShadow/>

- **Konfigurace pravidla je správná:** Dvakrát zkontrolujte svá pravidla přesměrování v nativní aplikaci Redirect Web. Ujistěte se, že vzory zdrojových URL, cílové URL a všechny podmínky jsou správně nastaveny. Chcete-li zkontrolovat, zda je vaše pravidlo platné, zadejte příklad URL do sekce Příklady na obrazovce Úpravy pravidla. Pokud je vaše pravidlo platné, uvidíte pod vaším vstupem očekávanou cílovou URL, takto:
    <Image src={RuleEditExampleResult} alt="Příklad obrazovky pro úpravu pravidla zobrazující výsledek příkladové URL" width="260" hasShadow/>

## Vyskakovací okno rozšíření zobrazuje chybu

<Image src={ErrorOverlay} alt="Překrytí chyby Redirect Web" width="260" hasShadow hasRoundedCorners/>

> Aplikace Redirect Web nebyla nainstalována nebo nefunguje správně.
> Získejte aplikaci z App Store.

Pokud se při otevírání vyskakovacího okna rozšíření zobrazí výše uvedená chyba, znamená to, že nativní aplikace Redirect Web není nainstalována nebo nefunguje správně.

Ujistěte se, že máte nainstalovanou nejnovější verzi Redirect Web z [App Store](https://apps.apple.com/app/id1571283503).

Po instalaci nativní aplikace možná budete muset znovu spustit prohlížeč.

> [!NOTE]
> * K používání rozšíření prohlížeče musíte nainstalovat nativní aplikaci Redirect Web. Samotné rozšíření nefunguje.
> * Uživatelé Safari nemusí instalovat rozšíření samostatně, protože je součástí nativní aplikace.

## Původní stránka krátce blikne před přesměrováním

K tomu dochází, když používáte pravidla přesměrování typu **Originální**. Tento typ poskytuje různé pokročilé funkce, ale může způsobit krátké bliknutí původní stránky, protože může umožnit načtení původní stránky před provedením přesměrování.

Otevřete obrazovku Úpravy pravidla a změňte **Typ** na **DNR**, abyste se tomuto problému vyhnuli.

<Image src={EditRuleDnr} alt="Vyskakovací okno rozšíření Redirect Web" height="260" hasShadow hasRoundedCorners/>

> [!NOTE]
> Ve Firefoxu by k tomuto problému nemělo docházet ani u typu Originální, protože se spoléhá na API WebRequest, které zabraňuje načtení původní stránky.

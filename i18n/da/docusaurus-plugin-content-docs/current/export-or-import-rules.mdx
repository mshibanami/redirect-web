# Eksport/Import af Regler

## Eksport af Regler {#export-rules}

### Eksport på macOS {#export-on-macos}

Klik enten på ellipseknappen eller Del-knappen som vist på dette skærmbillede:

![Del på macOS](/img/share-on-macos.png)

Du kan også multi-vælge dine regler og eksportere dem ved at sekundærklikke (højreklikke) på dem i sidemenuen med regellisten:

![Del flere regler på macOS](/img/multiselect-on-macos.png)

Derudover kan du eksportere alle regler fra menulinjen > Arkiv > Eksporter alle regler:

![Menulinje > Arkiv](/img/menubar-file.png)

> [!NOTE]
> Hvis du vil eksportere dine regler i et format, der er kompatibelt med [Einar Egilssons Redirector](https://einaregilsson.com/redirector/), skal du trykke på Option-tasten (`⌥`) i kontekstmenuen eller menulinjen, så vil du se disse:
> ![Menulinje > Arkiv (Redirector)](/img/menubar-file-redirector.png)
>
> ![Kontekstmenu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web er ikke fuldt kompatibel med Redirector, og omvendt. Nogle indstillinger, såsom "Erstat forekomster" for behandling af fangstgrupper, eksporteres ikke med Redirectors format.
> :::

## Import af Regler {#import-rules}

### Import på macOS {#import-on-macos}

- Du skal blot trække og slippe de eksporterede regelfiler til Redirect Webs vindue.
- Eller du kan også importere dem fra menulinjen > Arkiv > Importer regler

### Import på iOS {#import-on-ios}

Antag at din regelfil er i appen Filer. Åbn den i Filer, tryk på Del-knappen, og vælg derefter Redirect Web:

| Trin 1: Tryk på Del-knappen i værktøjslinjen                                     | Trin 2: Vælg Redirect Web                                         |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Tryk på del-knappen i værktøjslinjen](/img/files-share-ios.png) | ![Tryk på app-ikonet i arket](/img/share-sheet-ios.png) |

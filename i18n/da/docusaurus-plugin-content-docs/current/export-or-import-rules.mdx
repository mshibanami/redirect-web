# Eksport/Import af Regler

## Eksport af Regler {#export-rules}

### Eksport på macOS {#export-on-macos}

Klik enten på ellipsknappen eller deleknappen, som vist på dette skærmbillede:

![Del på macOS](/img/share-on-macos.png)

Du kan også vælge flere af dine regler og eksportere dem ved at sekundærklikke (højreklikke) på dem i sidepanelet med regeloversigten:

![Del flere regler på macOS](/img/multiselect-on-macos.png)

Derudover kan du eksportere alle regler fra menulinjen > Arkiv > Eksporter alle regler:

![Menulinje > Arkiv](/img/menubar-file.png)

> [!BEMÆRK]
> Hvis du vil eksportere dine regler i et format, der er kompatibelt med [Einar Egilssons Redirector](https://einaregilsson.com/redirector/), skal du trykke på Option-tasten (`⌥`) i kontekstmenuen eller på menulinjen, hvorefter du vil se disse:
> ![Menulinje > Arkiv (Redirector)](/img/menubar-file-redirector.png)
>
> ![Kontekstmenu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web er ikke fuldt ud kompatibel med Redirector, og omvendt. Nogle indstillinger, såsom "Erstat forekomster" for bearbejdning af fangstgrupper, eksporteres ikke med Redirectors format.
> :::

## Import af Regler {#import-rules}

### Import på macOS {#import-on-macos}

- Du skal blot trække og slippe de eksporterede regelfiler til Redirect Webs vindue.
- Eller du kan også importere dem fra menulinjen > Arkiv > Importér regler

### Import på iOS {#import-on-ios}

Antag, at din regelfil ligger i appen Filer. Åbn den i Filer, tryk på Del-knappen, og vælg derefter Redirect Web:

| Trin 1: Tryk på Del-knappen i Filer                                       | Trin 2: Vælg Redirect Web                                           |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Tryk på deleknappen i værktøjslinjen](/img/files-share-ios.png) | ![Tryk på appikonet i arket](/img/share-sheet-ios.png) |

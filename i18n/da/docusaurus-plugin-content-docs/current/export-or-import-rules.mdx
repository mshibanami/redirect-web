# Eksport-/importregler

## Eksportregler

### Eksport på macOS

Klik enten på ellipsknappen eller deleknappen som vist på dette skærmbillede:

![Share on macOS](/img/share-on-macos.png)

Du kan også multiselektere dine regler og eksportere dem ved at sekundært klikke (højreklikke) på dem i sidepanelet med regeloversigten:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Derudover kan du eksportere alle reglerne fra menulinjen > Arkiv > Eksporter alle regler:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Hvis du vil eksportere dine regler i et format, der er kompatibelt med [Einar Egilssons Redirector](https://einaregilsson.com/redirector/), skal du trykke på Option-tasten (`⌥`) i kontekstmenuen eller på menulinjen; så vil du se følgende:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web er ikke fuldt kompatibel med Redirector, og omvendt. Nogle indstillinger, såsom "Erstat forekomster" for bearbejdning af opfangende grupper, eksporteres ikke med Redirectors format.
> :::

## Importregler

### Import på macOS

- Du skal blot trække og slippe de eksporterede regelfiler til Redirect Webs vindue.
- Eller du kan importere dem fra menulinjen > Arkiv > Importer regler

### Import på iOS

Antag at din regelfil er i Filer-appen. Åbn den i Filer, tryk på Deleknappen, og vælg derefter Redirect Web:

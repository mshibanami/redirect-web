# Bibliotek

Velkommen til vores bibliotek, hvor du gratis kan finde nyttige, klar-til-brug regler! Du kan få adgang til dem via Redirect Web-appen.

![Library Screenshot](/img/library-screenshot.webp)

For at kopiere reglerne til Mine Regler skal du blot vælge knappen Hent:

![Get Button - iOS](/img/library-get-button-ios.gif)

## Bidrag med din regel til biblioteket

Det nuværende bibliotek har kun få regler, så vi sætter pris på dit bidrag!

Der er 2 muligheder for at bidrage med dine regler til biblioteket:

1.  **Slå din regel op på forummet**: Del din omdirigeringsregel med andre brugere på [forummet](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Derefter kan vi spørge, om vi også må lægge din regel i biblioteket. Dette kan være lettere end mulighed 2, fordi du kan overlade tilføjelsen til os.
2.  **Send os en pull request**: Du kan sende os en pull request. Se venligst [instruktionerne](https://github.com/mshibanami/redirect-web/tree/main/library) for at lære mere.

## Anden information

-   Du kan se alt indhold i [repository'et på GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
-   Vi hoster vores private API-server på Cloudflare for biblioteksindholdet, hvilket gør os i stand til at levere filtrering og andre tjenester. Appen henter biblioteksindholdet fra denne server. Som angivet i [vores privatlivspolitik](/privacy-policy), indsamler denne server ikke dine personlige oplysninger.

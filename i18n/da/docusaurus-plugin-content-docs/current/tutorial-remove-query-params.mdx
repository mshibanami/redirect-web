# Case 2: Fjern forespørgselsparametre fra URL

Antag, at der er en forespørgselsparameter `source=twitter` i en URL på `example.com`, og du besluttede at fjerne den for at anonymisere dig selv.

I dette tilfælde er **Behandling af fangstgrupper** den nemmeste måde:

*   **Omdiriger fra**: `https://example.com/*` (Jokertegn)
*   **Omdiriger til**: `$0`
*   **Behandling af fangstgrupper**:
    *   **Gruppe**: `$0`
    *   **Proces**: Erstat forekomster
        *   **Mål**: `&?source=[^&]*`
        *   **Erstatning**: *(ingen)*
        *   **Tekstmønster**: Regulært udtryk

**<a href={useBaseUrl('rules/remove-parameters.redirectweb')} download>⬇️ Download reglen</a>**

Denne regel fungerer som følger:

```
https://example.com/?source=twitter
➡️ https://example.com/?
```

```
https://example.com/?hello=world&source=twitter&foo=bar
➡️ https://example.com/?hello=world&foo=bar
```

Hvis du vil fjerne flere parametre, skal du tilføje flere processer.

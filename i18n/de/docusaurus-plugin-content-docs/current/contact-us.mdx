# Kontakt

Wir sind hier, um Ihnen bei Fragen, Feedback oder Problemen mit Redirect Web zu helfen. Hier sind die besten Wege, um mit uns in Kontakt zu treten:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Wenn Sie Fehler entdecken oder neue Funktionen anfordern möchten, besuchen Sie bitte die GitHub Issues-Seite. Sie können bestehende Issues durchsuchen, ein neues Issue einreichen und den Fortschritt Ihres Issues verfolgen.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Wenn Sie Fragen haben oder Hilfe vom Entwickler oder anderen Benutzern erhalten möchten, besuchen Sie bitte die GitHub Discussions-Seite. Sie können eine neue Diskussion starten, bestehenden Diskussionen beitreten und Hilfe von der Community erhalten.
- **Feedback-Formular**: Sie können uns Feedback über das Feedback-Formular in der Redirect Web App senden. Dies sendet eine E-Mail direkt an den Entwickler, sodass Ihr Feedback von anderen Benutzern privat gehalten wird.

Vielen Dank für die Nutzung von Redirect Web! Wir wissen Ihre Unterstützung und Ihr Feedback zu schätzen.

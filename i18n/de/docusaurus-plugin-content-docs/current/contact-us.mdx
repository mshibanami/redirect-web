# Kontakt

Wir sind für Sie da, wenn Sie Fragen, Feedback oder Probleme mit Redirect Web haben. Hier sind die besten Wege, um mit uns in Kontakt zu treten:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Wenn Sie auf Fehler stoßen oder neue Funktionen anfragen möchten, besuchen Sie bitte die GitHub Issues-Seite. Dort können Sie bestehende Issues durchsuchen, neue Issues einreichen und den Fortschritt Ihres Issues verfolgen.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Wenn Sie Fragen haben oder Hilfe vom Entwickler oder anderen Benutzern erhalten möchten, besuchen Sie bitte die GitHub Discussions-Seite. Dort können Sie eine neue Diskussion starten, bestehenden Diskussionen beitreten und Hilfe von der Community erhalten.
- **Feedback-Formular**: Sie können uns Feedback über das Feedback-Formular in der Redirect Web App senden. Dies sendet eine E-Mail direkt an den Entwickler, sodass Ihr Feedback vor anderen Benutzern privat bleibt.

Vielen Dank, dass Sie Redirect Web nutzen! Wir wissen Ihre Unterstützung und Ihr Feedback zu schätzen.

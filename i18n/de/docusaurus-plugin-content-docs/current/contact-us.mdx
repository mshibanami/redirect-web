# Kontaktieren Sie uns

Wir sind hier, um Ihnen zu helfen, wenn Sie Fragen, Feedback oder Probleme mit Redirect Web haben. Hier sind die besten Wege, um mit uns in Kontakt zu treten:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Wenn Sie Fehler finden oder neue Funktionen anfordern möchten, besuchen Sie bitte die GitHub Issues-Seite. Sie können bestehende Probleme durchsuchen, ein neues Problem einreichen und den Fortschritt Ihres Problems verfolgen.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Wenn Sie Fragen haben oder Hilfe vom Entwickler oder anderen Benutzern erhalten möchten, besuchen Sie bitte die GitHub Discussions-Seite. Sie können eine neue Diskussion starten, an bestehenden Diskussionen teilnehmen und Hilfe von der Community erhalten.
- **Feedback-Formular**: Sie können uns Feedback über das Feedback-Formular in der Redirect Web App senden. Dies sendet eine E-Mail direkt an den Entwickler, sodass Ihr Feedback vor anderen Benutzern privat bleibt.

Vielen Dank, dass Sie Redirect Web verwenden! Wir wissen Ihre Unterstützung und Ihr Feedback zu schätzen.

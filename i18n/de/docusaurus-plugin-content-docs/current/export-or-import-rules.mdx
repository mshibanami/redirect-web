# Regeln exportieren/importieren

## Regeln exportieren {#export-rules}

### Exportieren unter macOS {#export-on-macos}

Klicken Sie entweder auf die Ellipsen-Schaltfläche oder die Teilen-Schaltfläche, wie in diesem Screenshot gezeigt:

![Teilen unter macOS](/img/share-on-macos.png)

Sie können Ihre Regeln auch mehrfach auswählen und exportieren, indem Sie sie in der Regellisten-Seitenleiste mit einem Sekundärklick (Rechtsklick) anklicken:

![Mehrere Regeln unter macOS teilen](/img/multiselect-on-macos.png)

Zusätzlich können Sie alle Regeln über die Menüleiste > Datei > Alle Regeln exportieren:

![Menüleiste > Datei](/img/menubar-file.png)

> [!NOTE]
> Wenn Sie Ihre Regeln in einem Format exportieren möchten, das mit [Einar Egilssons Redirector](https://einaregilsson.com/redirector/) kompatibel ist, drücken Sie die Optionstaste (`⌥`) im Kontextmenü oder in der Menüleiste, dann sehen Sie Folgendes:
> ![Menüleiste > Datei (Redirector)](/img/menubar-file-redirector.png)
>
> ![Kontextmenü (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web ist nicht vollständig kompatibel mit Redirector und umgekehrt. Einige Einstellungen, wie z.B. "Vorkommen ersetzen" für die Verarbeitung von Erfassungsgruppen, werden nicht im Redirector-Format exportiert.
> :::

## Regeln importieren {#import-rules}

### Importieren unter macOS {#import-on-macos}

- Ziehen Sie die exportierten Regeldateien einfach per Drag & Drop in das Fenster von Redirect Web.
- Oder Sie können sie auch über die Menüleiste > Datei > Regeln importieren

### Importieren unter iOS {#import-on-ios}

Angenommen, Ihre Regeldatei befindet sich in der Dateien-App. Öffnen Sie sie in Dateien, tippen Sie auf die Teilen-Schaltfläche und wählen Sie dann Redirect Web aus:

| Schritt 1: Tippen Sie auf die Teilen-Schaltfläche in der Symbolleiste | Schritt 2: Wählen Sie Redirect Web aus |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Tippen Sie auf die Teilen-Schaltfläche in der Symbolleiste](/img/files-share-ios.png) | ![Tippen Sie auf das App-Symbol im Blatt](/img/share-sheet-ios.png) |

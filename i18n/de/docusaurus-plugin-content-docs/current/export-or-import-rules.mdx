# Regeln exportieren/importieren

## Regeln exportieren {#export-rules}

### Exportieren unter macOS {#export-on-macos}

Klicken Sie entweder auf die Schaltfläche mit den Auslassungspunkten oder auf die Schaltfläche "Teilen", wie in diesem Screenshot gezeigt:

<Image src={ShareOnMacOS} alt="Schaltfläche 'Teilen' unter macOS" width="460" hasShadow hasRoundedCorners/>

Wählen Sie dann "Regel exportieren" oder "Regel teilen..." aus dem Kontextmenü:

Sie können auch mehrere Regeln auswählen und sie durch Sekundärklick (Rechtsklick) auf der Regellisten-Seitenleiste exportieren:

<Image src={MultiSelectOnMacOS} alt="Mehrere Regeln unter macOS teilen" height="400" hasShadow hasRoundedCorners/>


Zusätzlich können Sie alle Regeln über die Menüleiste > Datei > Alle Regeln exportieren:

<Image src={MenubarFile} alt="Menüleiste > Datei" width="400" hasShadow hasRoundedCorners/>

> [!NOTE]
> Wenn Sie Ihre Regeln in einem Format exportieren möchten, das mit [Einar Egilssons Redirector](https://einaregilsson.com/redirector/) kompatibel ist, drücken Sie die Optionstaste (`⌥`) im Kontextmenü oder in der Menüleiste, dann sehen Sie Folgendes:
> <Image src={MenubarFileRedirector} alt="Menüleiste > Datei (Redirector)" width="400" hasShadow hasRoundedCorners/>
> <br/>
> <Image src={ContextMenuRedirector} alt="Kontextmenü (Redirector)" width="400" hasShadow hasRoundedCorners/>
>
> :::warning
> Redirect Web ist nicht vollständig kompatibel mit Redirector und umgekehrt. Einige Einstellungen, wie z. B. "Vorkommen ersetzen" für die Verarbeitung von Erfassungsgruppen, werden nicht im Redirector-Format exportiert.
> :::

## Regeln importieren {#import-rules}

### Importieren unter macOS {#import-on-macos}

- Ziehen Sie die exportierten Regelsdateien einfach in das Fenster von Redirect Web.
- Oder Sie können sie auch über die Menüleiste > Datei > Regeln importieren.

### Importieren unter iOS {#import-on-ios}

Angenommen, Ihre Regelsdatei befindet sich in der Dateien-App. Öffnen Sie sie in Dateien, tippen Sie auf die Schaltfläche "Teilen" und wählen Sie dann Redirect Web:

| Schritt 1: Tippen Sie in Dateien auf die Schaltfläche "Teilen" | Schritt 2: Wählen Sie Redirect Web |

# Export-/Import-Regeln

## Regeln exportieren {#export-rules}

### Exportieren auf macOS {#export-on-macos}

Klicken Sie entweder auf die Auslassungspunkte-Schaltfläche oder auf die Teilen-Schaltfläche, wie in diesem Screenshot gezeigt:

![Share on macOS](/img/share-on-macos.png)

Sie können Ihre Regeln auch mehrfach auswählen und durch einen Sekundärklick (Rechtsklick) auf sie in der Regel-Seitenleiste exportieren:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Zusätzlich können Sie alle Regeln über die Menüleiste > Ablage > Alle Regeln exportieren:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Wenn Sie Ihre Regeln in einem Format exportieren möchten, das mit [Einar Egilssons Redirector](https://einaregilsson.com/redirector/) kompatibel ist, drücken Sie die Optionstaste (`⌥`) im Kontextmenü oder in der Menüleiste. Dann sehen Sie Folgendes:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web ist nicht vollständig kompatibel mit Redirector und umgekehrt. Einige Einstellungen, wie z.B. „Ersetzungen vornehmen“ für die Verarbeitung von Erfassungsgruppen, werden nicht im Redirector-Format exportiert.
> :::

## Regeln importieren {#import-rules}

### Importieren auf macOS {#import-on-macos}

- Ziehen Sie die exportierten Regeldateien einfach per Drag-and-drop in das Fenster von Redirect Web.
- Alternativ können Sie sie auch über die Menüleiste > Ablage > Regeln importieren.

### Importieren auf iOS {#import-on-ios}

Angenommen, Ihre Regeldatei befindet sich in der Dateien-App. Öffnen Sie sie in der Dateien-App, tippen Sie auf die Teilen-Schaltfläche und wählen Sie dann Redirect Web:

| Schritt 1: Tippen Sie in der Symbolleiste auf die Teilen-Schaltfläche | Schritt 2: Wählen Sie Redirect Web                         |
| -------------------------------------------------------------------- | ---------------------------------------------------------- |
| ![Tap the share button in the toolbar](/img/files-share-ios.png) | ![Tap the app icon in the sheet](/img/share-sheet-ios.png) |

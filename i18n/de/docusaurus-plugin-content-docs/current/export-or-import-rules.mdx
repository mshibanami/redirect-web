# Regeln exportieren/importieren

## Regeln exportieren

### Exportieren unter macOS

Klicken Sie entweder auf die Ellipsen-Schaltfläche oder die Teilen-Schaltfläche, wie in diesem Screenshot gezeigt:

![Teilen unter macOS](/img/share-on-macos.png)

Sie können Ihre Regeln auch mehrfach auswählen und sie durch sekundäres Klicken (Rechtsklick) in der Regellisten-Seitenleiste exportieren:

![Mehrere Regeln unter macOS teilen](/img/multiselect-on-macos.png)

Zusätzlich können Sie alle Regeln über die Menüleiste > Datei > Alle Regeln exportieren:

![Menüleiste > Datei](/img/menubar-file.png)

> [!NOTE]
> Wenn Sie Ihre Regeln in einem Format exportieren möchten, das mit [Einar Egilssons Redirector](https://einaregilsson.com/redirector/) kompatibel ist, drücken Sie die Option-Taste (`⌥`) im Kontextmenü oder in der Menüleiste. Dann sehen Sie Folgendes:
> ![Menüleiste > Datei (Redirector)](/img/menubar-file-redirector.png)
>
> ![Kontextmenü (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web ist nicht vollständig mit Redirector kompatibel und umgekehrt. Einige Einstellungen, wie z.B. "Ersetzungen" für die Verarbeitung von Erfassungsgruppen, werden nicht im Redirector-Format exportiert.
> :::

## Regeln importieren

### Importieren unter macOS

- Ziehen Sie die exportierten Regeldateien einfach in das Fenster von Redirect Web.
- Oder Sie können sie auch über die Menüleiste > Datei > Regeln importieren.

### Importieren unter iOS

Angenommen, Ihre Regeldatei befindet sich in der Dateien-App. Öffnen Sie sie in Dateien, tippen Sie auf die Teilen-Schaltfläche und wählen Sie dann Redirect Web:

| Schritt 1: Tippen Sie auf die Teilen-Schaltfläche in Dateien | Schritt 2: Wählen Sie Redirect Web           |

# Bibliothek

Willkommen in unserer Bibliothek, wo Sie nützliche, sofort einsatzbereite Regeln kostenlos finden können! Sie können über die Redirect Web App darauf zugreifen.

![Library Screenshot](/img/library-screenshot.webp)

Um die Regeln zu "Meine Regeln" zu kopieren, klicken Sie einfach auf die Schaltfläche "Holen":

![Get Button - iOS](/img/library-get-button-ios.gif)

## Tragen Sie Ihre Regel zur Bibliothek bei

Die aktuelle Bibliothek enthält nur wenige Regeln, daher freuen wir uns über Ihren Beitrag!

Es gibt 2 Möglichkeiten, Ihre Regeln zur Bibliothek beizutragen:

1. **Veröffentlichen Sie Ihre Regel im Forum**: Teilen Sie Ihre Umleitungsregel mit anderen Benutzern im [Forum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Wir fragen Sie dann möglicherweise, ob wir Ihre Regel auch in die Bibliothek aufnehmen dürfen. Dies könnte einfacher sein als Option 2, da Sie die Hinzufügung uns überlassen können.
2. **Senden Sie uns einen Pull Request**: Sie können uns einen Pull Request senden. Bitte lesen Sie [die Anweisungen](https://github.com/mshibanami/redirect-web/tree/main/library), um mehr zu erfahren.

## Weitere Informationen

- Sie können alle Inhalte im [Repository auf GitHub](https://github.com/mshibanami/redirect-web/tree/main/library) einsehen.
- Wir hosten unseren privaten API-Server für die Bibliotheksinhalte auf Cloudflare, was uns ermöglicht, Filter- und andere Dienste anzubieten. Die App ruft die Bibliotheksinhalte von diesem Server ab. Wie in [unserer Datenschutzrichtlinie](privacy-policy.mdx) dargelegt, sammelt dieser Server keine Ihrer persönlichen Daten.

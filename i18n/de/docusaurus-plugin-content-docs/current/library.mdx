# Bibliothek

Willkommen in unserer Bibliothek, in der Sie nützliche, gebrauchsfertige Regeln kostenlos finden können! Sie können über die Redirect Web App darauf zugreifen.

![Library Screenshot](/img/library-screenshot.webp)

Um die Regeln in „Meine Regeln“ zu kopieren, können Sie einfach die Schaltfläche „Abrufen“ auswählen:

![Get Button - iOS](/img/library-get-button-ios.gif)

## Tragen Sie Ihre Regel zur Bibliothek bei

Die aktuelle Bibliothek hat nur wenige Regeln, daher freuen wir uns über Ihren Beitrag!

Es gibt 2 Möglichkeiten, Ihre Regeln zur Bibliothek beizutragen:

1.  **Veröffentlichen Sie Ihre Regel im Forum**: Teilen Sie Ihre Weiterleitungsregel mit anderen Benutzern im [Forum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Wir können Sie dann fragen, ob wir Ihre Regel auch in die Bibliothek aufnehmen dürfen. Dies ist möglicherweise einfacher als Option 2, da Sie die Hinzufügung uns überlassen können.
2.  **Senden Sie uns einen Pull Request**: Sie können uns einen Pull Request senden. Bitte beachten Sie [die Anweisungen](https://github.com/mshibanami/redirect-web/tree/main/library), um mehr zu erfahren.

## Weitere Informationen

-   Sie können alle Inhalte im [Repository auf GitHub](https://github.com/mshibanami/redirect-web/tree/main/library) einsehen.
-   Wir hosten unseren privaten API-Server auf Cloudflare für die Bibliotheksinhalte, wodurch wir Filter- und andere Dienste anbieten können. Die App ruft die Bibliotheksinhalte von diesem Server ab. Wie in [unserer Datenschutzerklärung](/privacy-policy) angegeben, sammelt dieser Server keine Ihrer persönlichen Informationen.

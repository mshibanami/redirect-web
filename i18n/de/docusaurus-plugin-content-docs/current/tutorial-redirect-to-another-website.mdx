# Fall 1: Weiterleitung zu einer anderen Website

Angenommen, Sie sind Twitter-süchtig und beschließen, Insight Timer zum Meditieren zu verwenden, wann immer Sie versehentlich Twitter öffnen. Erstellen wir eine Regel dafür!

### Schritt 1. Öffnen Sie den Bildschirm "Regel bearbeiten"

Öffnen Sie den Bildschirm *Regel bearbeiten*, um Ihre eigene Regel in der App zu erstellen. Dies können Sie tun, indem Sie auf die Plus-Schaltfläche (+) im Tab *Meine Regeln* klicken.

### Schritt 2. Richten Sie "Umleiten von" ein

Zuerst sollten Sie den Abschnitt *Umleiten von* im Bildschirm *Regel bearbeiten* in der App einrichten. Die Ziele sind URLs, die mit `https://twitter.com/` beginnen.

In diesem Fall können Sie das folgende Muster mit dem Modus [Wildcard](rule-settings#url-pattern-wildcard) angeben:

```
https://twitter.com/*
```

Im Wildcard-Modus bedeutet `*`, dass es alles übereinstimmt (= null oder mehr Zeichen).

Aber warten Sie. `twitter.com` wurde in `x.com` umbenannt. Sie können `twitter.com` einfach durch `x.com` ersetzen, aber niemand weiß, wann der neue Besitzer seine Meinung wieder ändern wird, um es rückgängig zu machen. Daher zielen wir auf `twitter.com` und `x.com` ab.

Wechseln Sie dazu von Wildcard zu [Regulärer Ausdruck](rule-settings#url-pattern-regular-expression) und legen Sie dies als Muster fest:

```
https://(twitter|x).com/.*
```

* `(twitter|x)`: Es zielt auf `twitter` und `x` ab. (`|` wird als *Pipe* bezeichnet.)
* `.*`: Es ist dasselbe wie Wildcard's `*`. Genauer gesagt bedeutet `.` *jedes Zeichen* und `*` bedeutet *alles vor diesem Symbol, das beliebig oft wiederholt wird*, was dazu führt, dass es alles übereinstimmt.

Reguläre Ausdrücke sind etwas kompliziert, aber sobald Sie sich daran gewöhnt haben, werden sie ein mächtiges Werkzeug sein. Wir empfehlen die Verwendung von [RegExr](https://regexr.com) als Spielplatz, um zu analysieren, wie Ihr Muster für reguläre Ausdrücke funktioniert.

> [!TIP]
> In regulären Ausdrücken wird `.` in `(twitter|x).com` auch als *jedes Zeichen* behandelt. Daher stimmt `(twitter|x).com/.*` auch mit beispielsweise `twitter1com/` oder `x_com/` überein.
> 
> Um genauer zu sein, können Sie es in `(twitter|x)\.com/.*` ändern. `\` wird verwendet, um `.` als wörtliches Punktzeichen und nicht als Sonderzeichen zu behandeln.
> 
> Solche URLs gibt es jedoch in der allgemeinen Internetumgebung nicht. Daher können Sie `.` als Sonderzeichen belassen, wenn Sie dies bevorzugen.

### Schritt 3: Richten Sie "Umleiten zu" ein

Geben Sie einfach die URL wie folgt an:

```
https://insighttimer.com/saraauster/guided-meditations/calm
```

### (Optional) Schritt 4: Überprüfen Sie die Regel

Sie können überprüfen, wie die Regel funktioniert, indem Sie eine Beispiel-URL im Abschnitt *Beispiele* im Bildschirm *Regel bearbeiten* hinzufügen. Wenn Sie beispielsweise `https://x.com/UN` eingeben, wird `https://insighttimer.com/saraauster/guided-meditations/calm` als Ergebnis angezeigt.

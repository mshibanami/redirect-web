# Εξαγωγή/Εισαγωγή Κανόνων

## Εξαγωγή Κανόνων

### Εξαγωγή σε macOS

Κάντε κλικ είτε στο κουμπί αποσιωπητικών είτε στο κουμπί Κοινή χρήση, όπως φαίνεται σε αυτό το στιγμιότυπο οθόνης:

![Share on macOS](/img/share-on-macos.png)

Μπορείτε επίσης να επιλέξετε πολλούς κανόνες και να τους εξαγάγετε κάνοντας δευτερεύον κλικ (δεξί κλικ) σε αυτούς στην πλαϊνή γραμμή λίστας κανόνων:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Επιπλέον, μπορείτε να εξαγάγετε όλους τους κανόνες από τη γραμμή μενού > Αρχείο > Εξαγωγή Όλων των Κανόνων:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Εάν θέλετε να εξαγάγετε τους κανόνες σας στη μορφή που είναι συμβατή με το [Redirector του Einar Egilsson](https://einaregilsson.com/redirector/), πατήστε το πλήκτρο Option (`⌥`) στο μενού περιβάλλοντος ή στη γραμμή μενού, τότε θα δείτε τα εξής:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Το Redirect Web δεν είναι πλήρως συμβατό με το Redirector, και αντίστροφα. Ορισμένες ρυθμίσεις, όπως το "Replace Occurrences" για την Επεξεργασία Ομάδων Καταγραφής (Capturing Group Processing), δεν εξάγονται με τη μορφή του Redirector.
> :::

## Εισαγωγή Κανόνων

### Εισαγωγή σε macOS

- Απλώς σύρετε και αποθέστε τα αρχεία κανόνων που εξήχθησαν στο παράθυρο του Redirect Web.
- Ή, μπορείτε επίσης να τα εισαγάγετε από τη γραμμή μενού > Αρχείο > Εισαγωγή Κανόνων

### Εισαγωγή σε iOS

Ας υποθέσουμε ότι το αρχείο κανόνων σας βρίσκεται στην εφαρμογή Αρχεία. Ανοίξτε το στην εφαρμογή Αρχεία, πατήστε το κουμπί Κοινή χρήση και, στη συνέχεια, επιλέξτε Redirect Web:

| Βήμα 1: Πατήστε το κουμπί Κοινή χρήση στην εφαρμογή Αρχεία | Βήμα 2: Επιλέξτε Redirect Web         |

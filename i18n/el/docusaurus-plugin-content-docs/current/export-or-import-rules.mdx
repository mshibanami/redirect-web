# Εξαγωγή/Εισαγωγή Κανόνων

## Εξαγωγή Κανόνων {#export-rules}

### Εξαγωγή σε macOS {#export-on-macos}

Κάντε κλικ είτε στο κουμπί αποσιωπητικών είτε στο κουμπί Κοινοποίηση, όπως φαίνεται σε αυτό το στιγμιότυπο οθόνης:

![Share on macOS](/img/share-on-macos.png)

Μπορείτε επίσης να επιλέξετε πολλούς κανόνες και να τους εξαγάγετε κάνοντας δευτερεύον κλικ (δεξί κλικ) σε αυτούς στην πλαϊνή γραμμή λίστας κανόνων:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Επιπλέον, μπορείτε να εξαγάγετε όλους τους κανόνες από τη γραμμή μενού > Αρχείο > Εξαγωγή Όλων των Κανόνων:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Αν θέλετε να εξαγάγετε τους κανόνες σας σε μορφή συμβατή με το [Redirector του Einar Egilsson](https://einaregilsson.com/redirector/), πατήστε το πλήκτρο Option (`⌥`) στο μενού περιβάλλοντος ή στη γραμμή μενού, και τότε θα δείτε τα εξής:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Το Redirect Web δεν είναι πλήρως συμβατό με το Redirector, και αντίστροφα. Ορισμένες ρυθμίσεις, όπως το "Replace Occurrences" για την Επεξεργασία Ομάδων Καταγραφής, δεν εξάγονται με τη μορφή του Redirector.
> :::

## Εισαγωγή Κανόνων {#import-rules}

### Εισαγωγή σε macOS {#import-on-macos}

- Απλώς σύρετε και αποθέστε τα εξαγόμενα αρχεία κανόνων στο παράθυρο του Redirect Web.
- Ή, μπορείτε επίσης να τα εισαγάγετε από τη γραμμή μενού > Αρχείο > Εισαγωγή Κανόνων

### Εισαγωγή σε iOS {#import-on-ios}

Υποθέστε ότι το αρχείο κανόνων σας βρίσκεται στην εφαρμογή Αρχεία (Files). Ανοίξτε το στα Αρχεία, πατήστε το κουμπί Κοινοποίηση, και μετά επιλέξτε Redirect Web:

| Βήμα 1: Πατήστε το κουμπί Κοινοποίηση στην εφαρμογή Αρχεία                     | Βήμα 2: Επιλέξτε Redirect Web                                     |
| ----------------------------------------------------------------------------- | ----------------------------------------------------------------- |
| ![Tap the share button in the toolbar](/img/files-share-ios.png) | ![Tap the app icon in the sheet](/img/share-sheet-ios.png) |

# Εξαγωγή/Εισαγωγή Κανόνων

## Εξαγωγή Κανόνων {#export-rules}

### Εξαγωγή σε macOS {#export-on-macos}

Κάντε κλικ είτε στο κουμπί με τις ελλείψεις είτε στο κουμπί Κοινή χρήση όπως φαίνεται σε αυτό το στιγμιότυπο οθόνης:

<Image src={ShareOnMacOS} alt="Κουμπί Κοινή χρήση σε macOS" width="460" hasShadow hasRoundedCorners/>

Στη συνέχεια, επιλέξτε "Εξαγωγή Κανόνα" ή "Κοινή χρήση Κανόνα..." από το μενού περιβάλλοντος:

Μπορείτε επίσης να επιλέξετε πολλούς κανόνες και να τους εξαγάγετε κάνοντας δευτερεύον κλικ (δεξί κλικ) σε αυτούς στην πλευρική γραμμή λίστας κανόνων:

<Image src={MultiSelectOnMacOS} alt="Κοινή χρήση πολλαπλών κανόνων σε macOS" height="400" hasShadow hasRoundedCorners/>


Επιπλέον, μπορείτε να εξαγάγετε όλους τους κανόνες από τη γραμμή μενού > Αρχείο > Εξαγωγή Όλων των Κανόνων:

<Image src={MenubarFile} alt="Γραμμή μενού > Αρχείο" width="400" hasShadow hasRoundedCorners/>

> [!NOTE]
> Εάν θέλετε να εξαγάγετε τους κανόνες σας σε μορφή συμβατή με το [Redirector του Einar Egilsson](https://einaregilsson.com/redirector/), πατήστε το πλήκτρο Option (`⌥`) στο μενού περιβάλλοντος ή στη γραμμή μενού, τότε θα δείτε αυτά:
> <Image src={MenubarFileRedirector} alt="Γραμμή μενού > Αρχείο (Redirector)" width="400" hasShadow hasRoundedCorners/>
> <br/>
> <Image src={ContextMenuRedirector} alt="Μενού περιβάλλοντος (Redirector)" width="400" hasShadow hasRoundedCorners/>
>
> :::warning
> Το Redirect Web δεν είναι πλήρως συμβατό με το Redirector, και το αντίστροφο. Ορισμένες ρυθμίσεις, όπως το "Αντικατάσταση Εμφανίσεων" για την Επεξεργασία Ομάδας Καταγραφής, δεν εξάγονται με τη μορφή του Redirector.
> :::

## Εισαγωγή Κανόνων {#import-rules}

### Εισαγωγή σε macOS {#import-on-macos}

- Απλώς σύρετε και αποθέστε τα εξαγόμενα αρχεία κανόνων στο παράθυρο του Redirect Web.
- Ή, μπορείτε επίσης να τα εισαγάγετε από τη γραμμή μενού > Αρχείο > Εισαγωγή Κανόνων

### Εισαγωγή σε iOS {#import-on-ios}

Ας υποθέσουμε ότι το αρχείο κανόνων σας βρίσκεται στην εφαρμογή Αρχεία. Ανοίξτε το στα Αρχεία, πατήστε το κουμπί Κοινή χρήση και, στη συνέχεια, επιλέξτε Redirect Web:

| Βήμα 1: Πατήστε το κουμπί Κοινή χρήση στα Αρχεία                                                                          | Βήμα 2: Επιλέξτε Redirect Web                                                                              |

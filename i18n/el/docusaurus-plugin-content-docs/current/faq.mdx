# Συχνές Ερωτήσεις

## Γιατί πρέπει να επιτρέψω στην επέκταση να διαβάζει/αλλάζει τις πληροφορίες μου;

Ενδέχεται να δείτε μια ειδοποίηση σαν αυτή όταν ρυθμίζετε το Redirect Web:

<IfBrowserStore stores='appStore'>
    <Image src={PermissionAlertSafari} alt="Μια ειδοποίηση που απαιτεί άδεια" width="200" margin="0 0 16px 0" hasShadow={true} />

    > Αυτή η επέκταση θα μπορεί να διαβάζει και να τροποποιεί ιστοσελίδες και να βλέπει το ιστορικό περιήγησής σας σε αυτόν τον ιστότοπο. **Αυτό μπορεί να περιλαμβάνει ευαίσθητες πληροφορίες, συμπεριλαμβανομένων κωδικών πρόσβασης, αριθμών τηλεφώνου και πιστωτικών καρτών.**
</IfBrowserStore>

<IfBrowserStore stores='appStore' includes={false}>
    <Image src={PermissionAlertChrome} alt="Μια ειδοποίηση που απαιτεί άδεια" height="200" hasRoundedCorners={true} hasShadow={true} margin="0 0 16px 0" />

    > Μπορεί:  
    > Να διαβάζει και να αλλάζει **όλα σας τα δεδομένα** σε **όλους τους ιστότοπους**
</IfBrowserStore>

Αυτή η επέκταση χρειάζεται την άδειά σας για να παρακολουθεί την πλοήγηση σελίδας, προκειμένου να ανακατευθύνει όπου θέλετε να μεταβείτε, και το πρόγραμμα περιήγησής σας εμφανίζει αυτό όταν η επέκταση ζητά τις σχετικές άδειες. Επομένως, αν δεν το επιτρέψετε, αυτή η επέκταση δεν μπορεί να λειτουργήσει.

Οι παραχωρηθείσες άδειες θα χρησιμοποιούνται μόνο για λειτουργίες της εφαρμογής, όπως η ανακατεύθυνση, σε συμμόρφωση με την **[πολιτική απορρήτου](./privacy-policy)**.

## Πρέπει να λάβω την εφαρμογή αφού εγκαταστήσω την επέκταση για Chrome, Firefox ή Edge; \{#app-for-non-safari-browsers}

Ναι. Πρέπει να κατεβάσετε [την εφαρμογή Redirect Web από το App Store](https://apps.apple.com/app/id1571283503) ανεξάρτητα από το πρόγραμμα περιήγησης που χρησιμοποιείτε. Μπορείτε να εγκαταστήσετε την επέκταση από το κατάστημα επεκτάσεων του προγράμματος περιήγησής σας, αλλά η εφαρμογή απαιτείται για τη διαχείριση των κανόνων σας, του ιστορικού αγορών και άλλων ρυθμίσεων σε όλες τις συσκευές μέσω του iCloud.

## Λειτουργεί το Redirect Web σε Windows, Android, Linux ή οποιαδήποτε άλλη πλατφόρμα; \{#non-apple-platforms}

Όχι αυτή τη στιγμή. Το Redirect Web είναι στενά συνδεδεμένο με το οικοσύστημα της Apple, συμπεριλαμβανομένων των iCloud και Universal Purchase, γεγονός που καθιστά δύσκολη την προσφορά του.

## Γιατί όχι το [Redirector](https://github.com/einaregilsson/Redirector);

Το Redirector εξακολουθεί να είναι μια εξαιρετική επέκταση ανακατεύθυνσης. Είναι εντελώς δωρεάν, ανοιχτού κώδικα και λειτουργεί και σε πλατφόρμες εκτός Apple. Αλλά το Redirect Web έχει μερικά επιπλέον χαρακτηριστικά που μπορεί να σας αρέσουν:

*   **Λεπτομερείς Ρυθμίσεις Κανόνων:** Πολλαπλά μοτίβα εξαίρεσης, πολλαπλές αντιστοιχίσεις διεργασιών και λειτουργίες αντικατάστασης συμβολοσειρών διαθέσιμες μόνο στο Redirect Web.
*   **Άνοιγμα σε εγγενείς εφαρμογές:** Σε Mac, μπορείτε να επιλέξετε πού να ανοίξει η διεύθυνση URL προορισμού. (Καλεί την εντολή `open` απευθείας. Όχι deep links.)
*   **Υποστήριξη [DNR](https://blog.mozilla.org/addons/2023/05/17/declarativenetrequest-available-in-firefox/):** Παρέχει απίστευτα γρήγορη ανακατεύθυνση.
*   **Κεντρική διαχείριση κανόνων:** Διαχειριστείτε κανόνες για όλα τα προγράμματα περιήγησης σε ένα μέρος μέσω της εγγενούς εφαρμογής.
*   **Υποστήριξη Manifest V3 & V2:** Χρήσιμο ενώ το V2 σταδιακά καταργείται στο Chrome.

## Γιατί αυτή η επέκταση απαιτεί νέα άδεια; \{#permission-alert-on-safari}

Ενδέχεται να δείτε αυτήν την ειδοποίηση στο Safari μετά την ενημέρωση του Redirect Web:

<img src={require("/img/safari-additional-permission-alert.webp").default} alt="Μια ειδοποίηση απενεργοποίησης της εφαρμογής" width="280" />

> Το "Redirect Web" έχει ενημερωθεί και ζητά πρόσθετες άδειες. Για την προστασία του απορρήτου σας, αυτή η επέκταση έχει απενεργοποιηθεί.
>
> Μπορείτε να ενεργοποιήσετε αυτήν την επέκταση στην ενότητα Επεκτάσεις των Ρυθμίσεων του Safari.

Αυτό οφείλεται στην άδεια [declarativeNetRequestWithHostAccess](https://developer.apple.com/documentation/safariservices/safari_web_extensions/blocking_content_with_your_safari_web_extension) που απαιτείται από τον τύπο [DNR](./rule-settings#type) που εισήχθη στην έκδοση 7. Παρακαλούμε επανενεργοποιήστε την επέκταση. Συγγνώμη για την αναστάτωση.

## Θα γίνουν επί πληρωμή τα περιεχόμενα της βιβλιοθήκης στο μέλλον;

Όχι, είναι πάντα δωρεάν. Μπορείτε επίσης να βρείτε την πηγή όλων των περιεχομένων [εδώ](https://github.com/mshibanami/redirect-web/tree/main/library).

## Εφάπαξ Αγορά ή Ετήσια Συνδρομή: Ποιο είναι το καλύτερο;

Συνιστούμε την εφάπαξ αγορά, καθώς η διαφορά τιμής είναι μόνο 2,00 US$.
Ο κύριος λόγος που προσφέρουμε τη συνδρομή είναι για να σας επιτρέψουμε να δοκιμάσετε όλες τις λειτουργίες μέσω της δωρεάν δοκιμής της. Ακολουθεί η αναμενόμενη περίπτωση χρήσης:

1.  Εγγραφείτε στην ετήσια συνδρομή.
2.  Ακυρώστε αμέσως τη συνδρομή μεταβαίνοντας στην εφαρμογή Ρυθμίσεις, αναζητώντας "Συνδρομές", πατώντας σε αυτήν την εφαρμογή και επιλέγοντας "Ακύρωση Συνδρομής". Αυτό σας επιτρέπει να απολαύσετε τη δωρεάν δοκιμή χωρίς καμία δέσμευση.
3.  Απολαύστε πλήρη πρόσβαση σε όλες τις λειτουργίες για 7 ημέρες.
4.  Αν σας αρέσουν οι λειτουργίες, επιλέξτε την εφάπαξ αγορά.

Φυσικά, δεν υπάρχει τίποτα κακό να συνεχίσετε με την ετήσια συνδρομή αν το προτιμάτε!

## Εισάγει το Redirect Web διαφημίσεις ή άλλο άσχετο περιεχόμενο σε ιστοσελίδες;

Όχι, δεν εισάγει τέτοιο περιεχόμενο. Μόνο κώδικας που είναι απαραίτητος για την παροχή των λειτουργιών που περιγράφονται σε αυτήν την τεκμηρίωση και στη σελίδα της εφαρμογής στο App Store εισάγεται σε ιστοσελίδες στο Safari.

## Υπάρχει περίπτωση το Redirect Web να πουληθεί σε κάποιον άλλο;

100% όχι. Οι επεκτάσεις των προγραμμάτων περιήγησης ασκούν βαθύ έλεγχο στην περιήγηση και τα δεδομένα σας, και η παράδοση του Redirect Web σε άλλον ιδιοκτήτη θα άνοιγε την πόρτα σε παραβιάσεις απορρήτου. Έχουμε δει επεκτάσεις να αλλάζουν χέρια και στη συνέχεια να συλλέγουν δεδομένα χρηστών ή να εισάγουν διαφημίσεις χωρίς προειδοποίηση. Αυτό δεν θα συμβεί ποτέ εδώ. Για περισσότερα σχετικά με αυτούς τους κινδύνους, δείτε το άρθρο του Matt Frisbie ["The Ugly Business of Monetizing Browser"](https://mattfrisbie.substack.com/p/the-ugly-business-of-monetizing-browser) και την έρευνα του Brian Krebs ["Is Your Browser Extension a Botnet Backdoor?"](https://krebsonsecurity.com/2021/03/is-your-browser-extension-a-botnet-backdoor/).

Το Redirect Web βασίζεται στην εμπιστοσύνη και την υπευθυνότητα. Θεωρούμε καθήκον μας να διατηρούμε ασφαλείς τις πληροφορίες σας και αρνούμαστε να μεταβιβάσουμε την ιδιοκτησία υπό οποιεσδήποτε συνθήκες. Το Redirect Web θα παραμείνει πάντα στα χέρια μας, ώστε να μπορείτε να περιηγείστε με απόλυτη σιγουριά.

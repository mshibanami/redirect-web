Καλώς ήρθατε στον ιστότοπο υποστήριξης για το Redirect Web, μια επέκταση προγράμματος περιήγησης που ανακατευθύνει όλες τις φορτώσεις σελίδων στο πρόγραμμα περιήγησής σας, διασφαλίζοντας ότι προσγειώνεστε πάντα εκεί που θέλετε. 

(Παλαιότερα γνωστό ως Redirect Web για Safari.)

## ⬇️ Σύνδεσμοι λήψης {#download-links}

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chrome Web Store)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Πρόσθετα Firefox)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Πρόσθετα Microsoft Edge)
- [Safari](https://apps.apple.com/app/id1571283503) (Η επέκταση είναι ήδη ενσωματωμένη στην εγγενή εφαρμογή.)

Το Redirect Web είναι διαθέσιμο σε:

- **Mac** (Safari, Chrome, Firefox, Edge, και άλλα)
- **iPhone**, **iPad**, και **Apple Vision Pro** (Safari)

:::note

- Πρέπει να εγκαταστήσετε [την εφαρμογή Redirect Web από το Mac App Store](https://apps.apple.com/app/id1571283503) για να χρησιμοποιήσετε την επέκταση, ανεξάρτητα από το πρόγραμμα περιήγησης που χρησιμοποιείτε, επειδή οι επεκτάσεις βασίζονται στην εφαρμογή για πρόσβαση στους κανόνες, τις ρυθμίσεις σας κ.λπ.
- Η υποστήριξη πολλαπλών προγραμμάτων περιήγησης είναι διαθέσιμη μόνο σε Mac.

:::

## Δημοφιλείς περιπτώσεις χρήσης {#popular-use-cases}

Για παράδειγμα, μπορείτε να δημιουργήσετε αυτούς τους κανόνες:

- Προβολή **Reddit** στην κλασική του σχεδίαση (old.reddit.com).
- Άνοιγμα **Apple Maps** κάθε φορά που εμφανίζεται ένας σύνδεσμος **Google Maps**.
- Απόκρυψη των επισκοπήσεων AI της Google από τα **αποτελέσματα αναζήτησης Google** προσθέτοντας **?udm=14** σε κάθε αναζήτηση.
- Μετάβαση από σελίδες Fandom στα επίσημα wikis **Terraria** ή **Minecraft**.
- Χρήση **Kagi**, **Startpage**, **ChatGPT**, και οποιασδήποτε άλλης μηχανής αναζήτησης ως την **προεπιλεγμένη μηχανή αναζήτησής** σας.
- Άνοιγμα συνδέσμων **Google Meet** στο Google Chrome. (Mac)
- Άνοιγμα συνδέσμων **Figma** απευθείας στην εφαρμογή Figma. (Mac)
- Άνοιγμα συνδέσμων **Notion** απευθείας στην εφαρμογή Notion. (Mac)

Μπορείτε να αποκτήσετε όλους αυτούς τους προκαθορισμένους κανόνες δωρεάν από την ενότητα Βιβλιοθήκη στην εφαρμογή!

## Χαρακτηριστικά {#features}

Το Redirect Web διαθέτει τις ακόλουθες λειτουργίες:

- Δημιουργία κανόνων χρησιμοποιώντας τύπους μοτίβων **Κανονικής Έκφρασης** ή **Μπαλαντέρ**
- Χρήση **αναφοράς προς τα πίσω** για τον καθορισμό δυναμικών URL προορισμού
- **Εξαίρεση** URL από την ανακατεύθυνση
- **Επεξεργασία καταγεγραμμένων κειμένων** χρησιμοποιώντας κωδικοποίηση/αποκωδικοποίηση URL, κωδικοποίηση/αποκωδικοποίηση base64 ή αντικατάσταση εμφανίσεων

Επιπλέον, μπορείτε να ξεκλειδώσετε τις ακόλουθες λειτουργίες μέσω συνδρομής ή εφάπαξ αγοράς:

- Ενεργοποίηση **απεριόριστου** αριθμού κανόνων
- **Συγχρονισμός κανόνων** με τα άλλα Mac, iPhone και iPad **μέσω iCloud**

(Η αγορά της εφαρμογής μας υποστηρίζει επίσης στη διατήρηση του Redirect Web. 🙏)

Η αγορά ή η συνδρομή σας μοιράζεται σε Mac, iPhone, iPad και Apple Vision Pro μέσω Universal Purchase. Η υποστήριξή σας βοηθά στη συνεχή ανάπτυξη.

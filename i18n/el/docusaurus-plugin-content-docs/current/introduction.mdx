Καλώς ήρθατε στον ιστότοπο υποστήριξης του Redirect Web, μιας επέκτασης προγράμματος περιήγησης που ανακατευθύνει όλες τις φορτώσεις σελίδων στο πρόγραμμα περιήγησής σας, διασφαλίζοντας ότι προσγειώνεστε πάντα εκεί που θέτετε.

(Πρώην γνωστό ως Redirect Web for Safari.)

## ⬇️ Σύνδεσμοι λήψης

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chrome Web Store)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Πρόσθετα Firefox)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Πρόσθετα Microsoft Edge)
- [Safari](https://apps.apple.com/app/id1571283503) (Η επέκταση περιλαμβάνεται ήδη στην εγγενή εφαρμογή.)

Το Redirect Web είναι διαθέσιμο σε:

- **Mac** (Safari, Chrome, Firefox, Edge και άλλα)
- **iPhone**, **iPad**, και **Apple Vision Pro** (Safari)

:::note

- Πρέπει να εγκαταστήσετε [την εφαρμογή Redirect Web από το Mac App Store](https://apps.apple.com/app/id1571283503) για να χρησιμοποιήσετε την επέκταση, ανεξάρτητα από το πρόγραμμα περιήγησης που χρησιμοποιείτε, επειδή οι επεκτάσεις βασίζονται στην εφαρμογή για την πρόσβαση στους κανόνες, τις ρυθμίσεις σας κ.λπ.
- Η υποστήριξη πολλαπλών προγραμμάτων περιήγησης είναι διαθέσιμη μόνο σε Mac.

:::

## Δημοφιλείς Περιπτώσεις Χρήσης

Για παράδειγμα, μπορείτε να δημιουργήσετε αυτούς τους κανόνες:

- Προβολή του **Reddit** στον κλασικό του σχεδιασμό (old.reddit.com).
- Άνοιγμα **Apple Maps** κάθε φορά που εμφανίζεται ένας σύνδεσμος **Google Maps**.
- Απόκρυψη των AI Overviews της Google από τα **αποτελέσματα αναζήτησης Google** προσθέτοντας **?udm=14** σε κάθε αναζήτηση.
- Μετάβαση από σελίδες Fandom στα επίσημα wikis του **Terraria** ή του **Minecraft**.
- Χρήση **Kagi**, **Startpage**, **ChatGPT** και οποιασδήποτε άλλης μηχανής αναζήτησης ως **προεπιλεγμένη μηχανή αναζήτησής** σας.
- Άνοιγμα συνδέσμων **Google Meet** στο Google Chrome. (Mac)
- Άνοιγμα συνδέσμων **Figma** απευθείας στην εφαρμογή Figma. (Mac)
- Άνοιγμα συνδέσμων **Notion** απευθείας στην εφαρμογή Notion. (Mac)

Μπορείτε να λάβετε όλα αυτά ως προκαθορισμένους κανόνες δωρεάν από την ενότητα Βιβλιοθήκη της εφαρμογής!

## Λειτουργίες

Το Redirect Web διαθέτει τις ακόλουθες λειτουργίες:

- Δημιουργία κανόνων χρησιμοποιώντας τύπους μοτίβων **Κανονικής Έκφρασης** ή **Μπαλαντέρ** (Wildcard)
- Χρήση **αντίστροφης αναφοράς** για τον καθορισμό δυναμικών URL προορισμού
- **Εξαίρεση** URL από την ανακατεύθυνση
- **Επεξεργασία καταγεγραμμένων κειμένων** χρησιμοποιώντας κωδικοποίηση/αποκωδικοποίηση URL, κωδικοποίηση/αποκωδικοποίηση base64 ή αντικατάσταση εμφανίσεων

Επιπλέον, μπορείτε να ξεκλειδώσετε τις ακόλουθες λειτουργίες μέσω συνδρομής ή εφάπαξ αγοράς:

- Ενεργοποίηση **απεριόριστου** αριθμού κανόνων
- **Συγχρονισμός κανόνων** με τα άλλα Mac, iPhone και iPad σας **μέσω iCloud**

(Η αγορά της εφαρμογής μας υποστηρίζει επίσης στη συντήρηση του Redirect Web. 🙏)

Η αγορά ή η συνδρομή σας κοινοποιείται σε Mac, iPhone, iPad και Apple Vision Pro μέσω Universal Purchase. Η υποστήριξή σας βοηθά στην συνεχή ανάπτυξη.

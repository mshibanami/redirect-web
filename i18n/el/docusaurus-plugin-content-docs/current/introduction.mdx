Το Redirect Web είναι διαθέσιμο για:

- **Mac** (Safari, Chrome, Firefox, Edge, and more)
- **iPhone**, **iPad**, και **Apple Vision Pro** (Safari)

:::note

- Πρέπει να εγκαταστήσετε [την εφαρμογή Redirect Web από το Mac App Store](https://apps.apple.com/app/id1571283503) για να χρησιμοποιήσετε την επέκταση, ανεξάρτητα από τον περιηγητή που χρησιμοποιείτε, επειδή οι επεκτάσεις βασίζονται στην εφαρμογή για πρόσβαση στους κανόνες, τις ρυθμίσεις κ.λπ.
- Η υποστήριξη πολλαπλών προγραμμάτων περιήγησης είναι διαθέσιμη μόνο σε Mac.

:::

## Δημοφιλείς Χρήσεις {#popular-use-cases}

Για παράδειγμα, μπορείτε να δημιουργήσετε αυτούς τους κανόνες:

- Προβολή **Reddit** με την κλασική σχεδίαση (old.reddit.com).
- Άνοιγμα **Apple Maps** κάθε φορά που εμφανίζεται ένας σύνδεσμος **Google Maps**.
- Απόκρυψη των Google’s AI Overviews από **Google search results** προσθέτοντας **?udm=14** σε κάθε αναζήτηση.
- Μετάβαση από σελίδες Fandom στους επίσημους wikis του **Terraria** ή του **Minecraft**.
- Χρήση των **Kagi**, **Startpage**, **ChatGPT**, και οποιασδήποτε άλλης μηχανής αναζήτησης ως την **προεπιλεγμένη μηχανή αναζήτησης** σας.
- Άνοιγμα συνδέσμων **Google Meet** στο Google Chrome. (Mac)
- Άνοιγμα συνδέσμων **Figma** απευθείας στην εφαρμογή Figma. (Mac)
- Άνοιγμα συνδέσμων **Notion** απευθείας στην εφαρμογή Notion. (Mac)

Μπορείτε να λάβετε όλα αυτά ως προκαθορισμένους κανόνες δωρεάν από την ενότητα Library στην εφαρμογή!

## Χαρακτηριστικά {#features}

Το Redirect Web διαθέτει τα εξής χαρακτηριστικά:

- Δημιουργία κανόνων χρησιμοποιώντας τύπους προτύπων **Regular Expression** ή **Wildcard**
- Χρήση **back-referencing** για να καθορίσετε δυναμικές διευθύνσεις URL προορισμού
- **Exclude** URLs από την ανακατεύθυνση
- **Process captured texts** χρησιμοποιώντας κωδικοποίηση/αποκωδικοποίηση URL, κωδικοποίηση/αποκωδικοποίηση base64 ή αντικατάσταση εμφανίσεων

Επιπλέον, μπορείτε να ξεκλειδώσετε τα παρακάτω χαρακτηριστικά μέσω συνδρομής ή μιας εφάπαξ αγοράς:

- Ενεργοποίηση **απεριόριστου** αριθμού κανόνων
- **Syncing rules** με τα υπόλοιπα Mac, iPhone, και iPad σας **μέσω iCloud**

(Η αγορά της εφαρμογής επίσης μας βοηθά στη συντήρηση του Redirect Web. 🙏)

Η αγορά ή η συνδρομή σας μοιράζεται σε Mac, iPhone, iPad, και Apple Vision Pro μέσω Universal Purchase. Η υποστήριξή σας βοηθά στη συνεχιζόμενη ανάπτυξη.

Καλώς ήρθατε στη βιβλιοθήκη μας, όπου μπορείτε να βρείτε χρήσιμους έτοιμους κανόνες δωρεάν! Μπορείτε να τους αποκτήσετε πρόσβαση μέσω της εφαρμογής Redirect Web.

![Get Button - iOS](/img/library-get-button-ios.gif)

## Συμβάλετε τον κανόνα σας στη βιβλιοθήκη

Η τρέχουσα βιβλιοθήκη έχει μόνο λίγους κανόνες, οπότε εκτιμούμε τη συμβολή σας!

Υπάρχουν 2 επιλογές για να συμβάλετε τους κανόνες σας στη βιβλιοθήκη:

1. **Δημοσιεύστε τον κανόνα σας στο Forum**: Μοιραστείτε τον κανόνα ανακατεύθυνσης με άλλους χρήστες στο [forum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Στη συνέχεια, ενδέχεται να σας ζητήσουμε αν μπορούμε να συμπεριλάβουμε τον κανόνα σας και στη βιβλιοθήκη. Αυτό μπορεί να είναι ευκολότερο από την επιλογή 2, επειδή μπορείτε να αφήσετε την προσθήκη σε εμάς.
2. **Στείλτε μας ένα pull request**: Μπορείτε να μας στείλετε ένα pull request. Παρακαλούμε ελέγξτε τις [οδηγίες](https://github.com/mshibanami/redirect-web/tree/main/library) για να μάθετε περισσότερα.

## Άλλες Πληροφορίες

- Μπορείτε να δείτε όλα τα περιεχόμενα στο [αποθετήριο στο GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- Φιλοξενούμε τον ιδιωτικό μας API server στο Cloudflare για τα περιεχόμενα της βιβλιοθήκης, επιτρέποντάς μας να παρέχουμε φιλτράρισμα και άλλες υπηρεσίες. Η εφαρμογή ανακτά τα περιεχόμενα της βιβλιοθήκης από αυτόν τον server. Όπως αναφέρεται στην [πολιτική απορρήτου μας](privacy-policy.mdx), αυτός ο server δεν συλλέγει τα προσωπικά σας δεδομένα.

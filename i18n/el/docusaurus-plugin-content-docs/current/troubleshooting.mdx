# Αντιμετώπιση προβλημάτων

Αυτή η σελίδα παραθέτει κοινά προβλήματα και τις λύσεις τους κατά τη χρήση του Redirect Web.

<TOCInline toc={toc} />

## Οι κανόνες ανακατεύθυνσης δεν λειτουργούν

Εάν οι κανόνες σας δεν λειτουργούν όπως αναμένεται, παρακαλούμε ελέγξτε τα ακόλουθα:

- **Η επέκταση του προγράμματος περιήγησης είναι εγκατεστημένη και ενεργοποιημένη για τη σωστή σελίδα:** Βεβαιωθείτε ότι έχετε εγκαταστήσει την επέκταση προγράμματος περιήγησης Redirect Web και ότι είναι ενεργοποιημένη στο πρόγραμμα περιήγησής σας για την ιστοσελίδα από την οποία θέλετε να ανακατευθύνετε. Εάν μπορείτε να ανοίξετε το αναδυόμενο παράθυρο της επέκτασης όπως αυτό στην αρχική ιστοσελίδα, είναι εγκατεστημένη σωστά:
    <Image src={ExtensionPopup} alt="Αναδυόμενο παράθυρο επέκτασης Redirect Web" width="260" hasShadow/>

- **Η διαμόρφωση του κανόνα είναι σωστή:** Ελέγξτε ξανά τους κανόνες ανακατεύθυνσης στην εγγενή εφαρμογή Redirect Web. Βεβαιωθείτε ότι τα μοτίβα URL προέλευσης, οι URL προορισμού και τυχόν συνθήκες έχουν ρυθμιστεί σωστά. Για να ελέγξετε αν ο κανόνας σας είναι έγκυρος ή όχι, εισαγάγετε ένα παράδειγμα URL στην ενότητα Παραδείγματα στην οθόνη Επεξεργασία Κανόνα. Εάν ο κανόνας σας είναι έγκυρος, θα δείτε την αναμενόμενη URL προορισμού κάτω από την εισαγωγή σας, όπως αυτό:
    <Image src={RuleEditExampleResult} alt="Ένα παράδειγμα οθόνης επεξεργασίας κανόνα που δείχνει το αποτέλεσμα του παραδείγματος URL" width="260" hasShadow/>

## Το αναδυόμενο παράθυρο της επέκτασης εμφανίζει ένα σφάλμα

<Image src={ErrorOverlay} alt="Επικάλυψη σφάλματος Redirect Web" width="260" hasShadow hasRoundedCorners/>

> Η εφαρμογή Redirect Web δεν έχει εγκατασταθεί ή δεν λειτουργεί σωστά.
> Παρακαλούμε αποκτήστε την εφαρμογή από το App Store.

Εάν δείτε το παραπάνω σφάλμα κατά το άνοιγμα του αναδυόμενου παραθύρου της επέκτασης, σημαίνει ότι η εγγενής εφαρμογή Redirect Web δεν είναι εγκατεστημένη ή δεν λειτουργεί σωστά.

Βεβαιωθείτε ότι έχετε εγκαταστήσει την τελευταία έκδοση του Redirect Web από το [App Store](https://apps.apple.com/app/id1571283503).

Ίσως χρειαστεί επίσης να επανεκκινήσετε το πρόγραμμα περιήγησής σας μετά την εγκατάσταση της εγγενούς εφαρμογής.

> [!NOTE]
> * Πρέπει να εγκαταστήσετε την εγγενή εφαρμογή Redirect Web για να χρησιμοποιήσετε την επέκταση του προγράμματος περιήγησης. Η επέκταση μόνη της δεν λειτουργεί.
> * Οι χρήστες του Safari δεν χρειάζεται να εγκαταστήσουν την επέκταση ξεχωριστά, καθώς είναι ενσωματωμένη στην εγγενή εφαρμογή.

## Η αρχική σελίδα αναβοσβήνει στιγμιαία πριν την ανακατεύθυνση

Αυτό συμβαίνει όταν χρησιμοποιείτε τους κανόνες ανακατεύθυνσης τύπου **Original**. Αυτός ο τύπος παρέχει διάφορες προηγμένες λειτουργίες, αλλά μπορεί να προκαλέσει ένα σύντομο αναβόσβημα της αρχικής σελίδας επειδή μπορεί να επιτρέψει στην αρχική σελίδα να φορτώσει πριν πραγματοποιήσει την ανακατεύθυνση.

Ανοίξτε την οθόνη Επεξεργασία Κανόνα και αλλάξτε τον **Τύπο** σε **DNR** για να αποφύγετε αυτό το πρόβλημα.

<Image src={EditRuleDnr} alt="Αναδυόμενο παράθυρο επέκτασης Redirect Web" height="260" hasShadow hasRoundedCorners/>

> [!NOTE]
> Στον Firefox, αυτό το πρόβλημα δεν θα πρέπει να εμφανίζεται ακόμη και με τον τύπο Original, καθώς βασίζεται στο WebRequest API, το οποίο αποτρέπει τη φόρτωση της αρχικής σελίδας.

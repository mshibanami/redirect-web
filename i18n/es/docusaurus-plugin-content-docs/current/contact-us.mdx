# Contáctanos

Estamos aquí para ayudarte si tienes alguna pregunta, comentario o problema con Redirect Web. Estas son las mejores formas de ponerte en contacto:

- **[Problemas de GitHub](https://github.com/mshibanami/RedirectWeb/issues)**: Si encuentras algún error o quieres solicitar nuevas funciones, visita la página de Problemas de GitHub. Puedes buscar problemas existentes, enviar un nuevo problema y seguir el progreso de tu problema.
- **[Discusiones de GitHub](https://github.com/mshibanami/RedirectWeb/discussions)**: Si tienes alguna pregunta o quieres obtener ayuda del desarrollador o de otros usuarios, visita la página de Discusiones de GitHub. Puedes iniciar una nueva discusión, unirte a discusiones existentes y obtener ayuda de la comunidad.
- **Formulario de comentarios**: Puedes enviarnos tus comentarios a través del formulario de comentarios en la aplicación Redirect Web. Esto enviará un correo electrónico directamente al desarrollador, por lo que tus comentarios se mantendrán privados de otros usuarios.

¡Gracias por usar Redirect Web! Agradecemos tu apoyo y comentarios.

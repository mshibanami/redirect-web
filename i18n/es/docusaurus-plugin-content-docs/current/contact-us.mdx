# Contacto

Estamos aquí para ayudarte si tienes alguna pregunta, comentario o problema con Redirect Web. Estas son las mejores formas de ponerte en contacto:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Si encuentras algún error o quieres solicitar nuevas funciones, visita la página de GitHub Issues. Puedes explorar los problemas existentes, enviar un nuevo problema y seguir su progreso.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Si tienes alguna pregunta o quieres obtener ayuda del desarrollador o de otros usuarios, visita la página de GitHub Discussions. Puedes iniciar una nueva discusión, unirte a las existentes y obtener ayuda de la comunidad.
- **Formulario de comentarios**: Puedes enviarnos tus comentarios a través del formulario de comentarios en la aplicación Redirect Web. Esto enviará un correo electrónico directamente al desarrollador, por lo que tus comentarios se mantendrán privados de otros usuarios.

¡Gracias por usar Redirect Web! Agradecemos tu apoyo y tus comentarios.

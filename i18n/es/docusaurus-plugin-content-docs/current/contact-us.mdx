# Contáctenos

Estamos aquí para ayudarle si tiene alguna pregunta, comentario o problema con Redirect Web. Estas son las mejores formas de ponerse en contacto:

- **[Problemas de GitHub](https://github.com/mshibanami/RedirectWeb/issues)**: Si encuentra algún error o desea solicitar nuevas funciones, visite la página de Problemas de GitHub. Puede buscar problemas existentes, enviar un nuevo problema y seguir el progreso de su problema.
- **[Discusiones de GitHub](https://github.com/mshibanami/RedirectWeb/discussions)**: Si tiene alguna pregunta o desea obtener ayuda del desarrollador o de otros usuarios, visite la página de Discusiones de GitHub. Puede iniciar una nueva discusión, unirse a discusiones existentes y obtener ayuda de la comunidad.
- **Formulario de comentarios**: Puede enviarnos sus comentarios a través del formulario de comentarios en la aplicación Redirect Web. Esto enviará un correo electrónico directamente al desarrollador, por lo que sus comentarios se mantendrán privados de otros usuarios.

¡Gracias por usar Redirect Web! Agradecemos su apoyo y comentarios.

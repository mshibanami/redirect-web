# Biblioteca

¡Bienvenido a nuestra biblioteca, donde puedes encontrar reglas útiles y listas para usar de forma gratuita! Puedes acceder a ellas a través de la aplicación Redirect Web.

![Library Screenshot](/img/library-screenshot.webp)

Para copiar las reglas a Mis Reglas, simplemente selecciona el botón Obtener:

![Get Button - iOS](/img/library-get-button-ios.gif)

## Contribuye con tu regla a la biblioteca

La biblioteca actual tiene solo unas pocas reglas, ¡así que agradecemos tu contribución!

Hay 2 opciones para contribuir con tus reglas a la biblioteca:

1.  **Publica tu regla en el Foro**: Comparte tu regla de redireccionamiento con otros usuarios en [el foro](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Luego, podríamos solicitar si podemos incluir tu regla también en la biblioteca. Esto podría ser más fácil que la opción 2 porque puedes dejarnos a nosotros la adición.
2.  **Envíanos una pull request**: Puedes enviarnos una pull request. Por favor, consulta [las instrucciones](https://github.com/mshibanami/redirect-web/tree/main/library) para obtener más información.

## Otra información

-   Puedes ver todo el contenido en [el repositorio de GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
-   Alojamientos nuestro servidor API privado en Cloudflare para el contenido de la biblioteca, lo que nos permite proporcionar filtrado y otros servicios. La aplicación obtiene el contenido de la biblioteca de este servidor. Como se indica en [nuestra política de privacidad](/privacy-policy), este servidor no recopila tu información personal.

# Biblioteca

¡Bienvenido a nuestra biblioteca, donde puede encontrar reglas útiles y listas para usar de forma gratuita! Puede acceder a ellas a través de la aplicación Redirect Web.

![Botón Obtener - iOS](/img/library-get-button-ios.gif)

## Contribuya con su regla a la biblioteca

La biblioteca actual solo tiene unas pocas reglas, ¡así que agradecemos su contribución!

Hay 2 opciones para contribuir con sus reglas a la biblioteca:

1. **Publique su regla en el Foro**: Comparta su regla de redirección con otros usuarios en [el foro](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Luego, podemos solicitar si podemos incluir su regla en la biblioteca también. Esto podría ser más fácil que la opción 2 porque puede dejarnos la adición a nosotros.
2. **Envíenos una solicitud de extracción**: Puede enviarnos una solicitud de extracción. Consulte [las instrucciones](https://github.com/mshibanami/redirect-web/tree/main/library) para obtener más información.

## Otra información

- Puede ver todo el contenido en [el repositorio en GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- Alojamientos nuestro servidor API privado en Cloudflare para el contenido de la biblioteca, lo que nos permite proporcionar filtrado y otros servicios. La aplicación obtiene el contenido de la biblioteca de este servidor. Como se indica en [nuestra política de privacidad](privacy-policy.mdx), este servidor no recopila su información personal.

# Caso 1: Redirigir a otro sitio web

Supongamos que eres adicto a Twitter y, para romper el hábito, decides ir a [Unrush](https://unrush.org) para ver frases motivacionales y hacer una breve meditación cada vez que abras Twitter accidentalmente. ¡Vamos a crear una regla para eso!

### Paso 1. Abrir la pantalla "Editar regla"

Abramos la pantalla *Editar regla* para comenzar a crear tu propia regla en la aplicación. Puedes hacerlo haciendo clic en el botón más (+) en la pestaña *Mis reglas*.

### Paso 2. Configurar "Redirigir desde"

Primero, debes configurar la sección *Redirigir desde* en la pantalla *Editar regla* de la aplicación. Los objetivos son las URL que comienzan con `https://twitter.com/`.

En este caso, puedes especificar el siguiente patrón con el modo [Comodín (Wildcard)](/rule-settings#url-pattern-wildcard):

```
https://twitter.com/*
```

En el modo Comodín, `*` significa que coincide con cualquier cosa (= cero o más caracteres).

Sin embargo, espera un momento. `twitter.com` pasó a llamarse `x.com`. Podrías simplemente reemplazar `twitter.com` con `x.com`, pero nadie sabe cuándo el nuevo dueño cambiará de opinión nuevamente para volver atrás. Por lo tanto, apuntemos tanto a `twitter.com` como a `x.com`.

Para hacer eso, cambia de Comodín a [Expresión regular](/rule-settings#url-pattern-regular-expression) y establece esto como patrón:

```
https://(twitter|x).com/.*
```

* `(twitter|x)`: apunta tanto a `twitter` como a `x`. (`|` se llama *tubería* o *pipe*).
* `.*`: Es lo mismo que el `*` de Comodín. Más específicamente, `.` significa *cualquier carácter* y `*` significa *cualquier cosa antes de este símbolo repetida cualquier número de veces*, lo que hace que coincida con cualquier cosa.

La Expresión Regular es un poco complicada, pero una vez que te acostumbras a ella, será una herramienta poderosa. Recomendamos usar [RegExr](https://regexr.com) como campo de pruebas para analizar cómo funciona tu patrón de Expresión Regular.

> [!TIP]
> En las Expresiones Regulares, el `.` en `(twitter|x).com` también se trata como *cualquier carácter*. Por lo tanto, `(twitter|x).com/.*` también coincide, por ejemplo, con `twitter1com/` o `x_com/`.
>
> Para ser más precisos, puedes cambiarlo a `(twitter|x)\.com/.*`. El `\` se utiliza para tratar el `.` como un carácter de punto literal en lugar de un carácter especial.
>
> Sin embargo, no existen tales URL en el entorno general de Internet. Por lo tanto, puedes dejar el `.` como un carácter especial si lo prefieres.

### Paso 3: Configurar "Redirigir a"

Simplemente especifica la URL de la siguiente manera:

```
https://unrush.org/
```

### (Opcional) Paso 4: Comprobar la regla

Puedes comprobar cómo funciona la regla añadiendo una URL de ejemplo en la sección *Ejemplos* de la pantalla *Editar regla*. Por ejemplo, si introduces `https://x.com/UN`, muestra `https://unrush.org/` como resultado.

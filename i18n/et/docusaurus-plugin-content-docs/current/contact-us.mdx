# Võta meiega ühendust

Oleme siin, et aidata, kui teil on küsimusi, tagasisidet või probleeme Redirect Webiga. Siin on parimad viisid meiega ühendust võtmiseks:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Kui leiate vigu või soovite taotleda uusi funktsioone, külastage palun GitHub Issues lehte. Saate sirvida olemasolevaid probleeme, esitada uue probleemi ja jälgida oma probleemi edenemist.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Kui teil on küsimusi või soovite abi arendajalt või teistelt kasutajatelt, külastage palun GitHub Discussions lehte. Saate alustada uut arutelu, liituda olemasolevate aruteludega ja saada abi kogukonnalt.
- **Tagasisidevorm**: Saate saata meile tagasisidet Redirect Webi rakenduses oleva tagasisidevormi kaudu. See saadab e-kirja otse arendajale, nii et teie tagasiside on teiste kasutajate eest privaatne.

Täname, et kasutate Redirect Webi! Hindame teie toetust ja tagasisidet.

# Reeglite eksport/import

## Reeglite eksport {#export-rules}

### Eksportimine macOS-is {#export-on-macos}

Klõpsake kas kolmikpunkti nupul või jagamisnupul, nagu on näidatud sellel ekraanipildil:

![Share on macOS](/img/share-on-macos.png)

Saate oma reeglid ka valiku alusel mitmikvalikuga eksportida, klõpsates neil parema hiireklahviga reeglite loendi külgribal:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Lisaks saate kõik reeglid eksportida menüüribalt > Fail > Ekspordi kõik reeglid:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Kui soovite oma reeglid eksportida formaadis, mis ühildub [Einar Egilssoni Redirectoriga](https://einaregilsson.com/redirector/), vajutage kontekstimenüüs või menüüribal klahvi Option (`⌥`), siis näete järgmist:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web ei ole Redirectoriga täielikult ühilduv ja vastupidi. Mõned seaded, näiteks "Replace Occurrences" hõivamisgrupi töötlemiseks, ei ekspordita Redirectori formaadiga.
> :::

## Reeglite importimine {#import-rules}

### Importimine macOS-is {#import-on-macos}

- Lihtsalt lohistage eksporditud reeglifailid Redirect Webi aknasse.
- Või saate need importida ka menüüribalt > Fail > Impordi reeglid.

### Importimine iOS-is {#import-on-ios}

Eeldame, et teie reeglifail asub rakenduses Files. Avage see rakenduses Files, puudutage jagamisnuppu ja seejärel valige Redirect Web:

| 1. samm: Puudutage tööriistaribal jagamisnuppu                                       | 2. samm: Valige lehel rakenduse ikoon                                          |
| ----------------------------------------------------------------------------------- | ----------------------------------------------------------------------------- |
| ![Tap the share button in the toolbar](/img/files-share-ios.png)           | ![Tap the app icon in the sheet](/img/share-sheet-ios.png)         |

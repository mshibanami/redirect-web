# Reeglite eksport/import

## Reeglite eksport

### Eksport macOS-is

Klõpsake kas kolmikpunkti nupul või jagamisnupul, nagu on näidatud sellel ekraanipildil:

![Share on macOS](/img/share-on-macos.png)

Saate ka mitu reeglit valida ja neid eksportida, paremklõpsates (sekundaarselt klõpsates) reeglite loendi külgribal:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Lisaks saate kõik reeglid eksportida menüüribalt > Fail > Ekspordi kõik reeglid:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Kui soovite oma reegleid eksportida vormingus, mis ühildub [Einar Egilssoni Redirectoriga](https://einaregilsson.com/redirector/), vajutage kontekstimenüüs või menüüribal klahvi Option (`⌥`), siis näete järgmist:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web ei ühildu täielikult Redirectoriga ja vastupidi. Mõned seaded, näiteks "Asendage esinemised" jäädvustamisgrupi töötlemiseks, ei ekspordita Redirectori vormingus.
> :::

## Reeglite importimine

### Import macOS-is

- Lihtsalt lohistage eksporditud reeglifailid Redirect Webi aknasse.
- Või saate need importida ka menüüribalt > Fail > Impordi reeglid

### Import iOS-is

Oletame, et teie reeglifail asub rakenduses Files. Avage see Filesis, puudutage jagamisnuppu ja seejärel valige Redirect Web:

| 1. samm: Puudutage jagamisnuppu Filesis | 2. samm: Valige Redirect Web          |

# Reeglite eksportimine/importimine

## Reeglite eksportimine {#export-rules}

### Eksportimine macOS-is {#export-on-macos}

Klõpsake kas kolme punkti nupul või jagamisnupul, nagu on näidatud sellel ekraanipildil:

![Jaga macOS-is](/img/share-on-macos.png)

Saate ka oma reeglid mitmikvalida ja eksportida, paremklõpsates neil reeglite loendi külgribal:

![Mitmikvalik macOS-is](/img/multiselect-on-macos.png)

Lisaks saate kõik reeglid eksportida menüüribalt > Fail > Ekspordi kõik reeglid:

![Menüüriba > Fail](/img/menubar-file.png)

> [!NOTE]
> Kui soovite oma reeglid eksportida [Einar Egilssoni Redirectori](https://einaregilsson.com/redirector/) ühilduvas vormingus, vajutage kontekstimenüüs või menüüribal klahvi Option (`⌥`), siis näete järgmist:
> ![Menüüriba > Fail (Redirector)](/img/menubar-file-redirector.png)
>
> ![Kontekstimenüü (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web ei ole Redirectoriga täielikult ühilduv ja vastupidi. Mõned seaded, näiteks "Asenduste arv" püüdurrühma töötlemiseks, ei ekspordita Redirectori vormingus.
> :::

## Reeglite importimine {#import-rules}

### Importimine macOS-is {#import-on-macos}

- Lihtsalt lohistage eksporditud reeglifailid Redirect Webi aknasse.
- Või saate need importida ka menüüribalt > Fail > Impordi reeglid

### Importimine iOS-is {#import-on-ios}

Oletame, et teie reeglifail on rakenduses Failid. Avage see rakenduses Failid, puudutage jagamisnuppu ja seejärel valige Redirect Web:

| 1. samm: Puudutage tööriistaribal jagamisnuppu | 2. samm: Valige lehel Redirect Web |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Puudutage tööriistaribal jagamisnuppu](/img/files-share-ios.png) | ![Puudutage lehel rakenduse ikooni](/img/share-sheet-ios.png) |

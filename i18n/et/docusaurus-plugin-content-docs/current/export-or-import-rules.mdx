# Reeglite eksportimine/importimine

## Reeglite eksportimine {#export-rules}

### Eksportimine macOS-is {#export-on-macos}

Klõpsake kas kolme punkti nupul või jagamisnupul, nagu on näidatud sellel ekraanipildil:

![Jaga macOS-is](/img/share-on-macos.png)

Saate ka mitu reeglit valida ja neid eksportida, klõpsates neil parema hiireklõpsuga reeglite loendi külgribal:

![Jaga mitut reeglit macOS-is](/img/multiselect-on-macos.png)

Lisaks saate kõik reeglid eksportida menüüribalt > Fail > Ekspordi kõik reeglid:

![Menüüriba > Fail](/img/menubar-file.png)

> [!NOTE]
> Kui soovite oma reegleid eksportida [Einar Egilssoni Redirectori](https://einaregilsson.com/redirector/) jaoks sobivas vormingus, vajutage kontekstimenüüs või menüüribal klahvi Option (`⌥`), siis näete järgmist:
> ![Menüüriba > Fail (Redirector)](/img/menubar-file-redirector.png)
>
> ![Kontekstimenüü (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web ei ole täielikult ühilduv Redirectoriga ja vastupidi. Mõned seaded, näiteks "Asendada esinemised" püüdmisrühma töötlemiseks, ei ekspordita Redirectori vormingus.
> :::

## Reeglite importimine {#import-rules}

### Importimine macOS-is {#import-on-macos}

- Lihtsalt lohistage eksporditud reeglifailid Redirect Webi aknasse.
- Või saate neid importida ka menüüribalt > Fail > Impordi reeglid

### Importimine iOS-is {#import-on-ios}

Oletame, et teie reeglifail asub rakenduses Failid. Avage see rakenduses Failid, puudutage jagamisnuppu ja seejärel valige Redirect Web:

| Samm 1: Puudutage jagamisnuppu tööriistaribal | Samm 2: Valige Redirect Web |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Puudutage jagamisnuppu tööriistaribal](/img/files-share-ios.png) | ![Puudutage rakenduse ikooni lehel](/img/share-sheet-ios.png) |

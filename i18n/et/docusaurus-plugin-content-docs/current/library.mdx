# Teek

Tere tulemast meie teeki, kust leiate tasuta kasulikke ja valmis reegleid! Saate neile juurde pääseda Redirect Web rakenduse kaudu.

![Teegi kuvatõmmis](/img/library-screenshot.webp)

Reeglite kopeerimiseks jaotisesse Minu reeglid peate lihtsalt valima nupu "Hangi":

![Nupp Hangi - iOS](/img/library-get-button-ios.gif)

## Panusta oma reegliga teeki

Praeguses teegis on vaid mõned reeglid, seega hindame teie panust!

Teeki panustamiseks on 2 võimalust:

1.  **Postita oma reegel foorumisse**: Jaga oma ümbersuunamisreeglit teiste kasutajatega [foorumis](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Seejärel võime paluda, kas saaksime teie reegli lisada ka teeki. See võib olla lihtsam kui variant 2, sest saate lisamise jätta meie hooleks.
2.  **Saada meile pull request**: Saate meile saata pull request'i. Lisateabe saamiseks vaadake [juhiseid](https://github.com/mshibanami/redirect-web/tree/main/library).

## Muu teave

-   Kogu sisu näete [GitHubi repositooriumis](https://github.com/mshibanami/redirect-web/tree/main/library).
-   Me majutame oma privaatset API-serverit Cloudflare'is teegi sisu jaoks, võimaldades meil pakkuda filtreerimist ja muid teenuseid. Rakendus hangib teegi sisu sellest serverist. Nagu on märgitud [meie privaatsuspoliitikas](/privacy-policy), ei kogu see server teie isikuandmeid.

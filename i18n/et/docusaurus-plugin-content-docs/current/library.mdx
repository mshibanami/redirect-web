# Kogu

Tere tulemast meie kogusse, kust leiate tasuta kasulikke ja koheselt kasutatavaid reegleid! Saate neile juurde pääseda Redirect Web rakenduse kaudu.

![Library Screenshot](/img/library-screenshot.webp)

Reeglite kopeerimiseks jaotisesse "Minu reeglid" saate lihtsalt valida nupu "Hangi":

![Get Button - iOS](/img/library-get-button-ios.gif)

## Panusta oma reegliga kogusse

Praeguses kogus on vaid mõned reeglid, seega hindame teie panust!

Oma reeglitega kogusse panustamiseks on 2 võimalust:

1.  **Postita oma reegel foorumisse**: Jaga oma ümbersuunamisreeglit teiste kasutajatega [foorumis](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Seejärel võime küsida, kas saaksime teie reegli lisada ka kogusse. See võib olla lihtsam kui variant 2, sest saate lisamise jätta meie hooleks.
2.  **Saada meile pull request**: Saate meile saata pull requesti. Lisateabe saamiseks vaadake palun [juhiseid](https://github.com/mshibanami/redirect-web/tree/main/library).

## Muu teave

-   Kogu sisu näete [GitHubi hoidlas](https://github.com/mshibanami/redirect-web/tree/main/library).
-   Kogu sisu jaoks kasutame Cloudflare'is oma privaatset API-serverit, mis võimaldab meil pakkuda filtreerimis- ja muid teenuseid. Rakendus hangib kogu sisu sellest serverist. Nagu on märgitud [meie privaatsuspoliitikas](privacy-policy.mdx), ei kogu see server teie isikuandmeid.

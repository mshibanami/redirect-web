# Teek

Tere tulemast meie teeki, kust leiate tasuta kasulikke kasutusvalmis reegleid! Neile pääsete ligi Redirect Web rakenduse kaudu.

![Teegi ekraanipilt](/img/library-screenshot.webp)

Reeglite kopeerimiseks Minu reeglitesse saate lihtsalt valida nupu Hangi:

![Hangi nupp - iOS](/img/library-get-button-ios.gif)

## Panusta oma reegel teeki

Praeguses teegis on vaid mõned reeglid, seega hindame teie panust!

Oma reeglite teeki panustamiseks on 2 võimalust:

1. **Postita oma reegel foorumisse**: Jaga oma ümbersuunamisreeglit teiste kasutajatega [foorumis](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Seejärel võime küsida, kas saaksime teie reegli ka teeki lisada. See võib olla lihtsam kui variant 2, sest saate lisamise jätta meie hooleks.
2. **Saada meile pull request**: Saate meile saata pull requesti. Lisateabe saamiseks vaadake palun [juhiseid](https://github.com/mshibanami/redirect-web/tree/main/library).

## Muu teave

- Kogu sisu näete [GitHubi repositooriumis](https://github.com/mshibanami/redirect-web/tree/main/library).
- Me majutame oma privaatset API-serverit Cloudflare'is teegi sisu jaoks, mis võimaldab meil pakkuda filtreerimist ja muid teenuseid. Rakendus hangib teegi sisu sellest serverist. Nagu on öeldud [meie privaatsuspoliitikas](/privacy-policy.mdx), ei kogu see server teie isikuandmeid.

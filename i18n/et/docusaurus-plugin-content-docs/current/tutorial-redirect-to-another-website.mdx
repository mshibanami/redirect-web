# Juhtum 1: Ümbersuunamine teisele veebisaidile

Oletame, et oled Twitterist sõltuvuses ja harjumuse murdmiseks otsustad minna saidile [Unrush](https://unrush.org) motivatsioonitsitaatide ja lühikese meditatsiooni saamiseks alati, kui kogemata Twitteri avad. Loome selleks reegli!

### Samm 1. Ava ekraan "Muuda reeglit"

Ava rakenduses ekraan *Muuda reeglit* (*Edit Rule*), et alustada oma reegli loomist. Saad seda teha, klõpsates vahekaardil *Minu reeglid* (*My Rules*) pluss (+) nuppu.

### Samm 2. Seadista "Suuna ümber asukohast"

Esmalt peaksid rakenduse ekraanil *Muuda reeglit* seadistama jaotise *Suuna ümber asukohast* (*Redirect From*). Sihtmärkideks on URL-id, mis algavad tekstiga `https://twitter.com/`.

Selles režiimis saad määrata järgmise mustri režiimiga [Wildcard](/rule-settings#url-pattern-wildcard) (metsamärk):

```
https://twitter.com/*
```

Wildcard-režiimis tähendab `*`, et see sobib kõigega (= null või enam märki).

Kuid oota hetk. `twitter.com` nimetati ümber saidiks `x.com`. Sa võid lihtsalt asendada `twitter.com` osaga `x.com`, kuid keegi ei tea, millal uus omanik jälle meelt muudab ja selle tagasi rullib. Seetõttu sihtigem nii `twitter.com` kui ka `x.com`.

Selleks lülitu režiimilt Wildcard režiimile [Regulaaravaldis](/rule-settings#url-pattern-regular-expression) (Regular Expression) ja määra mustriks see:

```
https://(twitter|x).com/.*
```

* `(twitter|x)`: see sihib nii `twitterit` kui ka `x`-i. (`|` sümbolit nimetatakse *püstkriipsuks* ehk *pipe*'iks.)
* `.*`: see on sama mis Wildcard-i `*`. Täpsemalt öeldes tähendab `.` *suvalist märki* ja `*` tähendab, et *sellele sümbolile eelnevat korratakse suvaline arv kordi*, mille tulemusena sobib see kõigega.

Regulaaravaldis on veidi keeruline, kuid kui sellega harjud, on see võimas tööriist. Soovitame kasutada saiti [RegExr](https://regexr.com) mänguväljakuna, et analüüsida, kuidas sinu regulaaravaldise muster töötab.

> [!TIP]
> Regulaaravaldistes töödeldakse punkti `.` mustris `(twitter|x).com` samuti kui *suvalist märki*. Seetõttu sobib `(twitter|x).com/.*` näiteks ka aadressidega `twitter1com/` või `x_com/`.
>
> Täpsemaks muutmiseks võid selle muuta mustriks `(twitter|x)\.com/.*`. Sümbolit `\` kasutatakse selleks, et käsitleda punkti `.` kui tavalist punktimärki, mitte erimärki.
>
> Tavapärases internetikeskkonnas selliseid URL-e siiski ei esine. Seetõttu võid soovi korral jätta punkti `.` erimärgiks.

### Samm 3: Seadista "Suuna ümber asukohta"

Lihtsalt määra URL jaotises *Suuna ümber asukohta* (*Redirect To*) järgmiselt:

```
https://unrush.org/
```

### (Valikuline) Samm 4: Kontrolli reeglit

Saad kontrollida reegli toimimist, lisades näidis-URL-i ekraani *Muuda reeglit* jaotisesse *Näited* (*Examples*). Kui sisestad näiteks `https://x.com/UN`, kuvatakse tulemusena `https://unrush.org/`.

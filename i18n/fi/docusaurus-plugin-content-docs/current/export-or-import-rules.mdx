# Sääntöjen vienti/tuonti

## Sääntöjen vienti {#export-rules}

### Vienti macOS:llä {#export-on-macos}

Napsauta joko kolmen pisteen painiketta tai Jaa-painiketta, kuten tässä kuvakaappauksessa näkyy:

![Share on macOS](/img/share-on-macos.png)

Voit myös valita useita sääntöjä ja viedä ne toissijaisella napsautuksella (oikealla napsautuksella) sääntöluettelon sivupalkissa:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Lisäksi voit viedä kaikki säännöt valikkoriviltä > Tiedosto > Vie kaikki säännöt:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Jos haluat viedä sääntösi [Einar Egilssonin Redirector](https://einaregilsson.com/redirector/)-yhteensopivassa muodossa, paina Optio-näppäintä (`⌥`) kontekstivalikossa tai valikkorivillä, niin näet nämä:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web ei ole täysin yhteensopiva Redirectorin kanssa, eikä päinvastoin. Joitakin asetuksia, kuten "Replace Occurrences" for Capturing Group Processing, ei viedä Redirectorin muodossa.
> :::

## Sääntöjen tuonti {#import-rules}

### Tuonti macOS:llä {#import-on-macos}

- Vedä ja pudota viedyt sääntötiedostot Redirect Webin ikkunaan.
- Tai voit tuoda ne myös valikkoriviltä > Tiedosto > Tuo säännöt

### Tuonti iOS:llä {#import-on-ios}

Oletetaan, että sääntötiedostosi on Tiedostot-sovelluksessa. Avaa se Tiedostoissa, napauta Jaa-painiketta ja valitse sitten Redirect Web:

| Vaihe 1: Napauta Jaa-painiketta Tiedostoissa                                     | Vaihe 2: Valitse Redirect Web                                         |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Tap the share button in the toolbar](/img/files-share-ios.png) | ![Tap the app icon in the sheet](/img/share-sheet-ios.png) |

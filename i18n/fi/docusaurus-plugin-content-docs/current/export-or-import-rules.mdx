# Sääntöjen vienti/tuonti

## Sääntöjen vienti {#export-rules}

### Vienti macOS:llä {#export-on-macos}

Napsauta joko ellipsi-painiketta tai Jaa-painiketta, kuten tässä kuvakaappauksessa näkyy:

![Jaa macOS:llä](/img/share-on-macos.png)

Voit myös monivalita säännöt ja viedä ne napsauttamalla hiiren oikealla painikkeella sääntölistauksen sivupalkissa:

![Jaa useita sääntöjä macOS:llä](/img/multiselect-on-macos.png)

Lisäksi voit viedä kaikki säännöt valikkopalkista > Tiedosto > Vie kaikki säännöt:

![Valikkopalkki > Tiedosto](/img/menubar-file.png)

> [!NOTE]
> Jos haluat viedä sääntösi [Einar Egilssonin Redirectorin](https://einaregilsson.com/redirector/) kanssa yhteensopivassa muodossa, paina Option-näppäintä (`⌥`) pikavalikossa tai valikkopalkissa, niin näet seuraavat:
> ![Valikkopalkki > Tiedosto (Redirector)](/img/menubar-file-redirector.png)
>
> ![Pikavalikko (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web ei ole täysin yhteensopiva Redirectorin kanssa, eikä päinvastoin. Joitakin asetuksia, kuten "Korvaa esiintymät" (Replace Occurrences) sieppaavan ryhmän käsittelyssä (Capturing Group Processing), ei viedä Redirectorin muodossa.
> :::

## Sääntöjen tuonti {#import-rules}

### Tuonti macOS:llä {#import-on-macos}

-   Vedä ja pudota viedyt sääntötiedostot yksinkertaisesti Redirect Webin ikkunaan.
-   Tai voit myös tuoda ne valikkopalkista > Tiedosto > Tuo säännöt

### Tuonti iOS:llä {#import-on-ios}

Oletetaan, että sääntötiedostosi on Tiedostot-sovelluksessa. Avaa se Tiedostot-sovelluksessa, napauta Jaa-painiketta ja valitse sitten Redirect Web:

| Vaihe 1: Napauta Jaa-painiketta työkalupalkissa                           | Vaihe 2: Valitse Redirect Web                                       |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Napauta työkalupalkin jakamispainiketta](/img/files-share-ios.png) | ![Napauta sovelluskuvaketta paneelissa](/img/share-sheet-ios.png) |

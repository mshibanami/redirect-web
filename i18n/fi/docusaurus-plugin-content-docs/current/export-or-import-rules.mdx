# Sääntöjen vienti/tuonti

## Sääntöjen vienti {#export-rules}

### Vienti macOS:llä {#export-on-macos}

Napsauta joko kolmen pisteen painiketta tai Jaa-painiketta, kuten tässä kuvakaappauksessa näkyy:

![Jaa macOS:llä](/img/share-on-macos.png)

Voit myös valita useita sääntöjä ja viedä ne napsauttamalla hiiren kakkospainikkeella (oikealla napsautuksella) sääntölistan sivupalkissa:

![Jaa useita sääntöjä macOS:llä](/img/multiselect-on-macos.png)

Lisäksi voit viedä kaikki säännöt valikkoriviltä > Tiedosto > Vie kaikki säännöt:

![Valikkorivi > Tiedosto](/img/menubar-file.png)

> [!NOTE]
> Jos haluat viedä sääntösi [Einar Egilssonin Redirectorin](https://einaregilsson.com/redirector/) kanssa yhteensopivassa muodossa, paina Option-näppäintä (`⌥`) kontekstivalikossa tai valikkorivillä, jolloin näet nämä:
> ![Valikkorivi > Tiedosto (Redirector)](/img/menubar-file-redirector.png)
>
> ![Kontekstivalikko (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web ei ole täysin yhteensopiva Redirectorin kanssa, eikä päinvastoin. Joitakin asetuksia, kuten "Korvaa esiintymät" sieppausryhmän käsittelyssä, ei viedä Redirectorin muodossa.
> :::

## Sääntöjen tuonti {#import-rules}

### Tuonti macOS:llä {#import-on-macos}

- Vedä ja pudota viedyt sääntötiedostot Redirect Webin ikkunaan.
- Tai voit tuoda ne myös valikkoriviltä > Tiedosto > Tuo säännöt

### Tuonti iOS:llä {#import-on-ios}

Oletetaan, että sääntötiedostosi on Tiedostot-sovelluksessa. Avaa se Tiedostoissa, napauta Jaa-painiketta ja valitse sitten Redirect Web:

| Vaihe 1: Napauta Jaa-painiketta työkalupalkissa | Vaihe 2: Valitse Redirect Web |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Napauta jakopainiketta työkalupalkissa](/img/files-share-ios.png) | ![Napauta sovelluskuvaketta arkissa](/img/share-sheet-ios.png) |

# Kirjasto

Tervetuloa kirjastoomme, josta löydät hyödyllisiä käyttövalmiita sääntöjä ilmaiseksi! Voit käyttää niitä Redirect Web -sovelluksen kautta.

![Library Screenshot](/img/library-screenshot.webp)

Kopioidaksesi säännöt Omiin sääntöihin, voit yksinkertaisesti valita Get-painikkeen:

![Get Button - iOS](/img/library-get-button-ios.gif)

## Lisää sääntösi kirjastoon

Nykyisessä kirjastossa on vain muutama sääntö, joten arvostamme panostasi!

Sääntöjesi lisäämiseen kirjastoon on 2 vaihtoehtoa:

1. **Julkaise sääntösi foorumilla**: Jaa uudelleenohjaussääntösi muiden käyttäjien kanssa [foorumilla](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Voimme sitten pyytää, saisimmeko lisätä sääntösi myös kirjastoon. Tämä saattaa olla helpompaa kuin vaihtoehto 2, koska voit jättää lisäyksen meille.
2. **Lähetä meille pull request**: Voit lähettää meille pull requestin. Tarkista [ohjeet](https://github.com/mshibanami/redirect-web/tree/main/library) saadaksesi lisätietoja.

## Muuta tietoa

- Voit nähdä kaikki sisällöt [GitHubin repositoriossa](https://github.com/mshibanami/redirect-web/tree/main/library).
- Isännöimme yksityistä API-palvelintamme Cloudflaressa kirjaston sisältöä varten, mikä mahdollistaa suodatuksen ja muiden palveluiden tarjoamisen. Sovellus hakee kirjaston sisällön tältä palvelimelta. Kuten [tietosuojakäytännössämme](privacy-policy.mdx) todetaan, tämä palvelin ei kerää henkilökohtaisia tietojasi.

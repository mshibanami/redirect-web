# Kirjasto

Tervetuloa kirjastoomme, josta löydät hyödyllisiä valmiita sääntöjä ilmaiseksi! Pääset niihin Redirect Web -sovelluksen kautta.

![Kirjasto – kuvakaappaus](/img/library-screenshot.webp)

Kopioidaksesi säännöt Omat säännöt -osioon, voit yksinkertaisesti valita Hae-painikkeen:

![Hae-painike - iOS](/img/library-get-button-ios.gif)

## Lisää oma sääntösi kirjastoon

Nykyisessä kirjastossa on vain muutama sääntö, joten arvostamme panostasi!

Voit lisätä sääntöjäsi kirjastoon kahdella tavalla:

1.  **Julkaise sääntösi foorumilla**: Jaa uudelleenohjaussääntösi muiden käyttäjien kanssa [foorumilla](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Saatamme sen jälkeen kysyä, voisimmeko lisätä sääntösi myös kirjastoon. Tämä voi olla helpompaa kuin vaihtoehto 2, koska voit jättää lisäyksen meidän huoleksemme.
2.  **Lähetä meille pull request**: Voit lähettää meille pull requestin. Katso lisätietoja [ohjeista](https://github.com/mshibanami/redirect-web/tree/main/library).

## Muuta tietoa

-   Näet kaikki sisällöt [GitHub-tietovarastossa](https://github.com/mshibanami/redirect-web/tree/main/library).
-   Isännöimme yksityistä API-palvelintamme Cloudflaressa kirjaston sisällölle, mikä mahdollistaa suodatuksen ja muiden palvelujen tarjoamisen. Sovellus hakee kirjaston sisällöt tältä palvelimelta. Kuten [tietosuojakäytännössämme](/privacy-policy) mainitaan, tämä palvelin ei kerää henkilökohtaisia tietojasi.

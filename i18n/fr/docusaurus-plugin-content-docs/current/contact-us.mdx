# Nous contacter

Nous sommes là pour vous aider si vous avez des questions, des commentaires ou des problèmes avec Redirect Web. Voici les meilleures façons de nous contacter :

- **[Problèmes GitHub](https://github.com/mshibanami/RedirectWeb/issues)**: Si vous rencontrez des bugs ou souhaitez demander de nouvelles fonctionnalités, veuillez visiter la page des Problèmes GitHub. Vous pouvez parcourir les problèmes existants, soumettre un nouveau problème et suivre l'avancement de votre problème.
- **[Discussions GitHub](https://github.com/mshibanami/RedirectWeb/discussions)**: Si vous avez des questions ou souhaitez obtenir de l'aide du développeur ou d'autres utilisateurs, veuillez visiter la page des Discussions GitHub. Vous pouvez lancer une nouvelle discussion, rejoindre des discussions existantes et obtenir de l'aide de la communauté.
- **Formulaire de commentaires**: Vous pouvez nous envoyer des commentaires via le formulaire de commentaires dans l'application Redirect Web. Cela enverra un e-mail directement au développeur, afin que vos commentaires restent privés des autres utilisateurs.

Merci d'utiliser Redirect Web ! Nous apprécions votre soutien et vos commentaires.

# Exporter/Importer des règles

## Exporter des règles

### Exporter sur macOS

Cliquez sur le bouton avec les points de suspension ou sur le bouton Partager, comme indiqué dans cette capture d'écran :

![Partager sur macOS](/img/share-on-macos.png)

Vous pouvez également sélectionner plusieurs de vos règles et les exporter en effectuant un clic secondaire (clic droit) dessus dans la barre latérale de la liste des règles :

![Partager plusieurs règles sur macOS](/img/multiselect-on-macos.png)

De plus, vous pouvez exporter toutes les règles depuis la barre de menus > Fichier > Exporter toutes les règles :

![Barre de menus > Fichier](/img/menubar-file.png)

> [!NOTE]
> Si vous souhaitez exporter vos règles dans un format compatible avec [Redirector d'Einar Egilsson](https://einaregilsson.com/redirector/), appuyez sur la touche Option (`⌥`) dans le menu contextuel ou la barre de menus. Vous verrez alors ceci :
> ![Barre de menus > Fichier (Redirector)](/img/menubar-file-redirector.png)
>
> ![Menu contextuel (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web n'est pas entièrement compatible avec Redirector, et vice versa. Certains paramètres, tels que « Remplacer les occurrences » pour le traitement des groupes de capture, ne sont pas exportés au format de Redirector.
> :::

## Importer des règles

### Importer sur macOS

- Glissez-déposez simplement les fichiers de règles exportés dans la fenêtre de Redirect Web.
- Ou, vous pouvez également les importer depuis la barre de menus > Fichier > Importer des règles

### Importer sur iOS

Supposons que votre fichier de règles se trouve dans l'application Fichiers. Ouvrez-le dans Fichiers, appuyez sur le bouton Partager, puis sélectionnez Redirect Web :

| Étape 1 : Appuyez sur le bouton Partager dans Fichiers | Étape 2 : Sélectionnez Redirect Web           |
| ------------------------------------- | ------------------------------------- |
| ![Texte alternatif](/img/files-share-ios.png) | ![Texte alternatif](/img/share-sheet-ios.png) |

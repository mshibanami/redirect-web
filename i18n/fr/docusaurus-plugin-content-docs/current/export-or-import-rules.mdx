# Exporter/Importer des règles

## Exporter des règles {#export-rules}

### Exporter sur macOS {#export-on-macos}

Cliquez sur le bouton des points de suspension ou sur le bouton Partager comme indiqué dans cette capture d'écran :

![Partager sur macOS](/img/share-on-macos.png)

Vous pouvez également sélectionner plusieurs règles et les exporter en cliquant avec le bouton secondaire (clic droit) sur la barre latérale de la liste des règles :

![Partager plusieurs règles sur macOS](/img/multiselect-on-macos.png)

De plus, vous pouvez exporter toutes les règles depuis la barre de menus > Fichier > Exporter toutes les règles :

![Barre de menus > Fichier](/img/menubar-file.png)

> [!NOTE]
> Si vous souhaitez exporter vos règles au format compatible avec [Redirector d'Einar Egilsson](https://einaregilsson.com/redirector/), appuyez sur la touche Option (`⌥`) dans le menu contextuel ou la barre de menus, puis vous verrez ceci :
> ![Barre de menus > Fichier (Redirector)](/img/menubar-file-redirector.png)
>
> ![Menu contextuel (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web n'est pas entièrement compatible avec Redirector, et vice versa. Certains paramètres, tels que "Remplacer les occurrences" pour le traitement des groupes de capture, ne sont pas exportés au format de Redirector.
> :::

## Importer des règles {#import-rules}

### Importer sur macOS {#import-on-macos}

- Il suffit de glisser-déposer les fichiers de règles exportés dans la fenêtre de Redirect Web.
- Ou, vous pouvez également les importer depuis la barre de menus > Fichier > Importer des règles

### Importer sur iOS {#import-on-ios}

Supposons que votre fichier de règles se trouve dans l'application Fichiers. Ouvrez-le dans Fichiers, appuyez sur le bouton Partager, puis sélectionnez Redirect Web :

| Étape 1 : Appuyez sur le bouton de partage dans la barre d'outils | Étape 2 : Sélectionnez Redirect Web |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Appuyez sur le bouton de partage dans la barre d'outils](/img/files-share-ios.png) | ![Appuyez sur l'icône de l'application dans la feuille](/img/share-sheet-ios.png) |

Bienvenue sur le site d'assistance de RedirectWeb, une extension de navigateur qui redirige tous les chargements de pages dans votre navigateur, vous assurant de toujours arriver là où vous le souhaitez.

(Anciennement connu sous le nom de _Redirect Web for Safari_ ou simplement _Redirect Web_.)

## ⬇️ Liens de téléchargement {/* #download-links */}

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chrome Web Store)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Add-ons de Firefox)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Add-ons de Microsoft Edge)
- [Safari](https://apps.apple.com/app/id1571283503) (L'extension est déjà intégrée dans l'application native.)

RedirectWeb est disponible sur :

- **Mac** (Safari, Chrome, Firefox, Edge, et plus)
- **iPhone**, **iPad** et **Apple Vision Pro** (Safari)

:::note

- Vous devez installer [l'application RedirectWeb depuis le Mac App Store](https://apps.apple.com/app/id1571283503) pour utiliser l'extension, quel que soit le navigateur que vous utilisez, car les extensions s'appuient sur l'application pour accéder à vos règles, paramètres, etc.
- La prise en charge multi-navigateur n'est disponible que sur Mac.

:::

## Cas d'utilisation populaires {/* #popular-use-cases */}

Par exemple, vous pouvez créer ces règles :

- Consulter **Reddit** dans sa conception classique (old.reddit.com).
- Ouvrir **Apple Maps** chaque fois qu'un lien **Google Maps** apparaît.
- Masquer les aperçus IA de Google des résultats de recherche **Google** en ajoutant **?udm=14** à chaque recherche.
- Passer des pages Fandom aux wikis officiels de **Terraria** ou **Minecraft**.
- Utiliser **Kagi**, **Startpage**, **ChatGPT** et tout autre moteur de recherche comme votre **moteur de recherche par défaut**.
- Ouvrir les liens **Google Meet** dans Google Chrome. (Mac)
- Ouvrir les liens **Figma** directement dans l'application Figma. (Mac)
- Ouvrir les liens **Notion** directement dans l'application Notion. (Mac)

Vous pouvez obtenir toutes ces règles prédéfinies gratuitement dans la section **Bibliothèque** de l'application !

## Fonctionnalités {/* #features */}

RedirectWeb propose les fonctionnalités suivantes :

- Créer des règles à l'aide de types de motifs **Expression régulière** (Regular Expression) ou **Caractère générique** (Wildcard)
- Utiliser le **back-referencing** pour spécifier des URL cibles dynamiques
- **Exclure** des URL de la redirection
- **Traiter les textes capturés** à l'aide de l'encodage/décodage d'URL, de l'encodage/décodage base64 ou du remplacement d'occurrences

De plus, vous pouvez débloquer les fonctionnalités suivantes via un abonnement ou un achat unique :

- Activer un nombre **illimité** de règles
- **Synchroniser les règles** avec vos autres Mac, iPhone et iPad **via iCloud**

(L'achat de l'application nous aide également à maintenir RedirectWeb. 🙏)

Votre achat ou abonnement est partagé sur votre Mac, iPhone, iPad et Apple Vision Pro via l'achat universel. Votre soutien aide au développement continu.

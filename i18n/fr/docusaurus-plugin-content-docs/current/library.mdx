# Bibliothèque

Bienvenue dans notre bibliothèque, où vous pouvez trouver gratuitement des règles utiles et prêtes à l'emploi ! Vous pouvez y accéder via l'application Redirect Web.

![Capture d'écran de la bibliothèque](/img/library-screenshot.webp)

Pour copier les règles dans Mes règles, il vous suffit de sélectionner le bouton Obtenir :

![Bouton Obtenir - iOS](/img/library-get-button-ios.gif)

## Contribuez votre règle à la bibliothèque

La bibliothèque actuelle ne contient que quelques règles, nous apprécions donc votre contribution !

Il existe 2 options pour contribuer vos règles à la bibliothèque :

1. **Publiez votre règle sur le Forum** : Partagez votre règle de redirection avec d'autres utilisateurs sur [le forum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Nous pourrions alors vous demander si nous pouvons également inclure votre règle dans la bibliothèque. Cela pourrait être plus facile que l'option 2 car vous pouvez nous laisser le soin de l'ajouter.
2. **Envoyez-nous une pull request** : Vous pouvez nous envoyer une pull request. Veuillez consulter [les instructions](https://github.com/mshibanami/redirect-web/tree/main/library) pour en savoir plus.

## Autres informations

- Vous pouvez voir tout le contenu dans [le dépôt sur GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- Nous hébergeons notre serveur API privé sur Cloudflare pour le contenu de la bibliothèque, ce qui nous permet de fournir des services de filtrage et autres. L'application récupère le contenu de la bibliothèque à partir de ce serveur. Comme indiqué dans [notre politique de confidentialité](privacy-policy.mdx), ce serveur ne collecte pas vos informations personnelles.

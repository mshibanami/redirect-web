# Kapcsolat

Segítünk, ha bármilyen kérdése, visszajelzése vagy problémája van a Redirect Web-bel kapcsolatban. Íme a legjobb módok a kapcsolatfelvételre:

- **[GitHub Problémák](https://github.com/mshibanami/RedirectWeb/issues)**: Ha bármilyen hibába ütközik, vagy új funkciókat szeretne kérni, kérjük, látogasson el a GitHub Problémák oldalra. Böngészhet a meglévő problémák között, új problémát nyújthat be, és nyomon követheti annak előrehaladását.
- **[GitHub Megbeszélések](https://github.com/mshibanami/RedirectWeb/discussions)**: Ha bármilyen kérdése van, vagy segítséget szeretne kapni a fejlesztőtől vagy más felhasználóktól, kérjük, látogasson el a GitHub Megbeszélések oldalra. Kezdeményezhet új megbeszélést, csatlakozhat meglévőkhöz, és segítséget kaphat a közösségtől.
- **Visszajelzési Űrlap**: Visszajelzést küldhet nekünk a Redirect Web alkalmazásban található visszajelzési űrlapon keresztül. Ez közvetlenül a fejlesztőnek küld egy e-mailt, így visszajelzése privát marad a többi felhasználó előtt.

Köszönjük, hogy a Redirect Web-et használja! Nagyra értékeljük támogatását és visszajelzését.

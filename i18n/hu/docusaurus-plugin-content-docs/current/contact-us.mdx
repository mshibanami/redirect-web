# Kapcsolat

Azért vagyunk itt, hogy segítsünk, ha bármilyen kérdése, visszajelzése vagy problémája van a Redirect Webbel kapcsolatban. Íme a legjobb módok a kapcsolatfelvételre:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Ha hibákat észlel, vagy új funkciókat szeretne kérni, kérjük, látogasson el a GitHub Issues oldalra. Böngészhet a meglévő problémák között, új problémát nyújthat be, és nyomon követheti a probléma előrehaladását.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Ha bármilyen kérdése van, vagy segítséget szeretne kapni a fejlesztőtől vagy más felhasználóktól, kérjük, látogasson el a GitHub Discussions oldalra. Új beszélgetést indíthat, csatlakozhat meglévő beszélgetésekhez, és segítséget kaphat a közösségtől.
- **Visszajelzési űrlap**: Visszajelzést küldhet nekünk a Redirect Web alkalmazás visszajelzési űrlapján keresztül. Ez közvetlenül a fejlesztőnek küld e-mailt, így visszajelzése privát marad a többi felhasználó számára.

Köszönjük, hogy a Redirect Webet használja! Nagyra értékeljük támogatását és visszajelzését.

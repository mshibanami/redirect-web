# Kapcsolat

Itt vagyunk, hogy segítsünk, ha bármilyen kérdése, visszajelzése vagy problémája van a Redirect Webbel. Íme a legjobb módok a kapcsolatfelvételre:

- **[GitHub Problémák (Issues)](https://github.com/mshibanami/RedirectWeb/issues)**: Ha bármilyen hibába ütközik, vagy új funkciókat szeretne kérni, kérjük, látogasson el a GitHub Issues oldalra. Böngészhet a meglévő problémák között, új problémát nyújthat be, és nyomon követheti a probléma előrehaladását.
- **[GitHub Beszélgetések (Discussions)](https://github.com/mshibanami/RedirectWeb/discussions)**: Ha bármilyen kérdése van, vagy segítséget szeretne kérni a fejlesztőtől vagy más felhasználóktól, kérjük, látogasson el a GitHub Discussions oldalra. Új beszélgetést indíthat, csatlakozhat meglévő beszélgetésekhez, és segítséget kaphat a közösségtől.
- **Visszajelzési űrlap**: Visszajelzést küldhet nekünk a Redirect Web alkalmazásban található visszajelzési űrlapon keresztül. Ez közvetlenül a fejlesztőnek küld e-mailt, így visszajelzése privát marad a többi felhasználó számára.

Köszönjük, hogy használja a Redirect Webet! Értékeljük támogatását és visszajelzését.

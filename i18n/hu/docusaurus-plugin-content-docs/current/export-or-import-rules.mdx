# Szabályok exportálása/importálása

## Szabályok exportálása

### Exportálás macOS rendszeren

Kattintson az ellipszis gombra vagy a Megosztás gombra az alábbi képernyőképen látható módon:

![Share on macOS](/img/share-on-macos.png)

Több szabályt is kijelölhet, és exportálhatja őket a jobb egérgombbal (másodlagos kattintás) a szabálylista oldalsávján:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Ezenkívül exportálhatja az összes szabályt a menüsorról > Fájl > Összes szabály exportálása:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Ha a szabályokat az [Einar Egilsson Redirector](https://einaregilsson.com/redirector/) programjával kompatibilis formátumban szeretné exportálni, nyomja meg az Option billentyűt (`⌥`) a helyi menüben vagy a menüsoron, ekkor a következőket fogja látni:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> A Redirect Web nem teljesen kompatibilis a Redirectorral, és fordítva. Néhány beállítás, mint például a "Replace Occurrences" a Capturing Group Processing-hez, nem exportálódik a Redirector formátumával.
> :::

## Szabályok importálása

### Importálás macOS rendszeren

- Egyszerűen húzza át az exportált szabályfájlokat a Redirect Web ablakába.
- Vagy importálhatja őket a menüsorról > Fájl > Szabályok importálása

### Importálás iOS rendszeren

Tegyük fel, hogy a szabályfájlja a Fájlok alkalmazásban van. Nyissa meg a Fájlokban, koppintson a Megosztás gombra, majd válassza a Redirect Webet:

| 1. lépés: Koppintson a Megosztás gombra a Fájlokban | 2. lépés: Válassza a Redirect Webet |

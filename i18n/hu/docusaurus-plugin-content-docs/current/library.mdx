# Könyvtár

Üdvözöljük könyvtárunkban, ahol ingyenesen találhat hasznos, azonnal használható szabályokat! Ezeket a Redirect Web alkalmazáson keresztül érheti el.

![Könyvtár képernyőképe](/img/library-screenshot.webp)

To copy the rules to My Rules, you can simply select Get button:

![Letöltés gomb - iOS](/img/library-get-button-ios.gif)

## Hozzájárulás a könyvtárhoz

A jelenlegi könyvtárban csak néhány szabály található, ezért nagyra értékeljük hozzájárulását!

Két lehetőség van a szabályok könyvtárhoz való hozzájárulására:

1. **Tegye közzé szabályát a Fórumon**: Ossza meg átirányítási szabályát más felhasználókkal [a fórumon](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Ezután kérhetjük, hogy tegyük be a szabályát a könyvtárba is. Ez könnyebb lehet, mint a 2. opció, mert ránk bízhatja a hozzáadást.
2. **Küldjön nekünk egy pull requestet**: Küldhet nekünk egy pull requestet. Kérjük, olvassa el [az utasításokat](https://github.com/mshibanami/redirect-web/tree/main/library) további információkért.

## Egyéb információk

- Az összes tartalmat megtekintheti [a GitHubon található tárolóban](https://github.com/mshibanami/redirect-web/tree/main/library).
- Privát API szerverünket a Cloudflare-en üzemeltetjük a könyvtári tartalmakhoz, lehetővé téve számunkra a szűrés és egyéb szolgáltatások nyújtását. Az alkalmazás erről a szerverről tölti le a könyvtári tartalmakat. Ahogy [adatvédelmi irányelveinkben](/privacy-policy.mdx) is szerepel, ez a szerver nem gyűjt személyes adatokat.

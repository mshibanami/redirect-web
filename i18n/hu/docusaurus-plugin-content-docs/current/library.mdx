# Könyvtár

Üdvözöljük könyvtárunkban, ahol hasznos, azonnal használható szabályokat talál ingyenesen! Ezeket a Redirect Web alkalmazáson keresztül érheti el.

![Könyvtár képernyőképe](/img/library-screenshot.webp)

A szabályok Saját szabályokhoz másolásához egyszerűen válassza a Get gombot:

![Get gomb - iOS](/img/library-get-button-ios.gif)

## Hozzájárulás a könyvtárhoz

A jelenlegi könyvtárban csak néhány szabály található, ezért nagyra értékeljük hozzájárulását!

2 lehetőség van a szabályok könyvtárhoz való hozzájárulására:

1. **Tegye közzé szabályát a Fórumon**: Ossza meg átirányítási szabályát más felhasználókkal [a fórumon](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Ezután kérhetjük, hogy tegyük be a szabályát a könyvtárba is. Ez könnyebb lehet, mint a 2. opció, mert ránk bízhatja a hozzáadást.
2. **Küldjön nekünk egy pull requestet**: Küldhet nekünk egy pull requestet. Kérjük, olvassa el [az utasításokat](https://github.com/mshibanami/redirect-web/tree/main/library) további információkért.

## Egyéb információk

- Az összes tartalmat megtekintheti [a GitHubon lévő tárolóban](https://github.com/mshibanami/redirect-web/tree/main/library).
- A könyvtár tartalmaihoz a Cloudflare-en üzemeltetjük privát API szerverünket, amely lehetővé teszi a szűrést és egyéb szolgáltatásokat. Az alkalmazás erről a szerverről tölti le a könyvtár tartalmát. Ahogy [adatvédelmi irányelveinkben](privacy-policy.mdx) is szerepel, ez a szerver nem gyűjt személyes adatokat.

# Könyvtár

Üdvözöljük könyvtárunkban, ahol hasznos, készen használható szabályokat találhat ingyen! Ezeket a Redirect Web alkalmazáson keresztül érheti el.

![Library Screenshot](/img/library-screenshot.webp)

A szabályok „Saját szabályok” közé másolásához egyszerűen válassza a „Get” gombot:

![Get Button - iOS](/img/library-get-button-ios.gif)

## Hozzájárulás szabályokkal a könyvtárhoz

A jelenlegi könyvtárban még csak kevés szabály található, ezért nagyra értékeljük hozzájárulását!

Két lehetőség van a szabályok beküldésére a könyvtárba:

1.  **Tedd közzé a szabályodat a Fórumon**: Oszd meg az átirányítási szabályodat más felhasználókkal [a fórumon](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Ezután kérhetjük, hogy a szabályodat mi is felvehessük a könyvtárba. Ez könnyebb lehet, mint a 2. opció, mert a hozzáadást ránk bízhatod.
2.  **Küldj nekünk egy pull requestet**: Küldhetsz nekünk egy pull requestet. Kérjük, tekintsd meg [az utasításokat](https://github.com/mshibanami/redirect-web/tree/main/library) további információkért.

## Egyéb információk

-   Minden tartalmat megtekinthet [a GitHub-tárhelyen](https://github.com/mshibanami/redirect-web/tree/main/library).
-   A könyvtári tartalmakhoz egy privát API szervert üzemeltetünk a Cloudflare-en, ami lehetővé teszi számunkra a szűrési és egyéb szolgáltatások nyújtását. Az alkalmazás erről a szerverről kéri le a könyvtár tartalmait. Ahogy [adatvédelmi irányelveinkben](privacy-policy.mdx) is szerepel, ez a szerver nem gyűjt személyes információkat.

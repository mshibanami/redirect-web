# Könyvtár

Üdvözöljük könyvtárunkban, ahol hasznos, azonnal használható szabályokat talál ingyen! Ezeket a Redirect Web alkalmazáson keresztül érheti el.

![Get Button - iOS](/img/library-get-button-ios.gif)

## Adja hozzá szabályát a könyvtárhoz

A jelenlegi könyvtárban még kevés szabály található, ezért nagyra értékeljük hozzájárulását!

2 lehetőség van szabályainak a könyvtárhoz való hozzáadására:

1. **Tegye közzé szabályát a Fórumon**: Ossza meg átirányítási szabályát más felhasználókkal a [fórumon](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Ezután megkérdezhetjük, hogy felvehetjük-e a szabályát a könyvtárba is. Ez könnyebb lehet, mint a 2. opció, mert a hozzáadást ránk bízhatja.
2. **Küldjön nekünk egy pull requestet**: Küldhet nekünk egy pull requestet. Kérjük, tekintse meg az [útmutatót](https://github.com/mshibanami/redirect-web/tree/main/library) további információkért.

## Egyéb információk

- Minden tartalom megtekinthető a [GitHubon lévő adattárban](https://github.com/mshibanami/redirect-web/tree/main/library).
- A könyvtár tartalmaihoz egy privát API szervert üzemeltetünk a Cloudflare-en, ami lehetővé teszi számunkra a szűrési és egyéb szolgáltatások nyújtását. Az alkalmazás erről a szerverről tölti le a könyvtár tartalmait. Ahogy az [adatvédelmi irányelveinkben](/privacy-policy) is szerepel, ez a szerver nem gyűjti az Ön személyes adatait.

# Hibaelhárítás

Ez az oldal a Redirect Web használata során felmerülő gyakori problémákat és azok megoldásait sorolja fel.

## Az átirányítási szabályok nem működnek

Ha a szabályai nem a várt módon működnek, kérjük, ellenőrizze a következőket:

- **A böngészőbővítmény telepítve van és engedélyezve van a megfelelő oldalhoz:** Győződjön meg arról, hogy telepítette a Redirect Web böngészőbővítményt, és engedélyezve van a böngészőjében ahhoz a weboldalhoz, ahonnan átirányítani szeretne. Ha a bővítmény felugró ablakát így meg tudja nyitni a forrás weboldalon, akkor helyesen van telepítve:
    <Image src={ExtensionPopup} alt="Redirect Web bővítmény felugró ablaka" width="260" hasShadow/>

- **A szabálykonfiguráció helyes:** Ellenőrizze újra az átirányítási szabályokat a Redirect Web natív alkalmazásban. Győződjön meg arról, hogy a forrás URL-minták, cél URL-ek és bármely feltétel helyesen van beállítva. Annak ellenőrzéséhez, hogy a szabály érvényes-e, írjon be egy példa URL-t a Szabály szerkesztése képernyő Példák szakaszába. Ha a szabály érvényes, akkor a bemenet alatt látni fogja a várható cél URL-t, így:
    <Image src={RuleEditExampleResult} alt="Példa a szabály szerkesztési képernyőre, amely az URL-példa eredményét mutatja" width="260" hasShadow/>

## A bővítmény felugró ablaka hibát jelez

<Image src={ErrorOverlay} alt="Redirect Web hiba overlay" width="260" hasShadow hasRoundedCorners/>

> A Redirect Web alkalmazás nincs telepítve, vagy nem működik megfelelően.
> Kérjük, szerezze be az alkalmazást az App Store-ból.

Ha a fenti hibát látja a bővítmény felugró ablakának megnyitásakor, az azt jelenti, hogy a Redirect Web natív alkalmazás nincs telepítve vagy nem fut megfelelően.

Kérjük, győződjön meg arról, hogy a Redirect Web legújabb verzióját telepítette az [App Store-ból](https://apps.apple.com/app/id1571283503).

Előfordulhat, hogy újra kell indítania a böngészőjét a natív alkalmazás telepítése után.

> [!NOTE]
> * A böngészőbővítmény használatához telepítenie kell a Redirect Web natív alkalmazást. A bővítmény önmagában nem működik.
> * A Safari felhasználóknak nem kell külön telepíteniük a bővítményt, mivel az a natív alkalmazással együtt van csomagolva.

## Az eredeti oldal röviden felvillan az átirányítás előtt

Ez akkor fordul elő, ha **Eredeti** típusú átirányítási szabályokat használ. Ez a típus különféle fejlett funkciókat biztosít, de az eredeti oldal rövid felvillanását okozhatja, mert lehetővé teheti az eredeti oldal betöltését az átirányítás végrehajtása előtt.

Nyissa meg a Szabály szerkesztése képernyőt, és módosítsa a **Típus** beállítást **DNR**-re a probléma elkerülése érdekében.

<Image src={EditRuleDnr} alt="Redirect Web bővítmény felugró ablaka" height="260" hasShadow hasRoundedCorners/>

> [!NOTE]
> Firefoxban ez a probléma még az Eredeti típussal sem fordulhat elő, mivel az a WebRequest API-ra támaszkodik, amely megakadályozza az eredeti oldal betöltését.

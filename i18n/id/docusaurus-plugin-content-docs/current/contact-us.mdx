# Hubungi Kami

Kami siap membantu jika Anda memiliki pertanyaan, masukan, atau masalah dengan Redirect Web. Berikut adalah cara terbaik untuk menghubungi kami:

- **[Masalah GitHub](https://github.com/mshibanami/RedirectWeb/issues)**: Jika Anda menemukan bug atau ingin meminta fitur baru, silakan kunjungi halaman Masalah GitHub. Anda dapat menelusuri masalah yang ada, mengirimkan masalah baru, dan melacak kemajuan masalah Anda.
- **[Diskusi GitHub](https://github.com/mshibanami/RedirectWeb/discussions)**: Jika Anda memiliki pertanyaan atau ingin mendapatkan bantuan dari pengembang atau pengguna lain, silakan kunjungi halaman Diskusi GitHub. Anda dapat memulai diskusi baru, bergabung dengan diskusi yang ada, dan mendapatkan bantuan dari komunitas.
- **Formulir Umpan Balik**: Anda dapat mengirimkan umpan balik kepada kami melalui formulir umpan balik di aplikasi Redirect Web. Ini akan mengirimkan email langsung ke pengembang, sehingga umpan balik Anda tetap pribadi dari pengguna lain.

Terima kasih telah menggunakan Redirect Web! Kami menghargai dukungan dan masukan Anda.

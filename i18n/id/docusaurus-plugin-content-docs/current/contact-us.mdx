# Hubungi Kami

Kami siap membantu jika Anda memiliki pertanyaan, umpan balik, atau masalah dengan Redirect Web. Berikut adalah cara terbaik untuk menghubungi kami:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Jika Anda menemukan bug atau ingin meminta fitur baru, silakan kunjungi halaman GitHub Issues. Anda dapat menelusuri masalah yang ada, mengajukan masalah baru, dan melacak kemajuan masalah Anda.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Jika Anda memiliki pertanyaan atau ingin mendapatkan bantuan dari pengembang atau pengguna lain, silakan kunjungi halaman GitHub Discussions. Anda dapat memulai diskusi baru, bergabung dengan diskusi yang sudah ada, dan mendapatkan bantuan dari komunitas.
- **Formulir Umpan Balik**: Anda dapat mengirimkan umpan balik melalui formulir umpan balik di aplikasi Redirect Web. Ini akan mengirimkan email langsung ke pengembang, sehingga umpan balik Anda akan tetap bersifat pribadi dari pengguna lain.

Terima kasih telah menggunakan Redirect Web! Kami menghargai dukungan dan umpan balik Anda.

# Aturan Ekspor/Impor

## Aturan Ekspor {#export-rules}

### Ekspor di macOS {#export-on-macos}

Klik tombol elipsis atau tombol Bagikan seperti yang ditunjukkan pada tangkapan layar ini:

![Share on macOS](/img/share-on-macos.png)

Anda juga dapat memilih beberapa aturan dan mengekspornya dengan mengklik sekunder (klik kanan) pada bilah samping daftar aturan:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Selain itu, Anda dapat mengekspor semua aturan dari bilah menu > File > Export All Rules:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Jika Anda ingin mengekspor aturan Anda dalam format yang kompatibel dengan [Redirector Einar Egilsson](https://einaregilsson.com/redirector/), tekan tombol Option (`âŒ¥`) pada menu konteks atau bilah menu, lalu Anda akan melihat ini:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web tidak sepenuhnya kompatibel dengan Redirector, dan sebaliknya. Beberapa pengaturan, seperti "Ganti Kemunculan" untuk Pemrosesan Grup Penangkapan, tidak diekspor dengan format Redirector.
> :::

## Aturan Impor {#import-rules}

### Impor di macOS {#import-on-macos}

- Cukup seret dan lepas file aturan yang diekspor ke jendela Redirect Web.
- Atau, Anda juga dapat mengimpornya dari bilah menu > File > Import Rules

### Impor di iOS {#import-on-ios}

Misalkan file aturan Anda ada di aplikasi File. Buka di File, ketuk tombol Bagikan, lalu pilih Redirect Web:

| Langkah 1: Ketuk tombol Bagikan di File                                     | Langkah 2: Pilih Redirect Web                                         |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Tap the share button in the toolbar](/img/files-share-ios.png) | ![Tap the app icon in the sheet](/img/share-sheet-ios.png) |

# Aturan Ekspor/Impor

## Aturan Ekspor

### Ekspor di macOS

Klik tombol elipsis atau tombol Bagikan seperti yang ditunjukkan pada tangkapan layar ini:

![Share on macOS](/img/share-on-macos.png)

Anda juga bisa memilih beberapa aturan dan mengekspornya dengan klik sekunder (klik kanan) pada bilah sisi daftar aturan:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Selain itu, Anda bisa mengekspor semua aturan dari bilah menu > File > Ekspor Semua Aturan:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Jika Anda ingin mengekspor aturan dalam format yang kompatibel dengan [Redirector Einar Egilsson](https://einaregilsson.com/redirector/), tekan tombol Option (`âŒ¥`) pada menu konteks atau bilah menu, maka Anda akan melihat ini:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web tidak sepenuhnya kompatibel dengan Redirector, dan sebaliknya. Beberapa pengaturan, seperti "Ganti Kemunculan" untuk Pemrosesan Grup Penangkapan, tidak diekspor dengan format Redirector.
> :::

## Impor Aturan

### Impor di macOS

- Cukup seret dan lepas file aturan yang diekspor ke jendela Redirect Web.
- Atau, Anda juga bisa mengimpornya dari bilah menu > File > Impor Aturan

### Impor di iOS

Misalkan file aturan Anda ada di aplikasi Files. Buka di Files, ketuk tombol Bagikan, lalu pilih Redirect Web:

| Langkah 1: Ketuk tombol Bagikan di Files | Langkah 2: Pilih Redirect Web         |

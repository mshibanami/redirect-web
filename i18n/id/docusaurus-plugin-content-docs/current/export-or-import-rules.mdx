# Ekspor/Impor Aturan

## Ekspor Aturan {#export-rules}

### Ekspor di macOS {#export-on-macos}

Klik tombol elipsis atau tombol Bagikan seperti yang ditunjukkan pada tangkapan layar ini:

<Image src={ShareOnMacOS} alt="Tombol Bagikan di macOS" width="460" hasShadow hasRoundedCorners/>

Kemudian, pilih "Ekspor Aturan" atau "Bagikan Aturan..." dari menu konteks:

Anda juga dapat memilih beberapa aturan dan mengekspornya dengan mengklik sekunder (klik kanan) pada bilah sisi daftar aturan:

<Image src={MultiSelectOnMacOS} alt="Bagikan beberapa aturan di macOS" height="400" hasShadow hasRoundedCorners/>


Selain itu, Anda dapat mengekspor semua aturan dari bilah menu > File > Ekspor Semua Aturan:

<Image src={MenubarFile} alt="Bilah Menu > File" width="400" hasShadow hasRoundedCorners/>

## Impor Aturan {#import-rules}

### Impor di macOS {#import-on-macos}

- Cukup seret dan lepas file aturan yang diekspor ke jendela RedirectWeb.
- Atau, Anda juga dapat mengimpornya dari bilah menu > File > Impor Aturan

### Impor di iOS {#import-on-ios}

Misalkan file aturan Anda ada di aplikasi File. Buka di File, ketuk tombol Bagikan, lalu pilih RedirectWeb:

| Langkah 1: Ketuk tombol Bagikan di File | Langkah 2: Pilih RedirectWeb |

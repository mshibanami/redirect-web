## Kasus Penggunaan Populer {#popular-use-cases}

Misalnya, Anda dapat membuat aturan ini:

- Melihat **Reddit** dalam tampilan klasiknya (old.reddit.com).
- Buka **Apple Maps** setiap kali terdapat tautan **Google Maps**.
- Sembunyikan ringkasan AI Google dari **hasil pencarian Google** dengan menambahkan **?udm=14** pada setiap pencarian.
- Melompat dari halaman Fandom ke wiki resmi **Terraria** atau **Minecraft**.
- Gunakan **Kagi**, **Startpage**, **ChatGPT**, atau mesin pencari lainnya sebagai **mesin pencari default** Anda.
- Buka tautan **Google Meet** di Google Chrome. (Mac)
- Buka tautan **Figma** langsung di aplikasi Figma. (Mac)
- Buka tautan **Notion** langsung di aplikasi Notion. (Mac)

Anda bisa mendapatkan semua ini sebagai aturan pra-definisi secara gratis dari bagian Library di aplikasi!

## Fitur {#features}

Redirect Web menawarkan fitur-fitur berikut:

- Membuat aturan menggunakan tipe pola **Regular Expression** atau **Wildcard**
- Menggunakan **back-referencing** untuk menentukan URL target dinamis
- Mengecualikan URL dari pengalihan
- Memproses teks yang ditangkap menggunakan URL encoding/decoding, base64 encoding/decoding, atau mengganti kemunculan

Selain itu, Anda dapat membuka fitur berikut melalui langganan atau pembelian sekali bayar:

- Mengaktifkan jumlah aturan **tak terbatas**
- **Menyinkronkan aturan** dengan Mac, iPhone, dan iPad Anda **melalui iCloud**

(Membeli aplikasi juga membantu kami dalam pemeliharaan Redirect Web. üôè)

Pembelian atau langganan Anda dibagikan ke Mac, iPhone, iPad, dan Apple Vision Pro melalui Universal Purchase. Dukungan Anda membantu pengembangan berkelanjutan.

# Perpustakaan

Selamat datang di perpustakaan kami, tempat Anda dapat menemukan aturan siap pakai yang berguna secara gratis! Anda dapat mengaksesnya melalui aplikasi Redirect Web.

![Get Button - iOS](/img/library-get-button-ios.gif)

## Sumbangkan aturan Anda ke perpustakaan

Perpustakaan saat ini hanya memiliki beberapa aturan, jadi kami menghargai kontribusi Anda!

Ada 2 opsi untuk menyumbangkan aturan Anda ke perpustakaan:

1.  **Posting aturan Anda di Forum**: Bagikan aturan pengalihan Anda dengan pengguna lain di [forum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Kemudian kami mungkin akan meminta izin apakah kami dapat menempatkan aturan Anda di perpustakaan juga. Ini mungkin lebih mudah daripada opsi 2 karena Anda dapat menyerahkan penambahannya kepada kami.
2.  **Kirimkan pull request kepada kami**: Anda dapat mengirimkan pull request kepada kami. Silakan periksa [petunjuknya](https://github.com/mshibanami/redirect-web/tree/main/library) untuk mempelajari lebih lanjut.

## Informasi Lain

-   Anda dapat melihat semua konten di [repositori di GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
-   Kami menghosting server API pribadi kami di Cloudflare untuk konten perpustakaan, memungkinkan kami menyediakan penyaringan dan layanan lainnya. Aplikasi mengambil konten perpustakaan dari server ini. Seperti yang dinyatakan dalam [kebijakan privasi kami](privacy-policy.mdx), server ini tidak mengumpulkan informasi pribadi Anda.

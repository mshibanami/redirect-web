# Perpustakaan

Selamat datang di perpustakaan kami, tempat Anda dapat menemukan aturan siap pakai yang berguna secara gratis! Anda dapat mengaksesnya melalui aplikasi Redirect Web.

![Tangkapan Layar Perpustakaan](/img/library-screenshot.webp)

Untuk menyalin aturan ke Aturan Saya, Anda cukup memilih tombol Dapatkan:

![Tombol Dapatkan - iOS](/img/library-get-button-ios.gif)

## Kontribusikan aturan Anda ke perpustakaan

Perpustakaan saat ini hanya memiliki beberapa aturan, jadi kami menghargai kontribusi Anda!

Ada 2 opsi untuk menyumbangkan aturan Anda ke perpustakaan:

1. **Posting aturan Anda di Forum**: Bagikan aturan pengalihan Anda dengan pengguna lain di [forum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Kemudian kami dapat meminta apakah kami dapat memasukkan aturan Anda ke perpustakaan juga. Ini mungkin lebih mudah daripada opsi 2 karena Anda dapat menyerahkan penambahan kepada kami.
2. **Kirimkan kami permintaan tarik**: Anda dapat mengirimkan kami permintaan tarik. Silakan periksa [instruksi](https://github.com/mshibanami/redirect-web/tree/main/library) untuk mempelajari lebih lanjut.

## Informasi Lainnya

- Anda dapat melihat semua konten di [repositori di GitHub](https://github.mshibanami/redirect-web/tree/main/library).
- Kami meng-host server API pribadi kami di Cloudflare untuk konten perpustakaan, memungkinkan kami menyediakan pemfilteran dan layanan lainnya. Aplikasi mengambil konten perpustakaan dari server ini. Seperti yang dinyatakan dalam [kebijakan privasi kami](/privacy-policy.mdx), server ini tidak mengumpulkan informasi pribadi Anda.

# Pustaka

Selamat datang di pustaka kami, tempat Anda dapat menemukan aturan siap pakai yang berguna secara gratis! Anda dapat mengaksesnya melalui aplikasi Web Redirect.

## Sumbangkan aturan Anda ke pustaka

Pustaka saat ini hanya memiliki sedikit aturan, jadi kami sangat menghargai kontribusi Anda!

Ada 2 opsi untuk menyumbangkan aturan Anda ke pustaka:

1.  **Posting aturan Anda di Forum**: Bagikan aturan pengalihan Anda dengan pengguna lain di [forum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Kemudian kami mungkin akan meminta izin apakah kami juga bisa memasukkan aturan Anda ke pustaka. Ini mungkin lebih mudah daripada opsi 2 karena Anda bisa menyerahkan penambahannya kepada kami.
2.  **Kirimkan *pull request* kepada kami**: Anda dapat mengirimkan *pull request* kepada kami. Silakan periksa [instruksinya](https://github.com/mshibanami/redirect-web/tree/main/library) untuk mempelajari lebih lanjut.

## Informasi Lain

-   Anda dapat melihat semua konten di [repositori di GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
-   Kami meng-host server API pribadi kami di Cloudflare untuk konten pustaka, memungkinkan kami menyediakan penyaringan dan layanan lainnya. Aplikasi mengambil konten pustaka dari server ini. Seperti yang dinyatakan dalam [kebijakan privasi kami](/privacy-policy), server ini tidak mengumpulkan informasi pribadi Anda.

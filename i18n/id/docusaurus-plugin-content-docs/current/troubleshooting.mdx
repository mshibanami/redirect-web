# Pemecahan Masalah

Halaman ini mencantumkan masalah umum dan solusinya saat menggunakan Redirect Web.

<TOCInline toc={toc} />

## Aturan pengalihan tidak berfungsi

Jika aturan Anda tidak berfungsi seperti yang diharapkan, harap periksa hal berikut:

- **Ekstensi browser terinstal dan diaktifkan untuk halaman yang benar:** Pastikan Anda telah menginstal ekstensi browser Redirect Web dan diaktifkan di browser Anda untuk halaman web yang ingin Anda alihkan. Jika Anda dapat membuka popup ekstensi seperti ini di halaman web sumber, itu terinstal dengan benar:
    <Image src={ExtensionPopup} alt="Popup ekstensi Redirect Web" width="260" hasShadow/>

- **Konfigurasi aturan sudah benar:** Periksa kembali aturan pengalihan Anda di aplikasi asli Redirect Web. Pastikan pola URL sumber, URL target, dan kondisi apa pun diatur dengan benar. Untuk memeriksa apakah aturan Anda valid atau tidak, masukkan contoh URL di bagian Contoh di layar Edit Aturan. Jika aturan Anda valid, Anda akan melihat URL tujuan yang diharapkan di bawah masukan Anda, seperti ini:
    <Image src={RuleEditExampleResult} alt="Contoh layar edit aturan yang menunjukkan hasil URL contoh" width="260" hasShadow/>

## Popup ekstensi menunjukkan kesalahan

<Image src={ErrorOverlay} alt="Overlay kesalahan Redirect Web" width="260" hasShadow hasRoundedCorners/>

> Aplikasi Redirect Web belum terinstal atau tidak berfungsi dengan baik.
> Harap dapatkan aplikasi dari App Store.

Jika Anda melihat kesalahan di atas saat membuka popup ekstensi, itu berarti aplikasi asli Redirect Web tidak terinstal atau tidak berjalan dengan baik.

Harap pastikan bahwa Anda telah menginstal versi terbaru Redirect Web dari [App Store](https://apps.apple.com/app/id1571283503).

Anda mungkin juga perlu meluncurkan ulang browser Anda setelah menginstal aplikasi asli.

> [!NOTE]
> * Anda perlu menginstal aplikasi asli Redirect Web untuk menggunakan ekstensi browser. Ekstensi saja tidak berfungsi.
> * Pengguna Safari tidak perlu menginstal ekstensi secara terpisah, karena sudah dibundel dengan aplikasi asli.

## Halaman asli berkedip sebentar sebelum pengalihan

Ini terjadi ketika Anda menggunakan aturan pengalihan tipe **Original**. Tipe ini menyediakan berbagai fitur canggih, tetapi dapat menyebabkan halaman asli berkedip sebentar karena mungkin memungkinkan halaman asli dimuat sebelum melakukan pengalihan.

Buka layar Edit Aturan dan ubah **Tipe** menjadi **DNR** untuk menghindari masalah ini.

<Image src={EditRuleDnr} alt="Popup ekstensi Redirect Web" height="260" hasShadow hasRoundedCorners/>

> [!NOTE]
> Di Firefox, masalah ini seharusnya tidak terjadi bahkan dengan tipe Original, karena mengandalkan WebRequest API, yang mencegah halaman asli dimuat.

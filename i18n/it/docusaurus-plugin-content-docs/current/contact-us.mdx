# Contattaci

Siamo qui per aiutarti se hai domande, feedback o problemi con Redirect Web. Ecco i modi migliori per metterti in contatto:

-   **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Se riscontri bug o desideri richiedere nuove funzionalità, visita la pagina GitHub Issues. Puoi consultare gli issue esistenti, inviarne uno nuovo e seguirne l'avanzamento.
-   **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Se hai domande o desideri ricevere aiuto dallo sviluppatore o da altri utenti, visita la pagina GitHub Discussions. Puoi avviare una nuova discussione, unirti a quelle esistenti e ottenere aiuto dalla community.
-   **Modulo di Feedback**: Puoi inviarci feedback tramite il modulo di feedback nell'app Redirect Web. Questo invierà un'email direttamente allo sviluppatore, in modo che il tuo feedback rimanga privato dagli altri utenti.

Grazie per aver usato Redirect Web! Apprezziamo il tuo supporto e il tuo feedback.

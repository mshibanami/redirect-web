# Contattaci

Siamo qui per aiutarti se hai domande, feedback o problemi con RedirectWeb. Ecco i modi migliori per metterti in contatto:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Se riscontri bug o desideri richiedere nuove funzionalità, visita la pagina GitHub Issues. Puoi sfogliare i problemi esistenti, inviare un nuovo problema e monitorare l'avanzamento del tuo problema.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Se hai domande o desideri ricevere aiuto dallo sviluppatore o da altri utenti, visita la pagina GitHub Discussions. Puoi avviare una nuova discussione, unirti a discussioni esistenti e ottenere aiuto dalla community.
- **Modulo di feedback**: Puoi inviarci feedback tramite il modulo di feedback nell'app RedirectWeb. Questo invierà un'e-mail direttamente allo sviluppatore, in modo che il tuo feedback rimanga privato dagli altri utenti.

Grazie per aver usato RedirectWeb! Apprezziamo il tuo supporto e feedback.

# Contattaci

Siamo qui per aiutarti se hai domande, feedback o problemi con Redirect Web. Ecco i modi migliori per metterti in contatto:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Se riscontri bug o desideri richiedere nuove funzionalità, visita la pagina GitHub Issues. Puoi sfogliare le segnalazioni esistenti, inviare una nuova segnalazione e monitorarne l'avanzamento.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Se hai domande o desideri ricevere aiuto dallo sviluppatore o da altri utenti, visita la pagina GitHub Discussions. Puoi avviare una nuova discussione, unirti a discussioni esistenti e ricevere aiuto dalla community.
- **Modulo di feedback**: Puoi inviarci il tuo feedback tramite il modulo di feedback nell'app Redirect Web. Questo invierà un'e-mail direttamente allo sviluppatore, quindi il tuo feedback rimarrà privato e non sarà visibile ad altri utenti.

Grazie per aver usato Redirect Web! Apprezziamo il tuo supporto e il tuo feedback.

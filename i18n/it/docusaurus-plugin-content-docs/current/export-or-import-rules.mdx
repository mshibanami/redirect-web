# Esportazione/Importazione Regole

## Esportazione Regole

### Esportazione su macOS

Fare clic sul pulsante con i puntini di sospensione o sul pulsante Condividi come mostrato in questo screenshot:

![Share on macOS](/img/share-on-macos.png)

Puoi anche selezionare più regole ed esportarle facendo clic con il pulsante secondario (clic destro) su di esse nella barra laterale dell'elenco delle regole:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Inoltre, puoi esportare tutte le regole dalla barra dei menu > File > Esporta tutte le regole:

![Menubar > File](/img/menubar-file.png)

> [!NOTA]
> Se desideri esportare le tue regole nel formato compatibile con [Redirector di Einar Egilsson](https://einaregilsson.com/redirector/), premi il tasto Opzione (`⌥`) nel menu contestuale o nella barra dei menu, quindi vedrai questo:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::avviso
> Redirect Web non è completamente compatibile con Redirector, e viceversa. Alcune impostazioni, come "Sostituisci Occorrenze" per l'Elaborazione del Gruppo di Cattura, non vengono esportate con il formato di Redirector.
> :::

## Importazione Regole

### Importazione su macOS

- Basta trascinare e rilasciare i file di regole esportati nella finestra di Redirect Web.
- Oppure, puoi anche importarli dalla barra dei menu > File > Importa regole

### Importazione su iOS

Supponiamo che il tuo file di regole si trovi nell'app File. Aprilo in File, tocca il pulsante Condividi, quindi seleziona Redirect Web:

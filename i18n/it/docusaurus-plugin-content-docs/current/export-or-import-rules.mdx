# Esporta/Importa Regole

## Esporta Regole {#export-rules}

### Esporta su macOS {#export-on-macos}

Fai clic sul pulsante con i puntini di sospensione o sul pulsante Condividi come mostrato in questo screenshot:

![Condividi su macOS](/img/share-on-macos.png)

Puoi anche selezionare più regole ed esportarle facendo clic secondario (clic destro) su di esse nella barra laterale dell'elenco delle regole:

![Condividi più regole su macOS](/img/multiselect-on-macos.png)

Inoltre, puoi esportare tutte le regole dalla barra dei menu > File > Esporta tutte le regole:

![Barra dei menu > File](/img/menubar-file.png)

> [!NOTE]
> Se desideri esportare le tue regole nel formato compatibile con [Redirector di Einar Egilsson](https://einaregilsson.com/redirector/), premi il tasto Opzione (`⌥`) sul menu contestuale o sulla barra dei menu, quindi vedrai questi:
> ![Barra dei menu > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Menu contestuale (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web non è completamente compatibile con Redirector e viceversa. Alcune impostazioni, come "Sostituisci occorrenze" per l'elaborazione dei gruppi di acquisizione, non vengono esportate con il formato di Redirector.
> :::

## Importa Regole {#import-rules}

### Importa su macOS {#import-on-macos}

- Semplicemente trascina e rilascia i file delle regole esportate nella finestra di Redirect Web.
- Oppure, puoi anche importarli dalla barra dei menu > File > Importa regole

### Importa su iOS {#import-on-ios}

Supponiamo che il tuo file di regole si trovi nell'app File. Aprilo in File, tocca il pulsante Condividi, quindi seleziona Redirect Web:

| Passaggio 1: Tocca il pulsante Condividi nella barra degli strumenti | Passaggio 2: Seleziona Redirect Web |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Tocca il pulsante di condivisione nella barra degli strumenti](/img/files-share-ios.png) | ![Tocca l'icona dell'app nel foglio](/img/share-sheet-ios.png) |

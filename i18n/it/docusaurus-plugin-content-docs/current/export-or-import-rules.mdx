# Esporta/Importa Regole

## Esporta Regole {#export-rules}

### Esporta su macOS {#export-on-macos}

Fai clic sul pulsante con i puntini di sospensione o sul pulsante Condividi come mostrato in questo screenshot:

<Image src={ShareOnMacOS} alt="Pulsante Condividi su macOS" width="460" hasShadow hasRoundedCorners/>

Quindi, seleziona "Esporta Regola" o "Condividi Regola..." dal menu contestuale:

Puoi anche selezionare più regole ed esportarle facendo clic secondario (clic destro) su di esse nella barra laterale dell'elenco delle regole:

<Image src={MultiSelectOnMacOS} alt="Condividi più regole su macOS" height="400" hasShadow hasRoundedCorners/>


Inoltre, puoi esportare tutte le regole dalla barra dei menu > File > Esporta tutte le regole:

<Image src={MenubarFile} alt="Barra dei menu > File" width="400" hasShadow hasRoundedCorners/>

> [!NOTE]
> Se desideri esportare le tue regole nel formato compatibile con [Redirector di Einar Egilsson](https://einaregilsson.com/redirector/), premi il tasto Opzione (`⌥`) nel menu contestuale o nella barra dei menu, quindi vedrai questi:
> <Image src={MenubarFileRedirector} alt="Barra dei menu > File (Redirector)" width="400" hasShadow hasRoundedCorners/>
> <br/>
> <Image src={ContextMenuRedirector} alt="Menu contestuale (Redirector)" width="400" hasShadow hasRoundedCorners/>
>
> :::warning
> Redirect Web non è completamente compatibile con Redirector, e viceversa. Alcune impostazioni, come "Sostituisci occorrenze" per l'elaborazione del gruppo di cattura, non vengono esportate con il formato di Redirector.
> :::

## Importa Regole {#import-rules}

### Importa su macOS {#import-on-macos}

- Semplicemente trascina e rilascia i file delle regole esportate nella finestra di Redirect Web.
- Oppure, puoi anche importarli dalla barra dei menu > File > Importa regole

### Importa su iOS {#import-on-ios}

Supponiamo che il tuo file di regole si trovi nell'app File. Aprilo in File, tocca il pulsante Condividi, quindi seleziona Redirect Web:

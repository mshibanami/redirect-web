# Libreria

Benvenuti nella nostra libreria, dove potete trovare utili regole pronte all'uso gratuitamente! Potete accedervi tramite l'app Redirect Web.

![Screenshot della Libreria](/img/library-screenshot.webp)

Per copiare le regole in Le mie regole, potete semplicemente selezionare il pulsante Ottieni:

![Pulsante Ottieni - iOS](/img/library-get-button-ios.gif)

## Contribuisci con la tua regola alla libreria

La libreria attuale ha solo poche regole, quindi apprezziamo il tuo contributo!

Ci sono 2 opzioni per contribuire con le tue regole alla libreria:

1. **Pubblica la tua regola sul Forum**: Condividi la tua regola di reindirizzamento con altri utenti sul [forum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Potremmo quindi chiederti se possiamo inserire la tua regola anche nella libreria. Questo potrebbe essere più facile dell'opzione 2 perché puoi lasciare l'aggiunta a noi.
2. **Inviaci una pull request**: Puoi inviarci una pull request. Controlla [le istruzioni](https://github.com/mshibanami/redirect-web/tree/main/library) per saperne di più.

## Altre informazioni

- Puoi vedere tutti i contenuti nel [repository su GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- Ospitiamo il nostro server API privato su Cloudflare per i contenuti della libreria, permettendoci di fornire filtri e altri servizi. L'app recupera i contenuti della libreria da questo server. Come indicato nella [nostra politica sulla privacy](privacy-policy.mdx), questo server non raccoglie le tue informazioni personali.

# Libreria

Benvenuti nella nostra libreria, dove potete trovare regole utili e pronte all'uso gratuitamente! Potete accedervi tramite l'app Redirect Web.

![Library Screenshot](/img/library-screenshot.webp)

Per copiare le regole in Le Mie Regole, potete semplicemente selezionare il pulsante Ottieni:

![Get Button - iOS](/img/library-get-button-ios.gif)

## Contribuire con la propria regola alla libreria

La libreria attuale ha solo poche regole, quindi apprezziamo il vostro contributo!

Ci sono 2 opzioni per contribuire con le vostre regole alla libreria:

1.  **Pubblicare la vostra regola sul Forum**: Condividete la vostra regola di reindirizzamento con altri utenti sul [forum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Potremmo poi chiedere se possiamo inserire la vostra regola anche nella libreria. Questo potrebbe essere più facile dell'opzione 2 perché potete lasciare a noi l'aggiunta.
2.  **Inviateci una pull request**: Potete inviarci una pull request. Controllate [le istruzioni](https://github.com/mshibanami/redirect-web/tree/main/library) per saperne di più.

## Altre informazioni

-   Potete vedere tutti i contenuti nel [repository su GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
-   Ospitiamo il nostro server API privato su Cloudflare per i contenuti della libreria, permettendoci di fornire filtri e altri servizi. L'app recupera i contenuti della libreria da questo server. Come indicato nella [nostra politica sulla privacy](privacy-policy.mdx), questo server non raccoglie le vostre informazioni personali.

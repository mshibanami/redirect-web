# Risoluzione dei problemi

Questa pagina elenca i problemi comuni e le relative soluzioni quando si utilizza Redirect Web.

<TOCInline toc={toc} />

## Le regole di reindirizzamento non funzionano

Se le tue regole non funzionano come previsto, controlla quanto segue:

- **L'estensione del browser è installata e abilitata per la pagina corretta:** Assicurati di aver installato l'estensione del browser Redirect Web e che sia abilitata nel tuo browser per la pagina web da cui desideri reindirizzare. Se puoi aprire il popup dell'estensione in questo modo sulla pagina web di origine, è installata correttamente:
    <Image src={ExtensionPopup} alt="Popup dell'estensione Redirect Web" width="260" hasShadow/>

- **La configurazione della regola è corretta:** Ricontrolla le tue regole di reindirizzamento nell'app nativa Redirect Web. Assicurati che i modelli di URL di origine, gli URL di destinazione e le eventuali condizioni siano impostati correttamente. Per verificare se la tua regola è valida o meno, inserisci un URL di esempio nella sezione Esempi nella schermata Modifica regola. Se la tua regola è valida, vedrai l'URL di destinazione previsto sotto il tuo input, in questo modo:
    <Image src={RuleEditExampleResult} alt="Un esempio di schermata di modifica della regola che mostra il risultato dell'URL di esempio" width="260" hasShadow/>

## Il popup dell'estensione mostra un errore

<Image src={ErrorOverlay} alt="Sovrapposizione di errore di Redirect Web" width="260" hasShadow hasRoundedCorners/>

> L'app Redirect Web non è stata installata o non funziona correttamente.
> Si prega di scaricare l'app dall'App Store.

Se vedi l'errore sopra quando apri il popup dell'estensione, significa che l'app nativa Redirect Web non è installata o non è in esecuzione correttamente.

Assicurati di aver installato l'ultima versione di Redirect Web dall'[App Store](https://apps.apple.com/app/id1571283503).

Potrebbe anche essere necessario riavviare il browser dopo aver installato l'app nativa.

> [!NOTE]
> * È necessario installare l'app nativa Redirect Web per utilizzare l'estensione del browser. L'estensione da sola non funziona.
> * Gli utenti di Safari non devono installare l'estensione separatamente, poiché è inclusa nell'app nativa.

## La pagina originale lampeggia brevemente prima del reindirizzamento

Questo accade quando si utilizzano le regole di reindirizzamento di tipo **Originale**. Questo tipo offre varie funzionalità avanzate, ma potrebbe causare un breve lampeggio della pagina originale perché potrebbe consentire il caricamento della pagina originale prima di eseguire il reindirizzamento.

Apri la schermata Modifica regola e cambia il **Tipo** in **DNR** per evitare questo problema.

<Image src={EditRuleDnr} alt="Popup dell'estensione Redirect Web" height="260" hasShadow hasRoundedCorners/>

> [!NOTE]
> In Firefox, questo problema non dovrebbe verificarsi anche con il tipo Originale, poiché si basa sull'API WebRequest, che impedisce il caricamento della pagina originale.

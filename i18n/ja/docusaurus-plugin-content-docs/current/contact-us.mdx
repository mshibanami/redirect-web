# お問い合わせ

Redirect Webに関するご質問、フィードバック、または問題がある場合は、お気軽にお問い合わせください。お問い合わせの最適な方法を以下にご紹介します。

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: バグを発見した場合や新機能のリクエストがある場合は、GitHub Issuesのページをご覧ください。既存のIssueを閲覧したり、新しいIssueを提出したり、ご自身のIssueの進捗状況を追跡したりできます。
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: ご質問がある場合や、開発者または他のユーザーからの助けが必要な場合は、GitHub Discussionsのページをご覧ください。新しい議論を開始したり、既存の議論に参加したり、コミュニティから助けを得ることができます。
- **フィードバックフォーム**: Redirect Webアプリ内のフィードバックフォームからフィードバックを送信できます。これは開発者に直接メールを送信するため、あなたのフィードバックは他のユーザーには非公開に保たれます。

Redirect Webをご利用いただきありがとうございます！皆様からのご支援とフィードバックに感謝いたします。

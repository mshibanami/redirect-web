# お問い合わせ

Redirect Webに関するご質問、フィードバック、問題がある場合は、お気軽にお問い合わせください。ご連絡いただくための最適な方法は以下の通りです。

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: バグに遭遇した場合や新機能のリクエストがある場合は、GitHub Issuesページをご覧ください。既存のIssueを閲覧したり、新しいIssueを提出したり、Issueの進捗を追跡したりできます。
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: ご質問がある場合や、開発者や他のユーザーからの助けを借りたい場合は、GitHub Discussionsページをご覧ください。新しいディスカッションを開始したり、既存のディスカッションに参加したり、コミュニティから助けを得ることができます。
- **フィードバックフォーム**: Redirect Webアプリ内のフィードバックフォームを通じてフィードバックを送信できます。これは開発者に直接メールを送信するため、あなたのフィードバックは他のユーザーには非公開に保たれます。

Redirect Webをご利用いただきありがとうございます！皆様のサポートとフィードバックに感謝いたします。

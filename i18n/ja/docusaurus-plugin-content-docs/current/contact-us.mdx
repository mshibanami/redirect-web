# お問い合わせ

Redirect Webに関するご質問、フィードバック、問題がございましたら、お気軽にお問い合わせください。ご連絡いただくには、以下の方法が最適です。

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: バグに遭遇した場合や新機能のリクエストがある場合は、GitHub Issuesページをご覧ください。既存のIssueを閲覧したり、新しいIssueを提出したり、Issueの進捗状況を追跡したりできます。
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: ご質問がある場合や、開発者や他のユーザーから助けを得たい場合は、GitHub Discussionsページをご覧ください。新しいディスカッションを開始したり、既存のディスカッションに参加したり、コミュニティから助けを得ることができます。
- **フィードバックフォーム**: Redirect Webアプリ内のフィードバックフォームからフィードバックを送信できます。これにより、開発者に直接メールが送信されるため、あなたのフィードバックは他のユーザーから非公開に保たれます。

Redirect Webをご利用いただきありがとうございます！皆様のサポートとフィードバックに感謝いたします。

# ルールのエクスポート / インポート

## ルールのエクスポート

### macOSでのエクスポート

このスクリーンショットに示されているように、省略記号ボタンまたは共有ボタンをクリックします。

![Share on macOS](/img/share-on-macos.png)

また、ルールを複数選択し、ルールリストのサイドバーで副ボタンクリック（右クリック）することでエクスポートできます。

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

さらに、メニューバー > ファイル > すべてのルールをエクスポート からすべてのルールをエクスポートできます。

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> [Einar Egilsson の Redirector](https://einaregilsson.com/redirector/) と互換性のある形式でルールをエクスポートしたい場合は、コンテキストメニューまたはメニューバーで Option キー (`⌥`) を押すと、これらが表示されます。
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web は Redirector と完全な互換性がなく、またその逆も同様です。キャプチャグループ処理の「出現箇所を置換」など、一部の設定は Redirector の形式ではエクスポートされません。
> :::

## ルールのインポート

### macOSでのインポート

- エクスポートされたルールファイルを Redirect Web のウィンドウにドラッグ＆ドロップするだけです。
- または、メニューバー > ファイル > ルールをインポート からインポートすることもできます。

### iOSでのインポート

ルールファイルが「ファイル」アプリにあるとします。「ファイル」アプリでそれを開き、共有ボタンをタップして、「Redirect Web」を選択します。

| ステップ1: 「ファイル」アプリで共有ボタンをタップ | ステップ2: 「Redirect Web」を選択 |

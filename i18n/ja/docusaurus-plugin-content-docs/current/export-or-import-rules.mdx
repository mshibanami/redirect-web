# エクスポート/インポートのルール

## ルールのエクスポート {#export-rules}

### macOSでのエクスポート {#export-on-macos}

このスクリーンショットに示すように、省略記号ボタンまたは共有ボタンをクリックします。

![Share on macOS](/img/share-on-macos.png)

ルール一覧のサイドバーで、ルールを複数選択し、副ボタン（右クリック）でエクスポートすることもできます。

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

さらに、メニューバー > ファイル > すべてのルールをエクスポート からすべてのルールをエクスポートできます。

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> [Einar Egilsson氏のRedirector](https://einaregilsson.com/redirector/)と互換性のある形式でルールをエクスポートしたい場合は、コンテキストメニューまたはメニューバーでOptionキー（`⌥`）を押すと、次のように表示されます。
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect WebはRedirectorと完全な互換性がなく、その逆も同様です。キャプチャグループ処理の「置換回数」など、一部の設定はRedirectorの形式ではエクスポートされません。
> :::

## ルールのインポート {#import-rules}

### macOSでのインポート {#import-on-macos}

- エクスポートしたルールファイルをRedirect Webのウィンドウにドラッグ＆ドロップするだけです。
- または、メニューバー > ファイル > ルールをインポート からインポートすることもできます。

### iOSでのインポート {#import-on-ios}

ルールファイルがファイルアプリにあるとします。ファイルアプリでそれを開き、共有ボタンをタップし、Redirect Webを選択します。

| ステップ1：ファイルアプリで共有ボタンをタップ                                     | ステップ2：Redirect Webを選択                                         |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Tap the share button in the toolbar](/img/files-share-ios.png) | ![Tap the app icon in the sheet](/img/share-sheet-ios.png) |

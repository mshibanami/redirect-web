# ルールのエクスポート/インポート

## ルールのエクスポート {#export-rules}

### macOSでのエクスポート {#export-on-macos}

このスクリーンショットに示すように、省略記号ボタンまたは共有ボタンをクリックします。

![macOSでの共有](/img/share-on-macos.png)

ルールリストのサイドバーで、ルールを複数選択し、副クリック（右クリック）してエクスポートすることもできます。

![macOSでの複数ルール選択](/img/multiselect-on-macos.png)

さらに、メニューバー > ファイル > すべてのルールをエクスポートからすべてのルールをエクスポートできます。

![メニューバー > ファイル](/img/menubar-file.png)

> [!NOTE]
> [Einar EgilssonのRedirector](https://einaregilsson.com/redirector/)と互換性のある形式でルールをエクスポートしたい場合は、コンテキストメニューまたはメニューバーでOptionキー（`⌥`）を押すと、以下が表示されます。
> ![メニューバー > ファイル (Redirector)](/img/menubar-file-redirector.png)
>
> ![コンテキストメニュー (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect WebはRedirectorと完全に互換性があるわけではなく、その逆も同様です。キャプチャグループ処理の「出現箇所の置換」などの一部の設定は、Redirectorの形式ではエクスポートされません。
> :::

## ルールのインポート {#import-rules}

### macOSでのインポート {#import-on-macos}

- エクスポートされたルールファイルをRedirect Webのウィンドウにドラッグ＆ドロップするだけです。
- または、メニューバー > ファイル > ルールのインポートからインポートすることもできます。

### iOSでのインポート {#import-on-ios}

ルールファイルがファイルアプリにあるとします。ファイルアプリでそれを開き、共有ボタンをタップし、Redirect Webを選択します。

| ステップ1：ファイルアプリの共有ボタンをタップ | ステップ2：Redirect Webを選択 |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![ツールバーの共有ボタンをタップ](/img/files-share-ios.png) | ![シート内のアプリアイコンをタップ](/img/share-sheet-ios.png) |

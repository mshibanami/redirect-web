# Export/Import Rules

## Export Rules {#export-rules}

### Export on macOS {#export-on-macos}

このスクリーンショットに示すように、省略記号ボタンまたは共有ボタンをクリックします。

<Image src={ShareOnMacOS} alt="Share button on macOS" width="460" hasShadow hasRoundedCorners/>

次に、コンテキストメニューから「Export Rule」または「Share Rule...」を選択します。

ルールを複数選択し、ルールリストのサイドバーで副クリック（右クリック）してエクスポートすることもできます。

<Image src={MultiSelectOnMacOS} alt="Share multiple rules on macOS" height="400" hasShadow hasRoundedCorners/>


さらに、メニューバー > ファイル > すべてのルールをエクスポート からすべてのルールをエクスポートできます。

<Image src={MenubarFile} alt="Menubar > File" width="400" hasShadow hasRoundedCorners/>

> [!NOTE]
> [Einar Egilsson's Redirector](https://einaregilsson.com/redirector/) と互換性のある形式でルールをエクスポートしたい場合は、コンテキストメニューまたはメニューバーで Option キー (`⌥`) を押すと、これらが表示されます。
> <Image src={MenubarFileRedirector} alt="Menubar > File (Redirector)" width="400" hasShadow hasRoundedCorners/>
> <br/>
> <Image src={ContextMenuRedirector} alt="Context Menu (Redirector)" width="400" hasShadow hasRoundedCorners/>
>
> :::warning
> Redirect Web は Redirector と完全に互換性があるわけではありません。また、その逆も同様です。キャプチャグループ処理の「Replace Occurrences」などの一部の設定は、Redirector の形式ではエクスポートされません。
> :::

## Import Rules {#import-rules}

### Import on macOS {#import-on-macos}

- エクスポートされたルールファイルを Redirect Web のウィンドウにドラッグアンドドロップするだけです。
- または、メニューバー > ファイル > ルールのインポート からインポートすることもできます。

### Import on iOS {#import-on-ios}

ルールファイルがファイルアプリにあるとします。ファイルアプリでファイルを開き、共有ボタンをタップして、Redirect Web を選択します。

| ステップ 1: ファイルアプリの共有ボタンをタップします                                                                          | ステップ 2: Redirect Web を選択します                                                                              |

Redirect Webのサポートウェブサイトへようこそ。Redirect Webは、ブラウザでのすべてのページ読み込みをリダイレクトし、常に目的の場所にアクセスできるようにするブラウザ拡張機能です。

(旧称：Redirect Web for Safari)

## ⬇️ ダウンロードリンク {#download-links}

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chromeウェブストア)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Firefoxアドオン)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Microsoft Edgeアドオン)
- [Safari](https://apps.apple.com/app/id1571283503) (拡張機能はネイティブアプリに同梱されています。)

Redirect Webは以下の環境で利用可能です：

- **Mac** (Safari, Chrome, Firefox, Edgeなど)
- **iPhone**、**iPad**、**Apple Vision Pro** (Safari)

:::note

- どのブラウザを使用する場合でも、拡張機能はアプリのルールや設定などに依存しているため、[Mac App StoreからRedirect Webアプリ](https://apps.apple.com/app/id1571283503)をインストールする必要があります。
- マルチブラウザ対応はMacのみで利用可能です。

:::

## 主な使用例 {#popular-use-cases}

例えば、次のようなルールを作成できます：

- **Reddit**をクラシックなデザイン(old.reddit.com)で表示する。
- **Google マップ**のリンクが表示されたときに常に**Apple マップ**を開く。
- すべての検索に**?udm=14**を追加して、**Google検索結果**からGoogleのAI概要を非表示にする。
- Fandomのページから公式の**Terraria**または**Minecraft**のWikiにジャンプする。
- **Kagi**、**Startpage**、**ChatGPT**、およびその他の検索エンジンを**デフォルトの検索エンジン**として使用する。
- **Google Meet**のリンクをGoogle Chromeで開く。(Mac)
- **Figma**のリンクをFigmaアプリで直接開く。(Mac)
- **Notion**のリンクをNotionアプリで直接開く。(Mac)

これらはすべて、アプリのライブラリセクションから定義済みルールとして無料で入手できます！

## 機能 {#features}

Redirect Webには以下の機能が含まれています：

- **正規表現**または**ワイルドカード**パターンタイプを使用してルールを作成する
- **後方参照**を使用して動的なターゲットURLを指定する
- リダイレクトからURLを**除外**する
- URLエンコード/デコード、base64エンコード/デコード、または置換を使用して**キャプチャしたテキストを処理**する

さらに、サブスクリプションまたは買い切りで購入することで、以下の機能をアンロックできます：

- **無制限**のルールを有効にする
- **iCloud**を介して、他のMac、iPhone、iPadと**ルールを同期**する

(アプリを購入していただくことは、Redirect Webの維持にもつながります。🙏)

お客様の購入またはサブスクリプションは、ユニバーサル購入を介してMac、iPhone、iPad、Apple Vision Proで共有されます。皆様のサポートが継続的な開発に役立ちます。

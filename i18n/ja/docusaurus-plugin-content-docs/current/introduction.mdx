Redirect Webのサポートウェブサイトへようこそ。Redirect Webは、ブラウザでのすべてのページ読み込みをリダイレクトし、常に目的の場所にアクセスできるようにするブラウザ拡張機能です。

(旧称：Redirect Web for Safari)

## ⬇️ ダウンロードリンク

- [Chrome、Opera、Brave、Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chrome ウェブストア)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Firefox Add-ons)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Microsoft Edge Add-ons)
- [Safari](https://apps.apple.com/app/id1571283503) (拡張機能はすでにネイティブアプリにバンドルされています。)

Redirect Webは以下で利用可能です：

- **Mac** (Safari、Chrome、Firefox、Edgeなど)
- **iPhone**、**iPad**、**Apple Vision Pro** (Safari)

:::note

- どのブラウザを使用する場合でも、拡張機能を使用するには[Mac App StoreからRedirect Webアプリをインストールする](https://apps.apple.com/app/id1571283503)必要があります。これは、拡張機能がルールや設定などにアクセスするためにアプリに依存しているためです。
- マルチブラウザサポートはMacでのみ利用可能です。

:::

## 人気のある使用例

例えば、以下のようなルールを作成できます：

- **Reddit**をクラシックデザインで表示する (old.reddit.com)。
- **Google マップ**のリンクが表示されたら常に**Apple マップ**を開く。
- すべての検索に**?udm=14**を追加して、**Google 検索結果**からGoogleのAI概要を非表示にする。
- Fandomページから公式の**Terraria**または**Minecraft** Wikiへジャンプする。
- **Kagi**、**Startpage**、**ChatGPT**、およびその他の検索エンジンを**デフォルトの検索エンジン**として使用する。
- **Google Meet**リンクをGoogle Chromeで開く。(Mac)
- **Figma**リンクをFigmaアプリで直接開く。(Mac)
- **Notion**リンクをNotionアプリで直接開く。(Mac)

これらのすべては、アプリのライブラリセクションから無料で事前定義されたルールとして入手できます！

## 機能

Redirect Webには以下の機能が搭載されています：

- **正規表現**または**ワイルドカード**パターンタイプを使用してルールを作成
- **後方参照**を使用して動的なターゲットURLを指定
- リダイレクトからURLを**除外**
- URLエンコード/デコード、Base64エンコード/デコード、または出現箇所の置換を使用して**キャプチャしたテキストを処理**

さらに、サブスクリプションまたは一括購入により、以下の機能を利用できます：

- 無制限の数のルールを**有効化**
- **iCloud**経由で他のMac、iPhone、iPadと**ルールを同期**

(アプリの購入は、Redirect Webの維持にも役立ちます。 🙏)

購入またはサブスクリプションは、ユニバーサル購入によりMac、iPhone、iPad、Apple Vision Proで共有されます。皆様のサポートは継続的な開発に役立ちます。

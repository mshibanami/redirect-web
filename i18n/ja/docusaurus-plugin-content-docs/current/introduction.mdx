Redirect Web のサポートサイトへようこそ。Redirect Web はブラウザ内のページ読み込みをすべてリダイレクトし、常に望む場所へ到達できるようにする拡張機能です。

（旧名: Redirect Web for Safari）

## ⬇️ ダウンロードリンク {#download-links}

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chrome Web Store)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Firefox Add-ons)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Microsoft Edge Add-ons)
- [Safari](https://apps.apple.com/app/id1571283503) (拡張機能はネイティブアプリにバンドルされています)

Redirect Web は次のプラットフォームで利用できます:

- **Mac**（Safari、Chrome、Firefox、Edge など）
- **iPhone**、**iPad**、および **Apple Vision Pro**（Safari）

:::note

- 拡張機能はルールや設定へアクセスするためにアプリに依存しているため、どのブラウザを使う場合でも利用するには必ず[Mac App Store から Redirect Web アプリをインストール](https://apps.apple.com/app/id1571283503)してください。
- マルチブラウザ対応は Mac のみで利用可能です。

:::

## よく使われるユースケース {#popular-use-cases}

例えば、次のようなルールを作成できます:

- **Reddit** をクラシックデザイン（`old.reddit.com`）で表示する
- **Google Maps** のリンクがあれば常に **Apple Maps** で開く
- **Google 検索結果** から Google の AI オーバービューを隠すために、すべての検索に **?udm=14** を追加する
- Fandom のページから公式の **Terraria** や **Minecraft** ウィキへジャンプする
- **Kagi**、**Startpage**、**ChatGPT**、その他の検索エンジンをデフォルト検索エンジンとして使用する
- **Google Meet** のリンクを Google Chrome で開く（Mac）
- **Figma** のリンクを直接 Figma アプリで開く（Mac）
- **Notion** のリンクを直接 Notion アプリで開く（Mac）

これらはすべて、アプリ内のライブラリセクションから無料の事前定義ルールとして入手できます！

## 機能 {#features}

Redirect Web は以下の機能を備えています:

- **正規表現** または **ワイルドカード** パターンタイプでルールを作成
- 動的なターゲット URL を指定するための **バックリファレンス（後方参照）** の使用
- リダイレクトから URL を **除外**
- キャプチャしたテキストを URL エンコード/デコード、base64 エンコード/デコード、または置換によって **処理**

さらに、サブスクリプションまたは一度限りの購入で次の機能をアンロックできます:

- **無制限** のルールを有効化
- **iCloud** 経由で他の Mac、iPhone、iPad とルールを **同期**

（アプリの購入は Redirect Web の維持・開発支援にもなります。🙏）

購入やサブスクリプションはユニバーサル購入を通じて Mac、iPhone、iPad、Apple Vision Pro で共有されます。皆さまのサポートが開発継続の助けになります。

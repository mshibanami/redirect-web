Redirect Webのサポートウェブサイトへようこそ。Redirect Webは、ブラウザのすべてのページ読み込みをリダイレクトし、常に目的の場所にたどり着けるようにするブラウザ拡張機能です。

（旧称：Safari用Redirect Web）

## ⬇️ ダウンロードリンク {#download-links}

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chrome Web Store)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Firefox Add-ons)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Microsoft Edge Add-ons)
- [Safari](https://apps.apple.com/app/id1571283503) (拡張機能はすでにネイティブアプリにバンドルされています。)

Redirect Webは以下で利用可能です。

- **Mac** (Safari, Chrome, Firefox, Edgeなど)
- **iPhone**, **iPad**, および **Apple Vision Pro** (Safari)

:::note

- どのブラウザを使用しているかに関わらず、拡張機能を使用するには[Mac App StoreからRedirect Webアプリをインストール](https://apps.apple.com/app/id1571283503)する必要があります。これは、拡張機能がルールや設定などにアクセスするためにアプリに依存しているためです。
- マルチブラウザサポートはMacでのみ利用可能です。

:::

## 人気のある使用例 {#popular-use-cases}

たとえば、次のようなルールを作成できます。

- **Reddit**をクラシックデザイン（old.reddit.com）で表示する。
- **Googleマップ**のリンクが表示されたら、常に**Appleマップ**を開く。
- すべての検索に**?udm=14**を追加して、**Google検索結果**からGoogleのAI概要を非表示にする。
- Fandomページから公式の**Terraria**または**Minecraft**のWikiにジャンプする。
- **Kagi**、**Startpage**、**ChatGPT**、およびその他の検索エンジンを**デフォルトの検索エンジン**として使用する。
- **Google Meet**のリンクをGoogle Chromeで開く。（Mac）
- **Figma**のリンクをFigmaアプリで直接開く。（Mac）
- **Notion**のリンクをNotionアプリで直接開く。（Mac）

これらはすべて、アプリのライブラリセクションから無料で事前定義されたルールとして入手できます！

## 機能 {#features}

Redirect Webには以下の機能が付属しています。

- **正規表現**または**ワイルドカード**パターンタイプを使用してルールを作成する
- 動的なターゲットURLを指定するために**後方参照**を使用する
- リダイレクトからURLを**除外**する
- URLエンコード/デコード、base64エンコード/デコード、または出現箇所の置換を使用して**キャプチャされたテキストを処理**する

さらに、サブスクリプションまたは一括購入を通じて、以下の機能のロックを解除できます。

- **無制限**の数のルールを有効にする
- **iCloudを介して**他のMac、iPhone、iPadと**ルールを同期**する

（アプリを購入することは、Redirect Webの維持にも役立ちます。🙏）

購入またはサブスクリプションは、ユニバーサル購入を介してMac、iPhone、iPad、Apple Vision Pro間で共有されます。皆様のサポートは継続的な開発に役立ちます。

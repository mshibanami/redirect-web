# ライブラリ

私たちのライブラリへようこそ。ここでは、無料で使える便利なルールを見つけることができます！Redirect Webアプリからアクセスできます。

![Library Screenshot](/img/library-screenshot.webp)

ルールを「マイ ルール」にコピーするには、「取得」ボタンを選択するだけです。

![Get Button - iOS](/img/library-get-button-ios.gif)

## ライブラリにルールを貢献する

現在のライブラリにはルールが少ないため、皆様の貢献を歓迎します！

ライブラリにルールを貢献するには、2つのオプションがあります。

1. **フォーラムにルールを投稿する**: [フォーラム](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules)で他のユーザーとリダイレクトルールを共有します。その後、あなたのルールをライブラリにも追加してもよいかリクエストする場合があります。これは、追加作業を私たちに任せられるため、オプション2よりも簡単かもしれません。
2. **プルリクエストを送信する**: プルリクエストを送信できます。詳細については、[手順](https://github.com/mshibanami/redirect-web/tree/main/library)を確認してください。

## その他の情報

- すべてのコンテンツは[GitHubのリポジトリ](https://github.com/mshibanami/redirect-web/tree/main/library)で確認できます。
- ライブラリコンテンツのためにCloudflare上にプライベートAPIサーバーをホストしており、フィルタリングやその他のサービスを提供しています。アプリはこのサーバーからライブラリコンテンツを取得します。[プライバシーポリシー](privacy-policy.mdx)に記載されているように、このサーバーは個人情報を収集しません。

# ライブラリ

Redirect Webアプリを通じてアクセスできる、無料で利用可能な便利なルールを見つけられる当社のライブラリへようこそ！

![Library Screenshot](/img/library-screenshot.webp)

ルールを「マイ・ルール」にコピーするには、「取得」ボタンを選択するだけです。

![Get Button - iOS](/img/library-get-button-ios.gif)

## ライブラリにルールを貢献する

現在のライブラリにはまだルールが少ないため、皆様からのご貢献を歓迎いたします！

ライブラリにルールを貢献する方法は2つあります。

1.  **フォーラムにルールを投稿する**: [フォーラム](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules)で他のユーザーとリダイレクトルールを共有してください。その後、そのルールをライブラリにも追加できるか、こちらからお願いする場合があります。追加作業を当社に任せられるため、オプション2よりも簡単かもしれません。
2.  **プルリクエストを送る**: プルリクエストを送ることができます。詳細については、[手順](https://github.com/mshibanami/redirect-web/tree/main/library)をご確認ください。

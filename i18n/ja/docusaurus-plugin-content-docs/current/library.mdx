# ライブラリ

Redirect Webアプリから無料で利用できる、便利な既成のルールを見つけることができるライブラリへようこそ！

![Library Screenshot](/img/library-screenshot.webp)

ルールを「マイ ルール」にコピーするには、「取得」ボタンを選択するだけです。

![Get Button - iOS](/img/library-get-button-ios.gif)

## ライブラリにルールを貢献する

現在のライブラリにはまだルールが少ないため、皆様からのご貢献をお待ちしております！

ルールをライブラリに貢献するには、2つのオプションがあります。

1.  **フォーラムにルールを投稿する**: [フォーラム](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules)で他のユーザーとリダイレクトルールを共有してください。その後、あなたのルールをライブラリにも追加できるか、こちらからお願いする場合があります。追加作業を私たちに任せられるため、オプション2よりも簡単かもしれません。
2.  **プルリクエストを送信する**: プルリクエストを送信することができます。[手順](https://github.com/mshibanami/redirect-web/tree/main/library)をご確認ください。

## その他の情報

-   すべての内容は[GitHubのリポジトリ](https://github.com/mshibanami/redirect-web/tree/main/library)で確認できます。
-   ライブラリコンテンツのために、Cloudflare上にプライベートAPIサーバーをホストしており、フィルタリングなどのサービスを提供しています。アプリはこのサーバーからライブラリコンテンツを取得します。[プライバシーポリシー](/privacy-policy)に記載されている通り、このサーバーは個人情報を収集しません。

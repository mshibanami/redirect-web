# ライブラリ

Redirect Webアプリから無料で利用できる便利なルールを見つけられるライブラリへようこそ！

![ライブラリのスクリーンショット](/img/library-screenshot.webp)

ルールを「マイ・ルール」にコピーするには、「取得」ボタンを選択するだけです。

![取得ボタン - iOS](/img/library-get-button-ios.gif)

## ライブラリにルールを貢献する

現在のライブラリにはまだ少数のルールしかありませんので、皆様からの貢献をお待ちしております！

ライブラリにルールを貢献する方法は2つあります。

1. **フォーラムにルールを投稿する**: [フォーラム](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules)で他のユーザーとリダイレクトルールを共有してください。その後、あなたのルールをライブラリに追加してもよいかお願いする場合があります。これは、追加作業を私たちに任せられるため、オプション2よりも簡単かもしれません。
2. **プルリクエストを送る**: プルリクエストを送ることができます。[手順](https://github.com/mshibanami/redirect-web/tree/main/library)を確認して詳細を学んでください。

## その他の情報

- すべてのコンテンツは[GitHubのリポジトリ](https://github.com/mshibanami/redirect-web/tree/main/library)で確認できます。
- ライブラリコンテンツのためにCloudflare上にプライベートAPIサーバーをホストしており、フィルタリングやその他のサービスを提供しています。アプリはこのサーバーからライブラリコンテンツを取得します。[プライバシーポリシー](privacy-policy.mdx)に記載されている通り、このサーバーはあなたの個人情報を収集しません。

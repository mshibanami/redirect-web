# 문의하기

Redirect Web에 대한 질문, 피드백 또는 문제가 있는 경우 도와드리겠습니다. 문의하실 수 있는 가장 좋은 방법은 다음과 같습니다.

- **[GitHub 이슈](https://github.com/mshibanami/RedirectWeb/issues)**: 버그를 발견했거나 새로운 기능을 요청하고 싶으시면 GitHub 이슈 페이지를 방문해 주세요. 기존 이슈를 찾아보고, 새 이슈를 제출하고, 이슈 진행 상황을 추적할 수 있습니다.
- **[GitHub 토론](https://github.com/mshibanami/RedirectWeb/discussions)**: 질문이 있거나 개발자 또는 다른 사용자에게 도움을 받고 싶으시면 GitHub 토론 페이지를 방문해 주세요. 새로운 토론을 시작하고, 기존 토론에 참여하며, 커뮤니티의 도움을 받을 수 있습니다.
- **피드백 양식**: Redirect Web 앱 내의 피드백 양식을 통해 피드백을 보내실 수 있습니다. 이 피드백은 개발자에게 직접 이메일로 전송되므로 다른 사용자에게 공개되지 않고 비공개로 유지됩니다.

Redirect Web을 이용해 주셔서 감사합니다! 여러분의 지원과 피드백에 감사드립니다.

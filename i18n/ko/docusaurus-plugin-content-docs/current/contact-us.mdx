# 문의하기

Redirect Web에 대한 질문, 피드백 또는 문제가 있는 경우 도움을 드릴 준비가 되어 있습니다. 다음은 연락할 수 있는 가장 좋은 방법입니다:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: 버그를 발견하거나 새로운 기능을 요청하려면 GitHub Issues 페이지를 방문하십시오. 기존 이슈를 찾아보고, 새 이슈를 제출하고, 이슈 진행 상황을 추적할 수 있습니다.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: 질문이 있거나 개발자 또는 다른 사용자로부터 도움을 받고 싶다면 GitHub Discussions 페이지를 방문하십시오. 새 토론을 시작하고, 기존 토론에 참여하고, 커뮤니티로부터 도움을 받을 수 있습니다.
- **피드백 양식**: Redirect Web 앱의 피드백 양식을 통해 피드백을 보낼 수 있습니다. 이 양식은 개발자에게 직접 이메일을 보내므로 귀하의 피드백은 다른 사용자에게 비공개로 유지됩니다.

Redirect Web을 이용해 주셔서 감사합니다! 여러분의 지원과 피드백에 감사드립니다.

# 규칙 내보내기/가져오기

## 규칙 내보내기 {#export-rules}

### macOS에서 내보내기 {#export-on-macos}

이 스크린샷에 표시된 줄임표 버튼 또는 공유 버튼을 클릭하십시오:

![macOS에서 공유](/img/share-on-macos.png)

규칙 목록 사이드바에서 규칙을 보조 클릭(오른쪽 클릭)하여 여러 규칙을 선택하고 내보낼 수도 있습니다:

![macOS에서 여러 규칙 공유](/img/multiselect-on-macos.png)

또한, 메뉴바 > 파일 > 모든 규칙 내보내기에서 모든 규칙을 내보낼 수 있습니다:

![메뉴바 > 파일](/img/menubar-file.png)

> [!NOTE]
> [Einar Egilsson의 Redirector](https://einaregilsson.com/redirector/)와 호환되는 형식으로 규칙을 내보내려면, 컨텍스트 메뉴 또는 메뉴바에서 Option 키(`⌥`)를 누르면 다음을 볼 수 있습니다:
> ![메뉴바 > 파일 (Redirector)](/img/menubar-file-redirector.png)
>
> ![컨텍스트 메뉴 (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web은 Redirector와 완전히 호환되지 않으며, 그 반대도 마찬가지입니다. 캡처 그룹 처리의 "발생 횟수 바꾸기"와 같은 일부 설정은 Redirector 형식으로 내보내지지 않습니다.
> :::

## 규칙 가져오기 {#import-rules}

### macOS에서 가져오기 {#import-on-macos}

- 내보낸 규칙 파일을 Redirect Web 창으로 간단히 드래그 앤 드롭합니다.
- 또는 메뉴바 > 파일 > 규칙 가져오기에서 가져올 수도 있습니다.

### iOS에서 가져오기 {#import-on-ios}

규칙 파일이 파일 앱에 있다고 가정해 봅시다. 파일 앱에서 파일을 열고 공유 버튼을 탭한 다음 Redirect Web을 선택합니다:

| 1단계: 파일 앱에서 공유 버튼 탭 | 2단계: Redirect Web 선택 |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![도구 모음에서 공유 버튼 탭](/img/files-share-ios.png) | ![시트에서 앱 아이콘 탭](/img/share-sheet-ios.png) |

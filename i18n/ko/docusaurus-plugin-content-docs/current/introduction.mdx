브라우저의 모든 페이지 로드를 리디렉션하여 항상 원하는 곳으로 연결해 주는 브라우저 확장 프로그램인 Redirect Web의 지원 웹사이트에 오신 것을 환영합니다.

(이전 이름: Redirect Web for Safari)

## ⬇️ 다운로드 링크 {#download-links}

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chrome 웹 스토어)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Firefox 애드온)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Microsoft Edge 애드온)
- [Safari](https://apps.apple.com/app/id1571283503) (확장 프로그램은 이미 네이티브 앱에 포함되어 있습니다.)

Redirect Web은 다음 플랫폼에서 사용할 수 있습니다:

- **Mac** (Safari, Chrome, Firefox, Edge 등)
- **iPhone**, **iPad**, **Apple Vision Pro** (Safari)

:::note

- 어떤 브라우저를 사용하든 확장 프로그램을 사용하려면 [Mac App Store에서 Redirect Web 앱을 설치](https://apps.apple.com/app/id1571283503)해야 합니다. 확장 프로그램은 규칙, 설정 등에 접근하기 위해 앱에 의존하기 때문입니다.
- 다중 브라우저 지원은 Mac에서만 가능합니다.

:::

## 인기 사용 사례 {#popular-use-cases}

예를 들어, 다음 규칙을 생성할 수 있습니다:

- **Reddit**을 클래식 디자인(old.reddit.com)으로 봅니다.
- **Google Maps** 링크가 나타날 때마다 **Apple Maps**를 엽니다.
- 모든 검색에 **?udm=14**를 추가하여 **Google 검색 결과**에서 Google의 AI 개요를 숨깁니다.
- Fandom 페이지에서 공식 **Terraria** 또는 **Minecraft** 위키로 이동합니다.
- **Kagi**, **Startpage**, **ChatGPT** 및 기타 모든 검색 엔진을 **기본 검색 엔진**으로 사용합니다.
- Google Chrome에서 **Google Meet** 링크를 엽니다. (Mac)
- Figma 앱에서 **Figma** 링크를 직접 엽니다. (Mac)
- Notion 앱에서 **Notion** 링크를 직접 엽니다. (Mac)

앱의 라이브러리 섹션에서 이 모든 것을 미리 정의된 규칙으로 무료로 받을 수 있습니다!

## 기능 {#features}

Redirect Web은 다음과 같은 기능을 제공합니다:

- **정규 표현식** 또는 **와일드카드** 패턴 유형을 사용하여 규칙 생성
- 동적 대상 URL을 지정하기 위해 **역참조** 사용
- 리디렉션에서 URL **제외**
- URL 인코딩/디코딩, base64 인코딩/디코딩 또는 발생 횟수 바꾸기를 사용하여 **캡처된 텍스트 처리**

또한, 구독 또는 일회성 구매를 통해 다음 기능을 잠금 해제할 수 있습니다:

- **무제한** 규칙 활성화
- 다른 Mac, iPhone, iPad와 **iCloud를 통해 규칙 동기화**

(앱 구매는 Redirect Web 유지 관리에 도움이 됩니다. 🙏)

구매 또는 구독은 유니버설 구매를 통해 Mac, iPhone, iPad, Apple Vision Pro에서 공유됩니다. 여러분의 지원은 지속적인 개발에 도움이 됩니다.

Redirect Web의 지원 사이트에 오신 것을 환영합니다. 이 브라우저 확장 프로그램은 브라우저에서 로드되는 모든 페이지를 리디렉션하여 항상 원하는 위치로 이동하도록 도와줍니다.

(이전 명칭: Redirect Web for Safari.)

## ⬇️ 다운로드 링크 {#download-links}

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chrome Web Store)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Firefox Add-ons)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Microsoft Edge Add-ons)
- [Safari](https://apps.apple.com/app/id1571283503) (확장 기능은 네이티브 앱에 번들되어 있습니다.)

Redirect Web은 다음 플랫폼에서 사용할 수 있습니다:

- **Mac** (Safari, Chrome, Firefox, Edge 등)
- **iPhone**, **iPad**, 및 **Apple Vision Pro** (Safari)

:::note

- 확장 기능을 사용하려면 브라우저 종류에 관계없이 [Mac App Store에서 Redirect Web 앱](https://apps.apple.com/app/id1571283503)을 설치해야 합니다. 확장 기능은 앱을 통해 규칙과 설정에 접근합니다.
- 다중 브라우저 지원은 Mac에서만 제공됩니다.

:::

## 인기 있는 사용 사례 {#popular-use-cases}

예를 들어, 다음과 같은 규칙을 만들 수 있습니다:

- **Reddit**을 클래식 디자인(old.reddit.com)으로 보기
- **Google Maps** 링크가 나타날 때마다 **Apple Maps**로 열기
- 모든 검색에 **?udm=14**를 추가하여 **Google 검색 결과**에서 Google의 AI 요약을 숨기기
- Fandom 페이지에서 공식 **Terraria** 또는 **Minecraft** 위키로 이동
- **Kagi**, **Startpage**, **ChatGPT** 등 원하는 검색 엔진을 **기본 검색 엔진**으로 사용
- **Google Meet** 링크를 Google Chrome에서 열기 (Mac)
- **Figma** 링크를 Figma 앱에서 직접 열기 (Mac)
- **Notion** 링크를 Notion 앱에서 직접 열기 (Mac)

이 모든 기능은 앱의 라이브러리 섹션에서 무료로 제공되는 미리 정의된 규칙으로 이용할 수 있습니다!

## 기능 {#features}

Redirect Web은 다음 기능을 제공합니다:

- **정규 표현식(Regular Expression)** 또는 **와일드카드(Wildcard)** 패턴 유형으로 규칙 생성
- 동적 대상 URL을 지정하기 위한 **백레퍼런싱(back-referencing)** 사용
- 리디렉션에서 URL **제외(Exclude)**
- 캡처한 텍스트를 URL 인코딩/디코딩, base64 인코딩/디코딩 또는 치환하여 **처리**

또한 구독 또는 일회성 구매를 통해 다음 기능을 잠금 해제할 수 있습니다:

- 무제한 규칙 허용
- **iCloud를 통한 규칙 동기화**로 Mac, iPhone 및 iPad 간 동기화

(앱을 구매하시면 Redirect Web 유지 관리를 지원합니다. 🙏)

구매 또는 구독은 Universal Purchase를 통해 Mac, iPhone, iPad 및 Apple Vision Pro에 공유됩니다. 여러분의 지원은 지속적인 개발에 도움이 됩니다.

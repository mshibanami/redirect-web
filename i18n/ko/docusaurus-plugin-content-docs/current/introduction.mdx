RedirectWeb의 지원 웹사이트에 오신 것을 환영합니다. RedirectWeb은 브라우저의 모든 페이지 로드를 리디렉션하여 항상 원하는 곳으로 이동할 수 있게 해주는 브라우저 확장 프로그램입니다.

(이전에는 _Redirect Web for Safari_ 또는 간단히 *Redirect Web*으로 알려졌습니다.)

## ⬇️ 다운로드 링크 {/* #download-links */}

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chrome 웹 스토어)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Firefox 부가 기능)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Microsoft Edge 부가 기능)
- [Safari](https://apps.apple.com/app/id1571283503) (확장 프로그램이 이미 네이티브 앱에 포함되어 있습니다.)

RedirectWeb은 다음에서 사용할 수 있습니다:

- **Mac** (Safari, Chrome, Firefox, Edge 등)
- **iPhone**, **iPad**, 및 **Apple Vision Pro** (Safari)

:::note

- 어떤 브라우저를 사용하든 확장 프로그램을 사용하려면 [Mac App Store에서 RedirectWeb 앱](https://apps.apple.com/app/id1571283503)을 설치해야 합니다. 확장 프로그램이 규칙, 설정 등에 접근하기 위해 이 앱에 의존하기 때문입니다.
- 다중 브라우저 지원은 Mac에서만 가능합니다.

:::

## 인기 있는 사용 사례 {/* #popular-use-cases */}

예를 들어 다음과 같은 규칙을 만들 수 있습니다:

- **Reddit**을 클래식 디자인(old.reddit.com)으로 보기.
- **Google Maps** 링크가 나타날 때마다 **Apple Maps**로 열기.
- 모든 검색에 **?udm=14**를 추가하여 **Google 검색 결과**에서 Google의 AI 개요(AI Overviews) 숨기기.
- Fandom 페이지에서 공식 **Terraria** 또는 **Minecraft** 위키로 바로 이동하기.
- **Kagi**, **Startpage**, **ChatGPT** 및 기타 모든 검색 엔진을 **기본 검색 엔진**으로 사용하기.
- **Google Meet** 링크를 Google Chrome에서 열기. (Mac)
- **Figma** 링크를 Figma 앱에서 직접 열기. (Mac)
- **Notion** 링크를 Notion 앱에서 직접 열기. (Mac)

앱의 라이브러리 섹션에서 이 모든 규칙을 사전 정의된 규칙으로 무료로 받을 수 있습니다!

## 기능 {/* #features */}

RedirectWeb은 다음과 같은 기능을 제공합니다:

- **정규 표현식(Regular Expression)** 또는 **와일드카드(Wildcard)** 패턴 유형을 사용하여 규칙 생성
- 동적 대상 URL을 지정하기 위해 **역참조(back-referencing)** 사용
- 리디렉션에서 URL **제외(Exclude)**
- URL 인코딩/디코딩, base64 인코딩/디코딩 또는 텍스트 치환을 사용하여 **캡처된 텍스트 처리**

또한 구독 또는 일회성 구매를 통해 다음과 같은 기능을 잠금 해제할 수 있습니다:

- 규칙 개수 **무제한** 활성화
- **iCloud를 통해** 다른 Mac, iPhone 및 iPad와 **규칙 동기화**

(앱 구매는 RedirectWeb의 유지 관리에도 큰 도움이 됩니다. 🙏)

구매 또는 구독은 범용 구매(Universal Purchase)를 통해 Mac, iPhone, iPad 및 Apple Vision Pro 간에 공유됩니다. 여러분의 지원은 지속적인 개발에 힘이 됩니다.

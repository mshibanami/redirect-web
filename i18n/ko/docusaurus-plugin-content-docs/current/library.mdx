# 라이브러리

유용하고 바로 사용 가능한 무료 규칙을 찾을 수 있는 저희 라이브러리에 오신 것을 환영합니다! Redirect Web 앱을 통해 이 규칙들에 액세스할 수 있습니다.

![Library Screenshot](/img/library-screenshot.webp)

규칙을 내 규칙(My Rules)으로 복사하려면 Get 버튼을 선택하기만 하면 됩니다:

![Get Button - iOS](/img/library-get-button-ios.gif)

## 나만의 규칙을 라이브러리에 기여하기

현재 라이브러리에는 몇 가지 규칙만 있으므로, 여러분의 기여를 환영합니다!

규칙을 라이브러리에 기여하는 2가지 옵션이 있습니다:

1.  **포럼에 규칙 게시**: [포럼](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules)에서 다른 사용자와 리다이렉트 규칙을 공유하세요. 그러면 저희가 해당 규칙을 라이브러리에 추가할 수 있는지 요청할 수 있습니다. 이 방법은 추가 작업을 저희에게 맡길 수 있어 옵션 2보다 쉬울 수 있습니다.
2.  **풀 리퀘스트 보내기**: 풀 리퀘스트를 보낼 수 있습니다. 자세한 내용은 [지침](https://github.com/mshibanami/redirect-web/tree/main/library)을 확인해주세요.

## 기타 정보

-   모든 내용은 [GitHub 저장소](https://github.com/mshibanami/redirect-web/tree/main/library)에서 확인할 수 있습니다.
-   저희는 라이브러리 콘텐츠를 위해 Cloudflare에 비공개 API 서버를 호스팅하여 필터링 및 기타 서비스를 제공할 수 있습니다. 앱은 이 서버에서 라이브러리 콘텐츠를 가져옵니다. [개인정보 처리방침](/privacy-policy)에 명시된 바와 같이, 이 서버는 귀하의 개인 정보를 수집하지 않습니다.

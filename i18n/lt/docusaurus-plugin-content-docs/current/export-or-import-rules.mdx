# Taisyklių eksportavimas / importavimas

## Taisyklių eksportavimas

### Eksportavimas sistemoje macOS

Spustelėkite elipsės mygtuką arba bendrinimo mygtuką, kaip parodyta šioje ekrano kopijoje:

![Share on macOS](/img/share-on-macos.png)

Taip pat galite pasirinkti kelias taisykles ir eksportuoti jas spustelėdami antriniu pelės mygtuku (dešiniuoju pelės mygtuku) taisyklų sąrašo šoninėje juostoje:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Be to, visas taisykles galite eksportuoti iš meniu juostos > Failas > Eksportuoti visas taisykles:

![Menubar > File](/img/menubar-file.png)

> [!PASTABA]
> Jei norite eksportuoti taisykles formatu, suderinamu su [Einar Egilsson's Redirector](https://einaregilsson.com/redirector/), paspauskite Option klavišą (`⌥`) kontekstiniame meniu arba meniu juostoje, tada pamatysite šiuos:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::įspėjimas
> Redirect Web nėra visiškai suderinamas su Redirector ir atvirkščiai. Kai kurie nustatymai, pvz., „Replace Occurrences“ (Pakeisti pasikartojimus) Capturing Group Processing, neeksportuojami Redirector formatu.
> :::

## Taisyklių importavimas

### Importavimas sistemoje macOS

- Tiesiog nuvilkite eksportuotus taisyklių failus į Redirect Web langą.
- Arba galite importuoti juos iš meniu juostos > Failas > Importuoti taisykles.

### Importavimas sistemoje iOS

Tarkime, kad jūsų taisyklės failas yra programoje „Files“. Atidarykite jį programoje „Files“, bakstelėkite bendrinimo mygtuką ir tada pasirinkite Redirect Web:

| 1 žingsnis: Bakstelėkite bendrinimo mygtuką programoje „Files“ | 2 žingsnis: Pasirinkite Redirect Web |

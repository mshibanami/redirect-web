Sveiki atvykę į Redirect Web palaikymo svetainę — naršyklės plėtinį, kuris peradresuoja visus jūsų naršyklės puslapių užkrovimus ir užtikrina, kad visada patektumėte ten, kur norite.

(Anksčiau žinomas kaip Redirect Web for Safari.)

## ⬇️ Atsisiuntimo nuorodos {#download-links}

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chrome Web Store)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Firefox Add-ons)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Microsoft Edge Add-ons)
- [Safari](https://apps.apple.com/app/id1571283503) (The extension is already bundled in the native app.)

Redirect Web pasiekiamas:

- **Mac** (Safari, Chrome, Firefox, Edge ir kt.)
- **iPhone**, **iPad** ir **Apple Vision Pro** (Safari)

:::note

- Norėdami naudoti plėtinį, turite įdiegti [Redirect Web programėlę iš Mac App Store](https://apps.apple.com/app/id1571283503), nepriklausomai nuo to, kurią naršyklę naudojate, nes plėtiniai naudoja programėlę, kad pasiektų jūsų taisykles, nustatymus ir kt.
- Kelių naršyklių palaikymas yra prieinamas tik Mac.

:::

## Populiarios naudojimo situacijos {#popular-use-cases}

Pavyzdžiui, galite sukurti šias taisykles:

- Žiūrėti **Reddit** klasikiniu dizainu (old.reddit.com).
- Atidaryti **Apple Maps**, kai pasirodo **Google Maps** nuoroda.
- Paslėpti Google AI apžvalgas iš **Google paieškos rezultatų**, pridėdami **?udm=14** prie kiekvienos paieškos.
- Peršokti iš Fandom puslapių į oficialias **Terraria** arba **Minecraft** žinynų (wiki).
- Naudoti **Kagi**, **Startpage**, **ChatGPT** ar bet kurią kitą paieškos sistemą kaip jūsų **numatytąją paieškos sistemą**.
- Atidaryti **Google Meet** nuorodas Google Chrome naršyklėje. (Mac)
- Atidaryti **Figma** nuorodas tiesiogiai Figma programėlėje. (Mac)
- Atidaryti **Notion** nuorodas tiesiogiai Notion programėlėje. (Mac)

Visas šias taisykles galite gauti nemokamai iš programėlės skilties `Library`!

## Funkcijos {#features}

Redirect Web turi šias funkcijas:

- Kurkite taisykles naudodami **Regular Expression** arba **Wildcard** šablonų tipus
- Naudokite **back-referencing** dinaminėms tikslinėms URL nurodyti
- **Exclude** URL iš peradresavimo
- **Process captured texts** naudojant URL kodavimą/dekodavimą, base64 kodavimą/dekodavimą arba pakeitimų vykdymą

Be to, galite atrakinti šias funkcijas prenumeratos arba vienkartinio pirkimo būdu:

- Leidimas kurti **neribotą** kiekį taisyklių
- **Syncing rules** su kitais jūsų Mac, iPhone ir iPad **per iCloud**

(Programėlės pirkimas taip pat padeda mums palaikyti Redirect Web. 🙏)

Jūsų pirkimas arba prenumerata dalijami tarp jūsų Mac, iPhone, iPad ir Apple Vision Pro per Universal Purchase. Jūsų parama padeda tęstiniam vystymui.

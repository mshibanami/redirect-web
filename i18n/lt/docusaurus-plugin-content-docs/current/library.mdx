# Biblioteka

Sveiki atvykę į mūsų biblioteką, kurioje galite rasti naudingų, paruoštų naudoti taisyklių nemokamai! Jas galite pasiekti per „Redirect Web“ programėlę.

![Library Screenshot](/img/library-screenshot.webp)

Norėdami nukopijuoti taisykles į Mano Taisykles, tiesiog pasirinkite mygtuką Gauti:

![Get Button - iOS](/img/library-get-button-ios.gif)

## Prisidėkite savo taisykle prie bibliotekos

Dabartinėje bibliotekoje yra tik kelios taisyklės, todėl vertiname jūsų indėlį!

Yra 2 būdai, kaip prisidėti savo taisyklėmis prie bibliotekos:

1. **Paskelbkite savo taisyklę forume**: Pasidalykite savo peradresavimo taisykle su kitais vartotojais [forume](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Tada mes galime paklausti, ar galėtume įtraukti jūsų taisyklę į biblioteką. Tai gali būti lengviau nei 2 variantas, nes papildymą galite palikti mums.
2. **Atsiųskite mums „pull request“**: Galite atsiųsti mums „pull request“. Norėdami sužinoti daugiau, peržiūrėkite [instrukcijas](https://github.com/mshibanami/redirect-web/tree/main/library).

## Kita informacija

- Visą turinį galite peržiūrėti [„GitHub“ saugykloje](https://github.com/mshibanami/redirect-web/tree/main/library).
- Bibliotekos turiniui talpiname savo privatų API serverį „Cloudflare“, kad galėtume teikti filtravimo ir kitas paslaugas. Programėlė gauna bibliotekos turinį iš šio serverio. Kaip nurodyta [mūsų privatumo politikoje](/privacy-policy), šis serveris nerenka jūsų asmeninės informacijos.

# Biblioteka

Sveiki atvykę į mūsų biblioteką, kurioje galite rasti naudingų, paruoštų naudoti taisyklių nemokamai! Jas galite pasiekti per „Redirect Web“ programėlę.

![Library Screenshot](/img/library-screenshot.webp)

Norėdami nukopijuoti taisykles į „Mano taisyklės“, galite tiesiog pasirinkti mygtuką „Gauti“:

![Get Button - iOS](/img/library-get-button-ios.gif)

## Pridėkite savo taisyklę į biblioteką

Dabartinėje biblioteroje yra tik kelios taisyklės, todėl vertiname jūsų indėlį!

Yra 2 galimybės prisidėti prie bibliotekos:

1.  **Paskelbkite savo taisyklę forume**: Pasidalykite savo peradresavimo taisykle su kitais vartotojais [forume](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Tuomet galime paprašyti, ar galėtume įtraukti jūsų taisyklę ir į biblioteką. Tai gali būti lengviau nei 2 variantas, nes pridėjimą galite palikti mums.
2.  **Atsiųskite mums „pull request“**: Galite atsiųsti mums „pull request“. Norėdami sužinoti daugiau, patikrinkite [instrukcijas](https://github.com/mshibanami/redirect-web/tree/main/library).

## Kita informacija

-   Visą turinį galite pamatyti [„GitHub“ saugykloje](https://github.com/mshibanami/redirect-web/tree/main/library).
-   Mes talpiname savo privatų API serverį „Cloudflare“ bibliotekos turiniui, leidžiantį mums teikti filtravimo ir kitas paslaugas. Programėlė gauna bibliotekos turinį iš šio serverio. Kaip nurodyta [mūsų privatumo politikoje](privacy-policy.mdx), šis serveris nerenka jūsų asmeninės informacijos.

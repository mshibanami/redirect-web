# Biblioteka

Sveiki atvykę į mūsų biblioteką, kur nemokamai rasite naudingų, paruoštų naudoti taisyklių! Jas galite pasiekti per „Redirect Web“ programėlę.

## Pridėkite savo taisyklę prie bibliotekos

Dabartinėje bibliotekoje yra tik kelios taisyklės, todėl labai vertiname jūsų indėlį!

Yra 2 būdai, kaip prisidėti savo taisyklėmis prie bibliotekos:

1.  **Paskelbkite savo taisyklę forume**: Pasidalykite savo peradresavimo taisykle su kitais vartotojais [forume](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Tada galime paklausti, ar galėtume įdėti jūsų taisyklę ir į biblioteką. Tai gali būti lengviau nei 2 variantas, nes taisyklės pridėjimą galite palikti mums.
2.  **Atsiųskite mums „pull request“**: Galite atsiųsti mums „pull request“. Norėdami sužinoti daugiau, peržiūrėkite [instrukcijas](https://github.com/mshibanami/redirect-web/tree/main/library).

## Kita informacija

-   Visą turinį galite pamatyti [„GitHub“ saugykloje](https://github.com/mshibanami/redirect-web/tree/main/library).
-   Savo privatų API serverį, skirtą bibliotekos turiniui, talpiname „Cloudflare“ platformoje, kas leidžia mums teikti filtravimo ir kitas paslaugas. Programėlė atsisiunčia bibliotekos turinį iš šio serverio. Kaip nurodyta [mūsų privatumo politikoje](privacy-policy.mdx), šis serveris nerenka jūsų asmeninės informacijos.

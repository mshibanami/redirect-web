# Sazinieties ar mums

Esam gatavi palīdzēt, ja jums ir jautājumi, atsauksmes vai problēmas ar Redirect Web. Šeit ir labākie veidi, kā sazināties:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Ja saskaraties ar kļūdām vai vēlaties pieprasīt jaunas funkcijas, lūdzu, apmeklējiet GitHub Issues lapu. Varat pārlūkot esošās problēmas, iesniegt jaunu un sekot līdzi tās progresam.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Ja jums ir jautājumi vai vēlaties saņemt palīdzību no izstrādātāja vai citiem lietotājiem, lūdzu, apmeklējiet GitHub Discussions lapu. Varat uzsākt jaunu diskusiju, pievienoties esošajām diskusijām un saņemt palīdzību no kopienas.
- **Atsauksmju forma**: Varat nosūtīt mums atsauksmes, izmantojot atsauksmju formu Redirect Web lietotnē. Tas nosūtīs e-pastu tieši izstrādātājam, tādējādi jūsu atsauksmes būs privātas un pieejamas tikai viņam, nevis citiem lietotājiem.

Paldies, ka izmantojat Redirect Web! Mēs novērtējam jūsu atbalstu un atsauksmes.

# Sazinieties ar mums

Mēs esam šeit, lai palīdzētu, ja jums ir kādi jautājumi, atsauksmes vai problēmas ar RedirectWeb. Šeit ir labākie veidi, kā sazināties:

- **[GitHub problēmas](https://github.com/mshibanami/RedirectWeb/issues)**: Ja saskaraties ar kļūdām vai vēlaties pieprasīt jaunas funkcijas, lūdzu, apmeklējiet GitHub problēmu lapu. Jūs varat pārlūkot esošās problēmas, iesniegt jaunu problēmu un sekot līdzi savas problēmas progresam.
- **[GitHub diskusijas](https://github.com/mshibanami/RedirectWeb/discussions)**: Ja jums ir kādi jautājumi vai vēlaties saņemt palīdzību no izstrādātāja vai citiem lietotājiem, lūdzu, apmeklējiet GitHub diskusiju lapu. Jūs varat sākt jaunu diskusiju, pievienoties esošajām diskusijām un saņemt palīdzību no kopienas.
- **Atsauksmju veidlapa**: Jūs varat nosūtīt mums atsauksmes, izmantojot atsauksmju veidlapu RedirectWeb lietotnē. Tas nosūtīs e-pastu tieši izstrādātājam, tāpēc jūsu atsauksmes tiks saglabātas privātas no citiem lietotājiem.

Paldies, ka izmantojat RedirectWeb! Mēs novērtējam jūsu atbalstu un atsauksmes.

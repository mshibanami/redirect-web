# Sazināties ar mums

Mēs esam šeit, lai palīdzētu, ja jums ir jautājumi, atsauksmes vai problēmas ar Redirect Web. Šeit ir labākie veidi, kā sazināties:

- **[GitHub kļūdas (Issues)](https://github.com/mshibanami/RedirectWeb/issues)**: Ja saskaraties ar kļūdām vai vēlaties pieprasīt jaunas funkcijas, lūdzu, apmeklējiet GitHub Issues lapu. Varat pārlūkot esošās kļūdas, iesniegt jaunu kļūdu un sekot līdzi tās progresam.
- **[GitHub diskusijas (Discussions)](https://github.com/mshibanami/RedirectWeb/discussions)**: Ja jums ir jautājumi vai vēlaties saņemt palīdzību no izstrādātāja vai citiem lietotājiem, lūdzu, apmeklējiet GitHub Discussions lapu. Varat sākt jaunu diskusiju, pievienoties esošajām diskusijām un saņemt palīdzību no kopienas.
- **Atsauksmju forma**: Jūs varat nosūtīt mums atsauksmes, izmantojot atsauksmju formu Redirect Web lietotnē. Tas nosūtīs e-pastu tieši izstrādātājam, tādējādi jūsu atsauksmes tiks saglabātas privātas no citiem lietotājiem.

Paldies, ka izmantojat Redirect Web! Mēs novērtējam jūsu atbalstu un atsauksmes.

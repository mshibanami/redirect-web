# Eksportēšanas/Importēšanas noteikumi

## Eksportēšanas noteikumi

### Eksportēšana operētājsistēmā macOS

Noklikšķiniet uz elipses pogas vai pogas Kopīgot, kā parādīts šajā ekrānuzņēmumā:

![Share on macOS](/img/share-on-macos.png)

Varat arī atlasīt vairākus noteikumus un eksportēt tos, veicot sekundāro klikšķi (labo peles taustiņu) uz tiem noteikumu saraksta sānjoslā:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Turklāt jūs varat eksportēt visus noteikumus no izvēļņu joslas > Fails > Eksportēt visus noteikumus:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Ja vēlaties eksportēt savus noteikumus formātā, kas ir saderīgs ar [Einara Egilsona Redirector](https://einaregilsson.com/redirector/), nospiediet taustiņu Option (`⌥`) konteksta izvēlnē vai izvēļņu joslā, un tad jūs redzēsiet šādus elementus:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web nav pilnībā saderīgs ar Redirector un otrādi. Daži iestatījumi, piemēram, "Aizstāt gadījumus" (Replace Occurrences) grupēšanas tvēruma apstrādei (Capturing Group Processing), netiek eksportēti Redirector formātā.
> :::

## Importēšanas noteikumi

### Importēšana operētājsistēmā macOS

- Vienkārši velciet un nometiet eksportētos noteikumu failus uz Redirect Web loga.
- Vai arī varat tos importēt no izvēļņu joslas > Fails > Importēt noteikumus

### Importēšana operētājsistēmā iOS

Pieņemsim, ka jūsu noteikumu fails atrodas lietotnē Fails. Atveriet to lietotnē Fails, pieskarieties pogai Kopīgot un pēc tam atlasiet Redirect Web:

# Eksportēt/Importēt noteikumus

## Eksportēt noteikumus {#export-rules}

### Eksportēt macOS {#export-on-macos}

Noklikšķiniet uz elipses pogas vai kopīgošanas pogas, kā parādīts šajā ekrānuzņēmumā:

![Kopīgot macOS](/img/share-on-macos.png)

Varat arī atlasīt vairākus noteikumus un eksportēt tos, sekundāri noklikšķinot (ar peles labo pogu) uz tiem noteikumu saraksta sānjoslā:

![Kopīgot vairākus noteikumus macOS](/img/multiselect-on-macos.png)

Turklāt jūs varat eksportēt visus noteikumus no izvēlnes joslas > Fails > Eksportēt visus noteikumus:

![Izvēlņu josla > Fails](/img/menubar-file.png)

> [!NOTE]
> Ja vēlaties eksportēt savus noteikumus formātā, kas saderīgs ar [Einar Egilsson's Redirector](https://einaregilsson.com/redirector/), nospiediet taustiņu Option (`⌥`) konteksta izvēlnē vai izvēlnes joslā, tad redzēsiet šos:
> ![Izvēlņu josla > Fails (Redirector)](/img/menubar-file-redirector.png)
>
> ![Konteksta izvēlne (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web nav pilnībā saderīgs ar Redirector, un otrādi. Daži iestatījumi, piemēram, "Aizstāt gadījumus" uztveršanas grupas apstrādei, netiek eksportēti ar Redirector formātu.
> :::

## Importēt noteikumus {#import-rules}

### Importēt macOS {#import-on-macos}

- Vienkārši velciet un nometiet eksportētos noteikumu failus uz Redirect Web logu.
- Vai arī varat tos importēt no izvēlnes joslas > Fails > Importēt noteikumus

### Importēt iOS {#import-on-ios}

Pieņemsim, ka jūsu noteikumu fails atrodas lietotnē Faili. Atveriet to lietotnē Faili, pieskarieties pogai Kopīgot un pēc tam atlasiet Redirect Web:

| 1. solis: Pieskarieties kopīgošanas pogai rīkjoslā                                     | 2. solis: Atlasiet Redirect Web                                         |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Pieskarieties kopīgošanas pogai rīkjoslā](/img/files-share-ios.png) | ![Pieskarieties lietotnes ikonai lapā](/img/share-sheet-ios.png) |

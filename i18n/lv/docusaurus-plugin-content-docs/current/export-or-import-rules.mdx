# Eksportēšanas/Importēšanas noteikumi

## Eksportēšanas noteikumi {#export-rules}

### Eksportēšana operētājsistēmā macOS {#export-on-macos}

Noklikšķiniet uz elipses pogas vai pogas Kopīgot, kā parādīts šajā ekrānuzņēmumā:

![Share on macOS](/img/share-on-macos.png)

Varat arī atlasīt vairākus noteikumus un eksportēt tos, veicot sekundāro klikšķi (labo peles pogu) uz tiem noteikumu saraksta sānjoslā:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Turklāt visus noteikumus var eksportēt no izvēļņu joslas > Fails > Eksportēt visus noteikumus:

![Menubar > File](/img/menubar-file.png)

> [!PIEZĪME]
> Ja vēlaties eksportēt noteikumus formātā, kas ir saderīgs ar [Einara Egilsona Redirector](https://einaregilsson.com/redirector/), konteksta izvēlnē vai izvēļņu joslā nospiediet taustiņu Option (`⌥`), tad redzēsiet šādus elementus:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::brīdinājums
> Redirect Web nav pilnībā saderīgs ar Redirector un otrādi. Daži iestatījumi, piemēram, "Aizstāt atkārtojumus" (Replace Occurrences) tveršanas grupu apstrādei (Capturing Group Processing), netiek eksportēti Redirector formātā.
> :::

## Importēšanas noteikumi {#import-rules}

### Importēšana operētājsistēmā macOS {#import-on-macos}

- Vienkārši velciet un nometiet eksportētos noteikumu failus uz Redirect Web loga.
- Vai arī, varat tos importēt no izvēļņu joslas > Fails > Importēt noteikumus

### Importēšana operētājsistēmā iOS {#import-on-ios}

Pieņemsim, ka jūsu noteikumu fails atrodas lietotnē Fails (Files). Atveriet to lietotnē Fails, pieskarieties pogai Kopīgot (Share) un pēc tam atlasiet Redirect Web:

| 1. solis: Pieskarieties pogai Kopīgot lietotnē Fails | 2. solis: Atlasiet Redirect Web |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Tap the share button in the toolbar](/img/files-share-ios.png) | ![Tap the app icon in the sheet](/img/share-sheet-ios.png) |

# Eksportēt/Importēt noteikumus

## Eksportēt noteikumus {#export-rules}

### Eksportēt macOS ierīcē {#export-on-macos}

Noklikšķiniet uz elipses pogas vai kopīgošanas pogas, kā parādīts šajā ekrānuzņēmumā:

<Image src={ShareOnMacOS} alt="Kopīgošanas poga macOS ierīcē" width="460" hasShadow hasRoundedCorners/>

Pēc tam konteksta izvēlnē atlasiet "Eksportēt noteikumu" vai "Kopīgot noteikumu...":

Varat arī atlasīt vairākus noteikumus un eksportēt tos, sekundāri noklikšķinot (ar peles labo pogu) uz tiem noteikumu saraksta sānjoslā:

<Image src={MultiSelectOnMacOS} alt="Kopīgot vairākus noteikumus macOS ierīcē" height="400" hasShadow hasRoundedCorners/>


Turklāt visus noteikumus varat eksportēt no izvēlnes joslas > Fails > Eksportēt visus noteikumus:

<Image src={MenubarFile} alt="Izvēlņu josla > Fails" width="400" hasShadow hasRoundedCorners/>

> [!NOTE]
> Ja vēlaties eksportēt noteikumus formātā, kas ir saderīgs ar [Einar Egilsson's Redirector](https://einaregilsson.com/redirector/), konteksta izvēlnē vai izvēlnes joslā nospiediet taustiņu Option (`⌥`), tad redzēsiet šos:
> <Image src={MenubarFileRedirector} alt="Izvēlņu josla > Fails (Redirector)" width="400" hasShadow hasRoundedCorners/>
> <br/>
> <Image src={ContextMenuRedirector} alt="Konteksta izvēlne (Redirector)" width="400" hasShadow hasRoundedCorners/>
>
> :::warning
> Redirect Web nav pilnībā saderīgs ar Redirector, un otrādi. Daži iestatījumi, piemēram, "Aizstāt gadījumus" tveršanas grupas apstrādei, netiek eksportēti ar Redirector formātu.
> :::

## Importēt noteikumus {#import-rules}

### Importēt macOS ierīcē {#import-on-macos}

- Vienkārši velciet un nometiet eksportētos noteikumu failus uz Redirect Web loga.
- Vai arī varat tos importēt no izvēlnes joslas > Fails > Importēt noteikumus

### Importēt iOS ierīcē {#import-on-ios}

Pieņemsim, ka jūsu noteikumu fails atrodas lietotnē Faili. Atveriet to lietotnē Faili, pieskarieties pogai Kopīgot un pēc tam atlasiet Redirect Web:

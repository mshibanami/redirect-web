Laipni lūgti Redirect Web atbalsta vietnē. Redirect Web ir pārlūka paplašinājums, kas pāradresē visas lapu ielādes jūsu pārlūkā, nodrošinot, ka vienmēr nokļūstat tur, kur vēlaties.

(Iepriekš zināms kā Redirect Web for Safari.)

## ⬇️ Lejupielādes saites {#download-links}

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chrome Web Store)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Firefox Add-ons)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Microsoft Edge Add-ons)
- [Safari](https://apps.apple.com/app/id1571283503) (The extension is already bundled in the native app.)

Redirect Web ir pieejams uz:

- **Mac** (Safari, Chrome, Firefox, Edge, and more)
- **iPhone**, **iPad**, un **Apple Vision Pro** (Safari)

:::note

- Lai izmantotu paplašinājumu, jums jāinstalē [Redirect Web lietotne no Mac App Store](https://apps.apple.com/app/id1571283503) neatkarīgi no izmantotā pārlūka, jo paplašinājumi paļaujas uz lietotni, lai piekļūtu jūsu noteikumiem, iestatījumiem utt.
- Vairāku pārlūku atbalsts pieejams tikai uz Mac.

:::

## Populāri lietošanas gadījumi {#popular-use-cases}

Piemēram, varat izveidot šādus noteikumus:

- Skatīt **Reddit** klasiskajā dizainā (old.reddit.com).
- Atvērt **Apple Maps**, kad parādās **Google Maps** saite.
- Slēpt Google AI pārskatus no **Google** meklēšanas rezultātiem, pievienojot katram meklēšanas vaicājumam **?udm=14**.
- Pārlēkt no Fandom lapām uz oficiālajām **Terraria** vai **Minecraft** wiki lapām.
- Izmantot **Kagi**, **Startpage**, **ChatGPT** vai jebkuru citu meklētājprogrammu kā noklusējuma meklētāju.
- Atvērt **Google Meet** saites Google Chrome pārlūkā. (Mac)
- Atvērt **Figma** saites tieši Figma lietotnē. (Mac)
- Atvērt **Notion** saites tieši Notion lietotnē. (Mac)

Visus šos noteikumus varat bez maksas iegūt kā iepriekš definētus noteikumus lietotnes Bibliotēkas sadaļā!

## Funkcijas {#features}

Redirect Web piedāvā šādas funkcijas:

- Izveidot noteikumus, izmantojot **Regular Expression** vai **Wildcard** parauga tipus
- Izmantot **back-referencing**, lai norādītu dinamiskas mērķa URL
- **Izslēgt** URL no pāradresēšanas
- **Apstrādāt iegūto tekstu**, izmantojot URL kodēšanu/dekodēšanu, base64 kodēšanu/dekodēšanu vai aizvietojot atbilstības

Turklāt varat atslēgt šādas funkcijas, abonējot vai veicot vienreizēju pirkumu:

- Iespējot **neierobežotu** noteikumu skaitu
- **Sinhronizēt noteikumus** ar citiem Mac, iPhone un iPad **izmantojot iCloud**

(Lietotnes iegāde arī atbalsta mūs Redirect Web uzturēšanā. 🙏)

Jūsu pirkums vai abonements tiek kopīgots starp jūsu Mac, iPhone, iPad un Apple Vision Pro, izmantojot Universal Purchase. Jūsu atbalsts palīdz turpināt izstrādi.

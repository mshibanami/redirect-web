Laipni lūdzam Redirect Web atbalsta vietnē, pārlūkprogrammas paplašinājumam, kas pārvirza visas lapu ielādes jūsu pārlūkprogrammā, nodrošinot, ka jūs vienmēr nonākat tur, kur vēlaties.

(Agrāk pazīstams kā Redirect Web for Safari.)

## ⬇️ Lejupielādes saites {#download-links}

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chrome interneta veikals)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Firefox papildinājumi)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Microsoft Edge papildinājumi)
- [Safari](https://apps.apple.com/app/id1571283503) (Paplašinājums jau ir iekļauts vietējā lietotnē.)

Redirect Web ir pieejams:

- **Mac** (Safari, Chrome, Firefox, Edge un citi)
- **iPhone**, **iPad** un **Apple Vision Pro** (Safari)

:::note

- Jums jāinstalē [Redirect Web lietotne no Mac App Store](https://apps.apple.com/app/id1571283503), lai izmantotu paplašinājumu, neatkarīgi no tā, kuru pārlūkprogrammu izmantojat, jo paplašinājumi paļaujas uz lietotni, lai piekļūtu jūsu noteikumiem, iestatījumiem utt.
- Vairāku pārlūkprogrammu atbalsts ir pieejams tikai Mac.

:::

## Populārākie lietošanas gadījumi {#popular-use-cases}

Piemēram, varat izveidot šādus noteikumus:

- Skatīt **Reddit** tā klasiskajā dizainā (old.reddit.com).
- Atvērt **Apple Maps** ikreiz, kad parādās **Google Maps** saite.
- Paslēpt Google AI pārskatus no **Google meklēšanas rezultātiem**, pievienojot **?udm=14** katrai meklēšanai.
- Pāriet no Fandom lapām uz oficiālajām **Terraria** vai **Minecraft** vikipēdijām.
- Izmantot **Kagi**, **Startpage**, **ChatGPT** un jebkuru citu meklētājprogrammu kā savu **noklusējuma meklētājprogrammu**.
- Atvērt **Google Meet** saites Google Chrome. (Mac)
- Atvērt **Figma** saites tieši Figma lietotnē. (Mac)
- Atvērt **Notion** saites tieši Notion lietotnē. (Mac)

Visus šos noteikumus varat iegūt bez maksas no lietotnes sadaļas Bibliotēka!

## Funkcijas {#features}

Redirect Web ir aprīkots ar šādām funkcijām:

- Izveidojiet noteikumus, izmantojot **regulārās izteiksmes** vai **aizstājējzīmju** modeļu tipus
- Izmantojiet **atpakaļatsauces**, lai norādītu dinamiskas mērķa URL adreses
- **Izslēgt** URL adreses no pārvirzīšanas
- **Apstrādāt tvertos tekstus**, izmantojot URL kodēšanu/dekodēšanu, base64 kodēšanu/dekodēšanu vai aizstājot gadījumus

Turklāt jūs varat atbloķēt šādas funkcijas, izmantojot abonementu vai vienreizēju pirkumu:

- Iespējot **neierobežotu** skaitu noteikumu
- **Sinhronizēt noteikumus** ar citiem Mac, iPhone un iPad **izmantojot iCloud**

(Lietotnes iegāde atbalsta arī mūs Redirect Web uzturēšanā. 🙏)

Jūsu pirkums vai abonements tiek koplietots jūsu Mac, iPhone, iPad un Apple Vision Pro, izmantojot Universal Purchase. Jūsu atbalsts palīdz turpināt attīstību.

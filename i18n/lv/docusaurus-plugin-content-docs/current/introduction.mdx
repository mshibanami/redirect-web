Laipni lūdzam atbalsta vietnē Redirect Web — pārlūka paplašinājumam, kas pāradresē visas lapu ielādes jūsu pārlūkprogrammā, nodrošinot, ka vienmēr nonākat tur, kur vēlaties.

(Iepriekš zināms kā Redirect Web for Safari.)

## ⬇️ Lejupielādes saites

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chrome Web Store)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Firefox Add-ons)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Microsoft Edge Add-ons)
- [Safari](https://apps.apple.com/app/id1571283503) (Paplašinājums jau ir iebūvēts vietējā lietotnē.)

Redirect Web ir pieejams:

- **Mac** (Safari, Chrome, Firefox, Edge un citiem)
- **iPhone**, **iPad** un **Apple Vision Pro** (Safari)

:::note

- Lai izmantotu paplašinājumu, jums jāinstalē [Redirect Web lietotne no Mac App Store](https://apps.apple.com/app/id1571283503), neatkarīgi no tā, kādu pārlūkprogrammu izmantojat, jo paplašinājumi paļaujas uz lietotni, lai piekļūtu jūsu noteikumiem, iestatījumiem utt.
- Vairāku pārlūku atbalsts ir pieejams tikai Mac ierīcēs.

:::

## Populārākie lietošanas gadījumi

Piemēram, varat izveidot šādus noteikumus:

- Skatīt **Reddit** tā klasiskajā dizainā (old.reddit.com).
- Atvērt **Apple Maps**, tiklīdz parādās **Google Maps** saite.
- Paslēpt Google AI pārskatus no **Google meklēšanas rezultātiem**, pievienojot **?udm=14** katrai meklēšanai.
- Pāriet no Fandom lapām uz oficiālajām **Terraria** vai **Minecraft** wiki lapām.
- Izmantot **Kagi**, **Startpage**, **ChatGPT** un jebkuru citu meklētājprogrammu kā **noklusējuma meklētājprogrammu**.
- Atvērt **Google Meet** saites Google Chrome. (Mac)
- Atvērt **Figma** saites tieši Figma lietotnē. (Mac)
- Atvērt **Notion** saites tieši Notion lietotnē. (Mac)

Visus šos noteikumus varat iegūt kā iepriekš definētus noteikumus bez maksas lietotnes sadaļā Bibliotēka!

## Funkcijas

Redirect Web piedāvā šādas funkcijas:

- Izveidojiet noteikumus, izmantojot **regulārās izteiksmes** vai **aizstājējzīmju** tipa modeļus
- Izmantojiet **atpakaļreferenci**, lai norādītu dinamiskus mērķa URL
- **Izslēgt** URL no pāradresācijas
- **Apstrādāt tvertos tekstus**, izmantojot URL kodēšanu/dekodēšanu, base64 kodēšanu/dekodēšanu vai gadījumu aizstāšanu

Turklāt varat atbloķēt šādas funkcijas, izmantojot abonementu vai vienreizēju pirkumu:

- Neierobežota skaita noteikumu **iespējošana**
- Noteikumu **sinhronizēšana** ar jūsu citiem Mac, iPhone un iPad **caur iCloud**

(Lietotnes iegāde atbalsta arī mūs Redirect Web uzturēšanā. 🙏)

Jūsu pirkums vai abonements tiek koplietots jūsu Mac, iPhone, iPad un Apple Vision Pro ierīcēs, izmantojot universālo pirkumu. Jūsu atbalsts palīdz nodrošināt nepārtrauktu attīstību.

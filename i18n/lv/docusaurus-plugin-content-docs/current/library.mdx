# Bibliotēka

Laipni lūdzam mūsu bibliotēkā, kur varat atrast noderīgus, gatavus lietošanai noteikumus bez maksas! Jūs varat tiem piekļūt, izmantojot Redirect Web lietotni.

![Bibliotēkas ekrānuzņēmums](/img/library-screenshot.webp)

Lai kopētu noteikumus uz Maniem noteikumiem, varat vienkārši atlasīt pogu Iegūt:

![Poga Iegūt - iOS](/img/library-get-button-ios.gif)

## Iesniedziet savu noteikumu bibliotēkai

Pašreizējā bibliotēkā ir tikai daži noteikumi, tāpēc mēs novērtējam jūsu ieguldījumu!

Ir 2 iespējas, kā iesniegt savus noteikumus bibliotēkai:

1. **Publicējiet savu noteikumu forumā**: Dalieties ar savu pārvirzīšanas noteikumu ar citiem lietotājiem [forumā](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Pēc tam mēs varam lūgt, vai mēs varētu ievietot jūsu noteikumu arī bibliotēkā. Tas varētu būt vieglāk nekā 2. iespēja, jo jūs varat atstāt pievienošanu mums.
2. **Nosūtiet mums pieprasījumu**: Jūs varat nosūtīt mums pieprasījumu. Lūdzu, pārbaudiet [instrukcijas](https://github.com/mshibanami/redirect-web/tree/main/library), lai uzzinātu vairāk.

## Cita informācija

- Visu saturu varat apskatīt [GitHub repozitorijā](https://github.com/mshibanami/redirect-web/tree/main/library).
- Mēs mitinām savu privāto API serveri Cloudflare bibliotēkas saturam, ļaujot mums nodrošināt filtrēšanu un citus pakalpojumus. Lietotne ielādē bibliotēkas saturu no šī servera. Kā norādīts [mūsu privātuma politikā](/privacy-policy.mdx), šis serveris neapkopo jūsu personisko informāciju.

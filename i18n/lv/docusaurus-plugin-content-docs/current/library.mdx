# Bibliotēka

Laipni lūdzam mūsu bibliotēkā, kur varat bez maksas atrast noderīgus, gatavus lietošanai noteikumus! Varat tiem piekļūt, izmantojot Redirect Web lietotni.

![Library Screenshot](/img/library-screenshot.webp)

Lai nokopētu noteikumus uz sadaļu Mani noteikumi, vienkārši atlasiet pogu Iegūt (Get):

![Get Button - iOS](/img/library-get-button-ios.gif)

## Iesniedziet savu noteikumu bibliotēkai

Pašreizējā bibliotēkā ir tikai daži noteikumi, tāpēc mēs novērtējam jūsu ieguldījumu!

Ir 2 iespējas, kā iesniegt savus noteikumus bibliotēkai:

1.  **Publicējiet savu noteikumu forumā**: Kopīgojiet savu novirzīšanas (redirect) noteikumu ar citiem lietotājiem [forumā](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Pēc tam mēs varam jautāt, vai mēs varētu iekļaut jūsu noteikumu arī bibliotēkā. Tas varētu būt vieglāk nekā 2. iespēja, jo jūs varat atstāt pievienošanu mums.
2.  **Nosūtiet mums "pull request"**: Varat nosūtīt mums "pull request". Lūdzu, pārbaudiet [instrukcijas](https://github.com/mshibanami/redirect-web/tree/main/library), lai uzzinātu vairāk.

## Cita informācija

-   Visu saturu varat skatīt [GitHub repozitorijā](https://github.com/mshibanami/redirect-web/tree/main/library).
-   Bibliotēkas saturam mēs mitinām savu privāto API serveri uz Cloudflare, ļaujot mums nodrošināt filtrēšanas un citus pakalpojumus. Lietotne iegūst bibliotēkas saturu no šī servera. Kā norādīts [mūsu privātuma politikā](/privacy-policy), šis serveris neapkopo jūsu personisko informāciju.

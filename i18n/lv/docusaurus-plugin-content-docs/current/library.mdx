# Bibliotēka

Laipni lūgti mūsu bibliotēkā, kur bez maksas atradīsiet noderīgus, gatavus lietošanai noteikumus! Jūs tiem varat piekļūt, izmantojot lietotni Redirect Web.

![Library Screenshot](/img/library-screenshot.webp)

Lai kopētu noteikumus uz sadaļu Mani noteikumi, vienkārši atlasiet pogu "Saņemt":

![Get Button - iOS](/img/library-get-button-ios.gif)

## Iesniedziet savu noteikumu bibliotēkai

Pašreizējā bibliotēkā ir tikai daži noteikumi, tāpēc mēs novērtējam jūsu ieguldījumu!

Ir 2 iespējas, kā iesniegt savus noteikumus bibliotēkai:

1.  **Publicējiet savu noteikumu forumā**: Kopīgojiet savu novirzīšanas noteikumu ar citiem lietotājiem [forumā](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Pēc tam mēs varam jautāt, vai mēs varētu iekļaut jūsu noteikumu arī bibliotēkā. Tas varētu būt vieglāk nekā 2. iespēja, jo jūs varat atstāt pievienošanu mūsu ziņā.
2.  **Nosūtiet mums izmaiņu pieprasījumu (pull request)**: Jūs varat nosūtīt mums izmaiņu pieprasījumu. Lūdzu, pārbaudiet [instrukcijas](https://github.com/mshibanami/redirect-web/tree/main/library), lai uzzinātu vairāk.

## Cita informācija

-   Visu saturu varat redzēt [GitHub repozitorijā](https://github.com/mshibanami/redirect-web/tree/main/library).
-   Mēs uzturam savu privāto API serveri uz Cloudflare bibliotēkas saturam, kas ļauj mums nodrošināt filtrēšanas un citus pakalpojumus. Lietotne saņem bibliotēkas saturu no šī servera. Kā norādīts [mūsu privātuma politikā](privacy-policy.mdx), šis serveris neapkopo jūsu personisko informāciju.

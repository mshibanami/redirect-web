# Bibliotēka

Laipni lūdzam mūsu bibliotēkā, kurā bez maksas varat atrast noderīgus, gatavus noteikumus! Jūs varat tiem piekļūt, izmantojot lietotni Redirect Web.

![Bibliotēkas ekrānuzņēmums](/img/library-screenshot.webp)

Lai kopētu noteikumus uz Maniem noteikumiem, varat vienkārši atlasīt pogu Iegūt:

![Iegūt pogu - iOS](/img/library-get-button-ios.gif)

## Iesniedziet savu noteikumu bibliotēkai

Pašreizējā bibliotēkā ir tikai daži noteikumi, tāpēc mēs novērtējam jūsu ieguldījumu!

Ir 2 iespējas, kā iesniegt savus noteikumus bibliotēkai:

1. **Publicējiet savu noteikumu forumā**: Dalieties ar savu novirzīšanas noteikumu ar citiem lietotājiem [forumā](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Tad mēs varam lūgt, vai mēs varētu ievietot jūsu noteikumu arī bibliotēkā. Tas varētu būt vieglāk nekā 2. variants, jo jūs varat atstāt pievienošanu mums.
2. **Nosūtiet mums pull pieprasījumu**: Jūs varat nosūtīt mums pull pieprasījumu. Lūdzu, pārbaudiet [instrukcijas](https://github.com/mshibanami/redirect-web/tree/main/library), lai uzzinātu vairāk.

## Cita informācija

- Visu saturu varat apskatīt [repozitorijā GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- Mēs mitinām savu privāto API serveri Cloudflare bibliotēkas saturam, kas ļauj mums nodrošināt filtrēšanu un citus pakalpojumus. Lietotne ielādē bibliotēkas saturu no šī servera. Kā norādīts [mūsu privātuma politikā](privacy-policy.mdx), šis serveris neapkopo jūsu personisko informāciju.

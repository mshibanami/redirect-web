# Problēmu novēršana

Šajā lapā ir uzskaitītas biežākās problēmas un to risinājumi, lietojot Redirect Web.

<TOCInline toc={toc} />

## Pāradresācijas noteikumi nedarbojas

Ja jūsu noteikumi nedarbojas, kā paredzēts, lūdzu, pārbaudiet sekojošo:

- **Pārlūkprogrammas paplašinājums ir instalēts un iespējots pareizajai lapai:** Pārliecinieties, vai esat instalējis Redirect Web pārlūkprogrammas paplašinājumu un vai tas ir iespējots jūsu pārlūkprogrammā tīmekļa lapai, no kuras vēlaties veikt pāradresāciju. Ja varat atvērt paplašinājuma uznirstošo logu, kā parādīts zemāk, avota tīmekļa lapā, tas ir instalēts pareizi:
    <Image src={ExtensionPopup} alt="Redirect Web paplašinājuma uznirstošais logs" width="260" hasShadow/>

- **Noteikumu konfigurācija ir pareiza:** Vēlreiz pārbaudiet savus pāradresācijas noteikumus Redirect Web vietējā lietotnē. Pārliecinieties, vai avota URL modeļi, mērķa URL un visi nosacījumi ir pareizi iestatīti. Lai pārbaudītu, vai jūsu noteikums ir derīgs, ievadiet piemēra URL sadaļā "Piemēri" ekrānā "Rediģēt noteikumu". Ja jūsu noteikums ir derīgs, zem jūsu ievades redzēsiet paredzēto galamērķa URL, piemēram:
    <Image src={RuleEditExampleResult} alt="Noteikumu rediģēšanas ekrāna piemērs, kas parāda piemēra URL rezultātu" width="260" hasShadow/>

## Paplašinājuma uznirstošajā logā tiek parādīta kļūda

<Image src={ErrorOverlay} alt="Redirect Web kļūdas pārklājums" width="260" hasShadow hasRoundedCorners/>

> Redirect Web lietotne nav instalēta vai nedarbojas pareizi.
> Lūdzu, iegūstiet lietotni no App Store.

Ja, atverot paplašinājuma uznirstošo logu, redzat iepriekš minēto kļūdu, tas nozīmē, ka Redirect Web vietējā lietotne nav instalēta vai nedarbojas pareizi.

Lūdzu, pārliecinieties, vai esat instalējis jaunāko Redirect Web versiju no [App Store](https://apps.apple.com/app/id1571283503).

Pēc vietējās lietotnes instalēšanas, iespējams, būs jārestartē pārlūkprogramma.

> [!NOTE]
> * Lai izmantotu pārlūkprogrammas paplašinājumu, jums ir jāinstalē Redirect Web vietējā lietotne. Paplašinājums pats par sevi nedarbojas.
> * Safari lietotājiem nav jāinstalē paplašinājums atsevišķi, jo tas ir iekļauts vietējā lietotnē.

# Contact opnemen

We helpen je graag als je vragen, feedback of problemen hebt met Redirect Web. Dit zijn de beste manieren om contact op te nemen:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Als je bugs tegenkomt of nieuwe functies wilt aanvragen, bezoek dan de GitHub Issues-pagina. Je kunt bestaande problemen bekijken, een nieuw probleem indienen en de voortgang van je probleem volgen.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Als je vragen hebt of hulp wilt krijgen van de ontwikkelaar of andere gebruikers, bezoek dan de GitHub Discussions-pagina. Je kunt een nieuwe discussie starten, deelnemen aan bestaande discussies en hulp krijgen van de community.
- **Feedbackformulier**: Je kunt ons feedback sturen via het feedbackformulier in de Redirect Web-app. Dit stuurt een e-mail rechtstreeks naar de ontwikkelaar, zodat je feedback priv√© blijft voor andere gebruikers.

Bedankt voor het gebruik van Redirect Web! We waarderen je steun en feedback.

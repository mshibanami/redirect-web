# Bibliotheek

Welkom in onze bibliotheek, waar je gratis handige, kant-en-klare regels kunt vinden! Je hebt er toegang toe via de Redirect Web-app.

![Bibliotheek Screenshot](/img/library-screenshot.webp)

Om de regels naar Mijn Regels te kopiëren, kun je eenvoudig de knop "Ophalen" selecteren:

![Ophalen Knop - iOS](/img/library-get-button-ios.gif)

## Draag je regel bij aan de bibliotheek

De huidige bibliotheek heeft slechts een paar regels, dus we waarderen je bijdrage!

Er zijn 2 opties om je regels bij te dragen aan de bibliotheek:

1. **Plaats je regel op het Forum**: Deel je omleidingsregel met andere gebruikers op [het forum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Dan kunnen we vragen of we je regel ook in de bibliotheek mogen opnemen. Dit is misschien gemakkelijker dan optie 2, omdat je de toevoeging aan ons kunt overlaten.
2. **Stuur ons een pull request**: Je kunt ons een pull request sturen. Bekijk [de instructies](https://github.com/mshibanami/redirect-web/tree/main/library) voor meer informatie.

## Overige informatie

- Je kunt alle inhoud bekijken in [de repository op GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- We hosten onze privé API-server op Cloudflare voor de bibliotheekinhoud, waardoor we filtering en andere services kunnen bieden. De app haalt de bibliotheekinhoud van deze server. Zoals vermeld in [ons privacybeleid](privacy-policy.mdx), verzamelt deze server geen persoonlijke informatie.

# Bibliotheek

Welkom in onze bibliotheek, waar u gratis nuttige, kant-en-klare regels kunt vinden! U kunt ze openen via de Redirect Web-app.

![Bibliotheek Screenshot](/img/library-screenshot.webp)

Om de regels naar Mijn Regels te kopiëren, kunt u eenvoudig de knop 'Ophalen' selecteren:

![Ophalen Knop - iOS](/img/library-get-button-ios.gif)

## Draag uw regel bij aan de bibliotheek

De huidige bibliotheek heeft slechts een paar regels, dus we waarderen uw bijdrage!

Er zijn 2 opties om uw regels bij te dragen aan de bibliotheek:

1. **Plaats uw regel op het Forum**: Deel uw omleidingsregel met andere gebruikers op [het forum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Dan kunnen we vragen of we uw regel ook in de bibliotheek mogen plaatsen. Dit is misschien eenvoudiger dan optie 2, omdat u de toevoeging aan ons kunt overlaten.
2. **Stuur ons een pull request**: U kunt ons een pull request sturen. Raadpleeg [de instructies](https://github.com/mshibanami/redirect-web/tree/main/library) voor meer informatie.

## Overige informatie

- U kunt alle inhoud bekijken in [de repository op GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- We hosten onze privé API-server op Cloudflare voor de bibliotheekinhoud, waardoor we filtering en andere services kunnen bieden. De app haalt de bibliotheekinhoud van deze server op. Zoals vermeld in [ons privacybeleid](/privacy-policy.mdx), verzamelt deze server geen persoonlijke informatie van u.

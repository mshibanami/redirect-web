# Skontaktuj się z nami

Jesteśmy tutaj, aby pomóc, jeśli masz pytania, uwagi lub problemy z Redirect Web. Oto najlepsze sposoby, aby się z nami skontaktować:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Jeśli napotkasz jakiekolwiek błędy lub chcesz poprosić o nowe funkcje, odwiedź stronę GitHub Issues. Możesz przeglądać istniejące zgłoszenia, wysyłać nowe i śledzić ich postępy.
- **[Dyskusje GitHub](https://github.com/mshibanami/RedirectWeb/discussions)**: Jeśli masz pytania lub chcesz uzyskać pomoc od dewelopera lub innych użytkowników, odwiedź stronę Dyskusje GitHub. Możesz rozpocząć nową dyskusję, dołączyć do istniejących i uzyskać pomoc od społeczności.
- **Formularz opinii**: Możesz przesłać nam opinię za pośrednictwem formularza opinii w aplikacji Redirect Web. Spowoduje to wysłanie wiadomości e-mail bezpośrednio do dewelopera, dzięki czemu Twoja opinia zostanie zachowana w prywatności przed innymi użytkownikami.

Dziękujemy za korzystanie z Redirect Web! Doceniamy Twoje wsparcie i opinie.

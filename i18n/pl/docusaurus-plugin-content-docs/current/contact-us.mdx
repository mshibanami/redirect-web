# Skontaktuj się z nami

Jesteśmy tutaj, aby pomóc, jeśli masz jakiekolwiek pytania, uwagi lub problemy z RedirectWeb. Oto najlepsze sposoby na skontaktowanie się z nami:

- **[Problemy GitHub](https://github.com/mshibanami/RedirectWeb/issues)**: Jeśli napotkasz jakiekolwiek błędy lub chcesz zgłosić nowe funkcje, odwiedź stronę GitHub Issues. Możesz przeglądać istniejące problemy, zgłaszać nowe i śledzić postępy swojego problemu.
- **[Dyskusje GitHub](https://github.com/mshibanami/RedirectWeb/discussions)**: Jeśli masz jakiekolwiek pytania lub chcesz uzyskać pomoc od dewelopera lub innych użytkowników, odwiedź stronę GitHub Discussions. Możesz rozpocząć nową dyskusję, dołączyć do istniejących dyskusji i uzyskać pomoc od społeczności.
- **Formularz opinii**: Możesz wysłać nam opinię za pośrednictwem formularza opinii w aplikacji RedirectWeb. Spowoduje to wysłanie wiadomości e-mail bezpośrednio do dewelopera, dzięki czemu Twoja opinia pozostanie prywatna dla innych użytkowników.

Dziękujemy za korzystanie z RedirectWeb! Doceniamy Twoje wsparcie i opinie.

# Eksport/Import reguł

## Eksport reguł {#export-rules}

### Eksport na macOS {#export-on-macos}

Kliknij przycisk wielokropka lub przycisk Udostępnij, jak pokazano na tym zrzucie ekranu:

![Udostępnij na macOS](/img/share-on-macos.png)

Możesz również zaznaczyć wiele reguł i wyeksportować je, klikając prawym przyciskiem myszy na pasku bocznym listy reguł:

![Udostępnij wiele reguł na macOS](/img/multiselect-on-macos.png)

Dodatkowo możesz wyeksportować wszystkie reguły z paska menu > Plik > Eksportuj wszystkie reguły:

![Pasek menu > Plik](/img/menubar-file.png)

> [!NOTE]
> Jeśli chcesz wyeksportować swoje reguły w formacie zgodnym z [Redirector Einar Egilsson](https://einaregilsson.com/redirector/), naciśnij klawisz Option (`⌥`) w menu kontekstowym lub na pasku menu, a następnie zobaczysz:
> ![Pasek menu > Plik (Redirector)](/img/menubar-file-redirector.png)
>
> ![Menu kontekstowe (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web nie jest w pełni kompatybilny z Redirector i odwrotnie. Niektóre ustawienia, takie jak „Zastąp wystąpienia” dla przetwarzania grup przechwytujących, nie są eksportowane w formacie Redirector.
> :::

## Import reguł {#import-rules}

### Import na macOS {#import-on-macos}

- Po prostu przeciągnij i upuść wyeksportowane pliki reguł do okna Redirect Web.
- Lub możesz je również zaimportować z paska menu > Plik > Importuj reguły

### Import na iOS {#import-on-ios}

Załóżmy, że plik reguł znajduje się w aplikacji Pliki. Otwórz go w Plikach, dotknij przycisku Udostępnij, a następnie wybierz Redirect Web:

| Krok 1: Dotknij przycisku Udostępnij na pasku narzędzi w Plikach | Krok 2: Wybierz Redirect Web |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Dotknij przycisku udostępniania na pasku narzędzi](/img/files-share-ios.png) | ![Dotknij ikony aplikacji w arkuszu](/img/share-sheet-ios.png) |

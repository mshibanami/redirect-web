# Eksportowanie/importowanie reguł

## Eksportowanie reguł {#export-rules}

### Eksportowanie na macOS {#export-on-macos}

Kliknij przycisk z trzema kropkami lub przycisk Udostępnij, jak pokazano na zrzucie ekranu:

![Share on macOS](/img/share-on-macos.png)

Możesz także zaznaczyć wiele reguł i wyeksportować je, klikając prawym przyciskiem myszy (lub dwoma palcami) na pasku bocznym z listą reguł:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Ponadto, możesz wyeksportować wszystkie reguły z paska menu > Plik > Eksportuj wszystkie reguły:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Jeśli chcesz wyeksportować swoje reguły w formacie kompatybilnym z [Redirector Einar Egilsson](https://einaregilsson.com/redirector/), naciśnij klawisz Option (`⌥`) w menu kontekstowym lub na pasku menu, a zobaczysz następujące opcje:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web nie jest w pełni kompatybilny z Redirector, i na odwrót. Niektóre ustawienia, takie jak „Zastąp wystąpienia” dla przetwarzania grup przechwytujących, nie są eksportowane w formacie Redirector.
> :::

## Importowanie reguł {#import-rules}

### Importowanie na macOS {#import-on-macos}

- Po prostu przeciągnij i upuść wyeksportowane pliki reguł do okna Redirect Web.
- Lub możesz je zaimportować z paska menu > Plik > Importuj reguły

### Importowanie na iOS {#import-on-ios}

Załóżmy, że plik z regułami znajduje się w aplikacji Pliki. Otwórz go w Plikach, naciśnij przycisk Udostępnij, a następnie wybierz Redirect Web:

| Krok 1: Naciśnij przycisk Udostępnij w aplikacji Pliki                                     | Krok 2: Wybierz Redirect Web                                         |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Tap the share button in the toolbar](/img/files-share-ios.png) | ![Tap the app icon in the sheet](/img/share-sheet-ios.png) |

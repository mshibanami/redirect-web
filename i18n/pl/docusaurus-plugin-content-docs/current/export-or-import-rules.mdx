# Eksport/Import Reguł

## Eksport Reguł

### Eksport na macOS

Kliknij przycisk wielokropka lub przycisk Udostępnij, jak pokazano na tym zrzucie ekranu:

![Udostępnij na macOS](/img/share-on-macos.png)

Możesz także zaznaczyć wiele reguł i wyeksportować je, klikając je wtórnie (prawym przyciskiem myszy) na pasku bocznym z listą reguł:

![Udostępnianie wielu reguł na macOS](/img/multiselect-on-macos.png)

Ponadto możesz wyeksportować wszystkie reguły z paska menu > Plik > Eksportuj wszystkie reguły:

![Pasek menu > Plik](/img/menubar-file.png)

> [!NOTE]
> Jeśli chcesz wyeksportować swoje reguły w formacie zgodnym z [Redirectorem Einara Egilssona](https://einaregilsson.com/redirector/), naciśnij klawisz Option (`⌥`) w menu kontekstowym lub na pasku menu, wtedy zobaczysz:
> ![Pasek menu > Plik (Redirector)](/img/menubar-file-redirector.png)
>
> ![Menu kontekstowe (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web nie jest w pełni kompatybilny z Redirectorem i vice versa. Niektóre ustawienia, takie jak „Zastąp wystąpienia” (Replace Occurrences) dla przetwarzania grup przechwytujących, nie są eksportowane w formacie Redirectora.
> :::

## Import Reguł

### Import na macOS

-   Po prostu przeciągnij i upuść wyeksportowane pliki reguł do okna Redirect Web.
-   Możesz także zaimportować je z paska menu > Plik > Importuj reguły.

### Import na iOS

Załóżmy, że plik z regułami znajduje się w aplikacji Pliki. Otwórz go w Plikach, naciśnij przycisk Udostępnij, a następnie wybierz Redirect Web:

Witamy na stronie wsparcia Redirect Web — rozszerzenia przeglądarki, które przekierowuje wszystkie ładowania stron w Twojej przeglądarce, dzięki czemu zawsze trafiasz tam, gdzie chcesz.

(Wcześniej znane jako Redirect Web for Safari.)

## ⬇️ Linki do pobrania {#download-links}

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chrome Web Store)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Firefox Add-ons)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Microsoft Edge Add-ons)
- [Safari](https://apps.apple.com/app/id1571283503) (The extension is already bundled in the native app.)

Redirect Web jest dostępny na:

- **Mac** (Safari, Chrome, Firefox, Edge i inne)
- **iPhone**, **iPad** i **Apple Vision Pro** (Safari)

:::note

- Aby korzystać z rozszerzenia, musisz zainstalować [aplikację Redirect Web z Mac App Store](https://apps.apple.com/app/id1571283503), niezależnie od używanej przeglądarki, ponieważ rozszerzenia korzystają z aplikacji, aby uzyskać dostęp do reguł, ustawień itp.
- Wsparcie dla wielu przeglądarek jest dostępne tylko na Mac.

:::

## Najpopularniejsze zastosowania {#popular-use-cases}

Na przykład możesz utworzyć następujące reguły:

- Wyświetl **Reddit** w klasycznym wyglądzie (old.reddit.com).
- Otwieraj **Apple Maps** gdy pojawi się link do **Google Maps**.
- Ukryj Google’s AI Overviews z wyników wyszukiwania **Google** dodając **?udm=14** do każdego wyszukiwania.
- Przeskakuj ze stron Fandom na oficjalne wiki **Terraria** lub **Minecraft**.
- Używaj **Kagi**, **Startpage**, **ChatGPT** i dowolnej innej wyszukiwarki jako swojej **domyślnej wyszukiwarki**.
- Otwieraj linki **Google Meet** w Google Chrome. (Mac)
- Otwieraj linki **Figma** bezpośrednio w aplikacji Figma. (Mac)
- Otwieraj linki **Notion** bezpośrednio w aplikacji Notion. (Mac)

Wszystkie te reguły znajdziesz w aplikacji w sekcji Library — dostępne bezpłatnie!

## Funkcje {#features}

Redirect Web oferuje następujące funkcje:

- Twórz reguły używając wzorców typu **Regular Expression** lub **Wildcard**
- Używaj **back-referencing** do określania dynamicznych docelowych adresów URL
- **Wykluczaj** adresy URL z przekierowań
- **Przetwarzaj przechwycone teksty** przy użyciu kodowania/dekodowania URL, kodowania/dekodowania base64 lub zamiany wystąpień

Dodatkowo możesz odblokować następujące funkcje poprzez subskrypcję lub jednorazowy zakup:

- Możliwość dodania **nieograniczonej** liczby reguł
- **Synchronizacja reguł** z innymi urządzeniami Mac, iPhone i iPad **za pomocą iCloud**

(Zakup aplikacji pomaga nam również w utrzymaniu Redirect Web. 🙏)

Twój zakup lub subskrypcja jest współdzielona między Mac, iPhone, iPad i Apple Vision Pro dzięki Universal Purchase. Twoje wsparcie pomaga w dalszym rozwoju.

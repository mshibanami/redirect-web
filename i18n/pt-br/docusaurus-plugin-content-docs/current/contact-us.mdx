# Contate-nos

Estamos aqui para ajudar se você tiver alguma dúvida, feedback ou problema com o RedirectWeb. Aqui estão as melhores maneiras de entrar em contato:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Se você encontrar algum bug ou quiser solicitar novos recursos, visite a página do GitHub Issues. Você pode navegar pelos problemas existentes, enviar um novo problema e acompanhar o progresso do seu problema.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Se você tiver alguma dúvida ou quiser obter ajuda do desenvolvedor ou de outros usuários, visite a página do GitHub Discussions. Você pode iniciar uma nova discussão, participar de discussões existentes e obter ajuda da comunidade.
- **Formulário de Feedback**: Você pode nos enviar feedback através do formulário de feedback no aplicativo RedirectWeb. Isso enviará um e-mail diretamente para o desenvolvedor, para que seu feedback seja mantido privado de outros usuários.

Obrigado por usar o RedirectWeb! Agradecemos seu apoio e feedback.

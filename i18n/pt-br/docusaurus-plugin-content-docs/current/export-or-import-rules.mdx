# Exportar/Importar Regras

## Exportar Regras {#export-rules}

### Exportar no macOS {#export-on-macos}

Clique no botão de reticências ou no botão Compartilhar, conforme mostrado nesta captura de tela:

<Image src={ShareOnMacOS} alt="Botão Compartilhar no macOS" width="460" hasShadow hasRoundedCorners/>

Em seguida, selecione "Exportar Regra" ou "Compartilhar Regra..." no menu de contexto:

Você também pode selecionar várias regras e exportá-las clicando com o botão secundário (botão direito) nelas na barra lateral de listagem de regras:

<Image src={MultiSelectOnMacOS} alt="Compartilhar várias regras no macOS" height="400" hasShadow hasRoundedCorners/>


Além disso, você pode exportar todas as regras da barra de menus > Arquivo > Exportar Todas as Regras:

<Image src={MenubarFile} alt="Barra de Menus > Arquivo" width="400" hasShadow hasRoundedCorners/>

> [!NOTE]
> Se você deseja exportar suas regras no formato compatível com o [Redirector de Einar Egilsson](https://einaregilsson.com/redirector/), pressione a tecla Option (`⌥`) no menu de contexto ou na barra de menus, então você verá isto:
> <Image src={MenubarFileRedirector} alt="Barra de Menus > Arquivo (Redirector)" width="400" hasShadow hasRoundedCorners/>
> <br/>
> <Image src={ContextMenuRedirector} alt="Menu de Contexto (Redirector)" width="400" hasShadow hasRoundedCorners/>
>
> :::warning
> O Redirect Web não é totalmente compatível com o Redirector, e vice-versa. Algumas configurações, como "Substituir Ocorrências" para Processamento de Grupo de Captura, não são exportadas no formato do Redirector.
> :::

## Importar Regras {#import-rules}

### Importar no macOS {#import-on-macos}

- Basta arrastar e soltar os arquivos de regra exportados na janela do Redirect Web.
- Ou, você também pode importá-los da barra de menus > Arquivo > Importar Regras

### Importar no iOS {#import-on-ios}

Suponha que seu arquivo de regra esteja no aplicativo Arquivos. Abra-o em Arquivos, toque no botão Compartilhar e selecione Redirect Web:

| Passo 1: Toque no botão Compartilhar em Arquivos                                                                          | Passo 2: Selecione Redirect Web                                                                              |

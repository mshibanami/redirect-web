# Exportar/Importar Regras

## Exportar Regras

### Exportar no macOS

Clique no botão de reticências ou no botão Compartilhar, conforme mostrado nesta captura de tela:

![Compartilhar no macOS](/img/share-on-macos.png)

Você também pode selecionar múltiplas regras e exportá-las clicando com o botão secundário (botão direito) sobre elas na barra lateral de listagem de regras:

![Compartilhar múltiplas regras no macOS](/img/multiselect-on-macos.png)

Além disso, você pode exportar todas as regras na barra de menus > Arquivo > Exportar Todas as Regras:

![Barra de Menus > Arquivo](/img/menubar-file.png)

> [!NOTA]
> Se você deseja exportar suas regras no formato compatível com o [Redirector de Einar Egilsson](https://einaregilsson.com/redirector/), pressione a tecla Option (`⌥`) no menu de contexto ou na barra de menus, então você verá isto:
> ![Barra de Menus > Arquivo (Redirector)](/img/menubar-file-redirector.png)
>
> ![Menu de Contexto (Redirector)](/img/context-menu-redirector.png)
>
> :::cuidado
> O Redirect Web não é totalmente compatível com o Redirector, e vice-versa. Algumas configurações, como "Substituir Ocorrências" para o Processamento de Grupos de Captura, não são exportadas com o formato do Redirector.
> :::

## Importar Regras

### Importar no macOS

- Basta arrastar e soltar os arquivos de regra exportados para a janela do Redirect Web.
- Ou, você também pode importá-los na barra de menus > Arquivo > Importar Regras

### Importar no iOS

Suponha que seu arquivo de regra esteja no aplicativo Arquivos. Abra-o no Arquivos, toque no botão Compartilhar e, em seguida, selecione Redirect Web:

| Passo 1: Toque no botão Compartilhar no Arquivos | Passo 2: Selecione Redirect Web           |
| ------------------------------------- | ------------------------------------- |
| ![Texto alternativo](/img/files-share-ios.png) | ![Texto alternativo](/img/share-sheet-ios.png) |

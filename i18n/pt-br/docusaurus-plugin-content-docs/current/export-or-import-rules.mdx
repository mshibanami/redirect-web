# Exportar/Importar Regras

## Exportar Regras {#export-rules}

### Exportar no macOS {#export-on-macos}

Clique no botão de reticências ou no botão Compartilhar, conforme mostrado nesta captura de tela:

![Compartilhar no macOS](/img/share-on-macos.png)

Você também pode selecionar várias regras e exportá-las clicando com o botão direito na barra lateral de listagem de regras:

![Compartilhar várias regras no macOS](/img/multiselect-on-macos.png)

Além disso, você pode exportar todas as regras na barra de menus > Arquivo > Exportar Todas as Regras:

![Barra de Menus > Arquivo](/img/menubar-file.png)

> [!NOTE]
> Se você deseja exportar suas regras no formato compatível com o [Redirector de Einar Egilsson](https://einaregilsson.com/redirector/), pressione a tecla Option (`⌥`) no menu de contexto ou na barra de menus, então você verá:
> ![Barra de Menus > Arquivo (Redirector)](/img/menubar-file-redirector.png)
>
> ![Menu de Contexto (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> O Redirect Web não é totalmente compatível com o Redirector, e vice-versa. Algumas configurações, como "Substituir Ocorrências" para Processamento de Grupo de Captura, não são exportadas com o formato do Redirector.
> :::

## Importar Regras {#import-rules}

### Importar no macOS {#import-on-macos}

- Basta arrastar e soltar os arquivos de regra exportados na janela do Redirect Web.
- Ou, você também pode importá-los na barra de menus > Arquivo > Importar Regras

### Importar no iOS {#import-on-ios}

Suponha que seu arquivo de regra esteja no aplicativo Arquivos. Abra-o em Arquivos, toque no botão Compartilhar e selecione Redirect Web:

| Passo 1: Toque no botão Compartilhar na barra de ferramentas de Arquivos | Passo 2: Selecione Redirect Web |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Toque no botão de compartilhamento na barra de ferramentas](/img/files-share-ios.png) | ![Toque no ícone do aplicativo na folha](/img/share-sheet-ios.png) |

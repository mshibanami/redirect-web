# Exportar/Importar Regras

## Exportar Regras {#export-rules}

### Exportar no macOS {#export-on-macos}

Clique no botão de reticências ou no botão Compartilhar, conforme mostrado nesta captura de tela:

![Share on macOS](/img/share-on-macos.png)

Você também pode selecionar várias regras e exportá-las clicando com o botão secundário (botão direito) sobre elas na barra lateral de listagem de regras:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Além disso, você pode exportar todas as regras pela barra de menu > Arquivo > Exportar Todas as Regras:

![Menubar > File](/img/menubar-file.png)

> [!NOTA]
> Se você deseja exportar suas regras no formato compatível com o [Redirector de Einar Egilsson](https://einaregilsson.com/redirector/), pressione a tecla Option (`⌥`) no menu de contexto ou na barra de menu, então você verá isto:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web não é totalmente compatível com Redirector, e vice-versa. Algumas configurações, como "Substituir Ocorrências" para Processamento de Grupo de Captura, não são exportadas com o formato do Redirector.
> :::

## Importar Regras {#import-rules}

### Importar no macOS {#import-on-macos}

- Basta arrastar e soltar os arquivos de regra exportados na janela do Redirect Web.
- Ou, você também pode importá-los pela barra de menu > Arquivo > Importar Regras

### Importar no iOS {#import-on-ios}

Suponha que seu arquivo de regra esteja no aplicativo Arquivos. Abra-o nos Arquivos, toque no botão Compartilhar e, em seguida, selecione Redirect Web:

| Passo 1: Toque no botão Compartilhar nos Arquivos                      | Passo 2: Selecione Redirect Web                                     |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Tap the share button in the toolbar](/img/files-share-ios.png) | ![Tap the app icon in the sheet](/img/share-sheet-ios.png) |

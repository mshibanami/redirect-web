# Biblioteca

Bem-vindo à nossa biblioteca, onde você pode encontrar regras úteis e prontas para usar gratuitamente! Você pode acessá-las através do aplicativo Redirect Web.

![Captura de Tela da Biblioteca](/img/library-screenshot.webp)

Para copiar as regras para Minhas Regras, basta selecionar o botão Obter:

![Botão Obter - iOS](/img/library-get-button-ios.gif)

## Contribua com sua regra para a biblioteca

A biblioteca atual tem poucas regras, então agradecemos sua contribuição!

Existem 2 opções para contribuir com suas regras para a biblioteca:

1. **Publique sua regra no Fórum**: Compartilhe sua regra de redirecionamento com outros usuários no [fórum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Então, podemos solicitar se podemos adicionar sua regra à biblioteca também. Isso pode ser mais fácil do que a opção 2, porque você pode deixar a adição conosco.
2. **Envie-nos um pull request**: Você pode nos enviar um pull request. Por favor, verifique [as instruções](https://github.com/mshibanami/redirect-web/tree/main/library) para saber mais.

## Outras Informações

- Você pode ver todo o conteúdo no [repositório no GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- Nós hospedamos nosso servidor de API privado no Cloudflare para o conteúdo da biblioteca, permitindo-nos fornecer filtragem e outros serviços. O aplicativo busca o conteúdo da biblioteca neste servidor. Conforme declarado em [nossa política de privacidade](privacy-policy.mdx), este servidor não coleta suas informações pessoais.

# Caso 1: Redirecionar para outro site

Suponha que você seja viciado em Twitter e, para quebrar o hábito, decida ir para o [Unrush](https://unrush.org) para obter frases motivacionais e uma breve meditação sempre que abrir o Twitter acidentalmente. Vamos criar uma regra para isso!

### Passo 1. Abra a tela "Editar Regra"

Vamos abrir a tela *Editar Regra* para começar a criar sua própria regra no aplicativo. Você pode fazer isso clicando no botão de mais (+) na aba *Minhas Regras*.

### Passo 2. Configure o "Redirecionar De"

Primeiro, você deve configurar a seção *Redirecionar De* na tela *Editar Regra* no aplicativo. Os alvos são URLs que começam com `https://twitter.com/`.

Nesse caso, você pode especificar o seguinte padrão com o modo [Curinga](/rule-settings#url-pattern-wildcard):

```
https://twitter.com/*
```

No modo Curinga, `*` significa que ele corresponde a qualquer coisa (= zero ou mais caracteres).

No entanto, espere um pouco. `twitter.com` foi renomeado para `x.com`. Você pode simplesmente substituir `twitter.com` por `x.com`, mas ninguém sabe quando o novo proprietário mudará de ideia novamente para reverter a mudança. Portanto, vamos direcionar tanto `twitter.com` quanto `x.com`.

Para fazer isso, mude de Curinga para [Expressão Regular](/rule-settings#url-pattern-regular-expression) e defina este como o padrão:

```
https://(twitter|x).com/.*
```

* `(twitter|x)`: direciona tanto `twitter` quanto `x`. (`|` é chamado de *pipe*.)
* `.*`: É o mesmo que o `*` do modo Curinga. Mais especificamente, `.` significa *qualquer caractere* e `*` significa que *o que vier antes deste símbolo pode ser repetido qualquer número de vezes*, resultando em uma correspondência com qualquer coisa.

Expressão Regular é um pouco complicada, mas quando você se acostuma, torna-se uma ferramenta poderosa. Recomendamos usar o [RegExr](https://regexr.com) como um ambiente de testes para analisar como seu padrão de Expressão Regular funciona.

> [!TIP]
> Em Expressões Regulares, o `.` em `(twitter|x).com` também é tratado como *qualquer caractere*. Portanto, `(twitter|x).com/.*` também corresponde a, por exemplo, `twitter1com/` ou `x_com/`.
>
> Para ser mais preciso, você pode alterá-lo para `(twitter|x)\.com/.*`. O `\` é usado para tratar o `.` como um caractere de ponto literal, em vez de um caractere especial.
>
> No entanto, não existem URLs desse tipo no ambiente geral da internet. Portanto, você pode deixar o `.` como um caractere especial, se preferir.

### Passo 3: Configure o "Redirecionar Para"

Basta especificar a URL da seguinte forma:

```
https://unrush.org/
```

### (Opcional) Passo 4: Verifique a regra

Você pode verificar como a regra funciona adicionando um exemplo de URL na seção *Exemplos* na tela *Editar Regra*. Por exemplo, se você inserir `https://x.com/UN`, o resultado mostrado será `https://unrush.org/`.

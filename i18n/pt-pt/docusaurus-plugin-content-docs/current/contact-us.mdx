# Contacte-nos

Estamos aqui para ajudar se tiver alguma questão, feedback ou problemas com o Redirect Web. Eis as melhores formas de entrar em contacto:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Se encontrar algum erro ou quiser solicitar novas funcionalidades, por favor, visite a página de GitHub Issues. Pode procurar problemas existentes, submeter um novo problema e acompanhar o progresso do seu problema.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Se tiver alguma questão ou quiser obter ajuda do desenvolvedor ou de outros utilizadores, por favor, visite a página de GitHub Discussions. Pode iniciar uma nova discussão, juntar-se a discussões existentes e obter ajuda da comunidade.
- **Formulário de Feedback**: Pode enviar-nos feedback através do formulário de feedback na aplicação Redirect Web. Isto enviará um email diretamente para o desenvolvedor, para que o seu feedback seja mantido privado de outros utilizadores.

Obrigado por usar o Redirect Web! Agradecemos o seu apoio e feedback.

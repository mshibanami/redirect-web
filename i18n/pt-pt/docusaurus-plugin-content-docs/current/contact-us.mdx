# Contactos

Estamos aqui para ajudar se tiver alguma questão, feedback ou problemas com o Redirect Web. Aqui estão as melhores formas de entrar em contacto:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Se encontrar algum bug ou quiser solicitar novas funcionalidades, por favor visite a página de GitHub Issues. Pode consultar os problemas existentes, submeter um novo problema e acompanhar o progresso do seu problema.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Se tiver alguma questão ou quiser obter ajuda do programador ou de outros utilizadores, por favor visite a página de GitHub Discussions. Pode iniciar uma nova discussão, juntar-se a discussões existentes e obter ajuda da comunidade.
- **Formulário de Feedback**: Pode enviar-nos feedback através do formulário de feedback na aplicação Redirect Web. Isto enviará um e-mail diretamente ao programador, mantendo o seu feedback privado para outros utilizadores.

Obrigado por usar o Redirect Web! Agradecemos o seu apoio e feedback.

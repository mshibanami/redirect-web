# Exportar/Importar Regras

## Exportar Regras

### Exportar no macOS

Clique no botão de reticências ou no botão Partilhar, conforme mostrado nesta captura de ecrã:

![Share on macOS](/img/share-on-macos.png)

Também pode selecionar múltiplas regras e exportá-las clicando com o botão secundário (direito) sobre elas na barra lateral de listagem de regras:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Adicionalmente, pode exportar todas as regras através da barra de menu > Ficheiro > Exportar Todas as Regras:

![Menubar > File](/img/menubar-file.png)

> [!NOTA]
> Se quiser exportar as suas regras no formato compatível com o [Redirector de Einar Egilsson](https://einaregilsson.com/redirector/), prima a tecla Option (`⌥`) no menu de contexto ou na barra de menu, e então verá o seguinte:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> O Redirect Web não é totalmente compatível com o Redirector, e vice-versa. Algumas definições, como "Substituir Ocorrências" para o Processamento de Grupos de Captura, não são exportadas com o formato do Redirector.
> :::

## Importar Regras

### Importar no macOS

- Basta arrastar e largar os ficheiros de regras exportados para a janela do Redirect Web.
- Ou, também pode importá-los através da barra de menu > Ficheiro > Importar Regras

### Importar no iOS

Suponha que o seu ficheiro de regras está na aplicação Ficheiros. Abra-o nos Ficheiros, toque no botão Partilhar e depois selecione Redirect Web:

| Passo 1: Toque no botão Partilhar nos Ficheiros | Passo 2: Selecione Redirect Web       |
| ----------------------------------------------- | ------------------------------------- |
| ![Alt text](/img/files-share-ios.png)           | ![Alt text](/img/share-sheet-ios.png) |

# Biblioteca

Bem-vindo(a) à nossa biblioteca, onde pode encontrar regras úteis e prontas a usar gratuitamente! Pode aceder-lhes através da aplicação Redirect Web.

![Library Screenshot](/img/library-screenshot.webp)

Para copiar as regras para As Minhas Regras, basta selecionar o botão Obter:

![Get Button - iOS](/img/library-get-button-ios.gif)

## Contribua com a sua regra para a biblioteca

A biblioteca atual tem apenas algumas regras, por isso agradecemos a sua contribuição!

Existem 2 opções para contribuir com as suas regras para a biblioteca:

1. **Publique a sua regra no Fórum**: Partilhe a sua regra de redirecionamento com outros utilizadores no [fórum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Depois, poderemos pedir-lhe se podemos também incluir a sua regra na biblioteca. Isto pode ser mais fácil do que a opção 2, porque pode deixar a adição connosco.
2. **Envie-nos um pull request**: Pode enviar-nos um pull request. Por favor, consulte [as instruções](https://github.com/mshibanami/redirect-web/tree/main/library) para saber mais.

## Outras Informações

- Pode ver todo o conteúdo no [repositório no GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- Alojamos o nosso servidor API privado no Cloudflare para o conteúdo da biblioteca, permitindo-nos fornecer filtragem e outros serviços. A aplicação obtém o conteúdo da biblioteca deste servidor. Conforme indicado na [nossa política de privacidade](/privacy-policy), este servidor não recolhe as suas informações pessoais.

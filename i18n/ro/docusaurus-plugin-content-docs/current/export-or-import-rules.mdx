# Reguli de export/import

## Reguli de export {#export-rules}

### Export pe macOS {#export-on-macos}

Faceți clic fie pe butonul cu elipsă, fie pe butonul Partajare, așa cum se arată în această captură de ecran:

![Share on macOS](/img/share-on-macos.png)

Puteți, de asemenea, să selectați mai multe reguli și să le exportați făcând clic secundar (clic dreapta) pe ele în bara laterală de listare a regulilor:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

În plus, puteți exporta toate regulile din bara de meniu > File > Export All Rules:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Dacă doriți să exportați regulile dumneavoastră în formatul compatibil cu [Redirector de la Einar Egilsson](https://einaregilsson.com/redirector/), apăsați tasta Option (`⌥`) în meniul contextual sau în bara de meniu, apoi veți vedea acestea:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web nu este pe deplin compatibil cu Redirector, și vice versa. Unele setări, cum ar fi "Replace Occurrences" pentru Procesarea Grupurilor de Captură, nu sunt exportate cu formatul Redirector.
> :::

## Reguli de import {#import-rules}

### Import pe macOS {#import-on-macos}

- Pur și simplu trageți și plasați fișierele de reguli exportate în fereastra Redirect Web.
- Sau, le puteți importa și din bara de meniu > File > Import Rules

### Import pe iOS {#import-on-ios}

Să presupunem că fișierul dumneavoastră de reguli se află în aplicația Files. Deschideți-l în Files, atingeți butonul Partajare, apoi selectați Redirect Web:

| Pasul 1: Atingeți butonul Partajare din bara de instrumente                            | Pasul 2: Selectați Redirect Web                                     |
| ------------------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Tap the share button in the toolbar](/img/files-share-ios.png) | ![Tap the app icon in the sheet](/img/share-sheet-ios.png) |

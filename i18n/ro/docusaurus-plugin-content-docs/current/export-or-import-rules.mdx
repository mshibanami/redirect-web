# Export/Import Reguli

## Export Reguli {#export-rules}

### Export pe macOS {#export-on-macos}

Faceți clic pe butonul cu elipsă sau pe butonul Partajare, așa cum se arată în această captură de ecran:

![Partajare pe macOS](/img/share-on-macos.png)

De asemenea, puteți selecta mai multe reguli și le puteți exporta făcând clic secundar (clic dreapta) pe ele în bara laterală de listare a regulilor:

![Partajare mai multe reguli pe macOS](/img/multiselect-on-macos.png)

În plus, puteți exporta toate regulile din bara de meniu > Fișier > Exportă toate regulile:

![Bara de meniu > Fișier](/img/menubar-file.png)

> [!NOTE]
> Dacă doriți să exportați regulile dvs. în formatul compatibil cu [Redirector de la Einar Egilsson](https://einaregilsson.com/redirector/), apăsați tasta Option (`⌥`) din meniul contextual sau din bara de meniu, apoi veți vedea acestea:
> ![Bara de meniu > Fișier (Redirector)](/img/menubar-file-redirector.png)
>
> ![Meniu contextual (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web nu este pe deplin compatibil cu Redirector și invers. Unele setări, cum ar fi „Înlocuire apariții” pentru procesarea grupurilor de captură, nu sunt exportate cu formatul Redirectorului.
> :::

## Import Reguli {#import-rules}

### Import pe macOS {#import-on-macos}

- Pur și simplu trageți și plasați fișierele de reguli exportate în fereastra Redirect Web.
- Sau, le puteți importa și din bara de meniu > Fișier > Importă reguli

### Import pe iOS {#import-on-ios}

Să presupunem că fișierul dvs. de reguli se află în aplicația Fișiere. Deschideți-l în Fișiere, atingeți butonul Partajare, apoi selectați Redirect Web:

| Pasul 1: Atingeți butonul Partajare din bara de instrumente | Pasul 2: Selectați Redirect Web |
| --------------------------------------------------------- | --------------------------------------------------- |
| ![Atingeți butonul de partajare din bara de instrumente](/img/files-share-ios.png) | ![Atingeți pictograma aplicației din foaie](/img/share-sheet-ios.png) |

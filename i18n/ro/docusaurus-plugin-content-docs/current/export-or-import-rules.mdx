# Reguli de export/import

## Reguli de export

### Export pe macOS

Faceți clic fie pe butonul cu elipsă, fie pe butonul Partajare, așa cum este arătat în această captură de ecran:

![Partajare pe macOS](/img/share-on-macos.png)

Puteți, de asemenea, să selectați mai multe reguli și să le exportați făcând clic secundar (clic dreapta) pe ele în bara laterală de listare a regulilor:

![Partajare mai multe reguli pe macOS](/img/multiselect-on-macos.png)

În plus, puteți exporta toate regulile din bara de meniu > Fichier > Export All Rules (Exportați toate regulile):

![Bara de meniu > Fichier](/img/menubar-file.png)

> [!NOTE]
> Dacă doriți să exportați regulile în format compatibil cu [Redirector de Einar Egilsson](https://einaregilsson.com/redirector/), apăsați tasta Option (`⌥`) în meniul contextual sau în bara de meniu, apoi veți vedea următoarele:
> ![Bara de meniu > Fichier (Redirector)](/img/menubar-file-redirector.png)
>
> ![Meniu contextual (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web nu este pe deplin compatibil cu Redirector, și invers. Unele setări, cum ar fi „Replace Occurrences” (Înlocuiește aparițiile) pentru Capturing Group Processing (Procesarea grupurilor de captură), nu sunt exportate cu formatul Redirector.
> :::

## Reguli de import

### Import pe macOS

- Pur și simplu trageți și plasați fișierele de reguli exportate în fereastra Redirect Web.
- Sau, le puteți importa și din bara de meniu > Fichier > Import Rules (Importați reguli)

### Import pe iOS

Presupunem că fișierul dumneavoastră cu reguli se află în aplicația Files. Deschideți-l în Files, atingeți butonul Partajare, apoi selectați Redirect Web:

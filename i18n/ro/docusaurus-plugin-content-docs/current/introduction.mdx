(Cunoscut anterior sub numele de Redirect Web pentru Safari.)

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Magazinul web Chrome)

Redirect Web este disponibil pe:

- **Mac** (Safari, Chrome, Firefox, Edge și altele)
- **iPhone**, **iPad** și **Apple Vision Pro** (Safari)

:::note

- Trebuie să instalați [aplicația Redirect Web din Mac App Store](https://apps.apple.com/app/id1571283503) pentru a utiliza extensia, indiferent de browserul pe care îl folosiți, deoarece extensiile se bazează pe aplicație pentru a accesa regulile, setările etc.
- Suportul multi-browser este disponibil numai pe Mac.

:::

## Cazuri de utilizare populare {#popular-use-cases}

De exemplu, puteți crea aceste reguli:

- Vizualizați **Reddit** în designul său clasic (old.reddit.com).
- Deschideți **Apple Maps** ori de câte ori apare un link **Google Maps**.
- Ascundeți Prezentările AI de la Google din **rezultatele căutării Google** adăugând **?udm=14** la fiecare căutare.
- Treceți de la paginile Fandom la wiki-urile oficiale **Terraria** sau **Minecraft**.
- Utilizați **Kagi**, **Startpage**, **ChatGPT** și orice alt motor de căutare ca **motor de căutare implicit**.
- Deschideți link-urile **Google Meet** în Google Chrome. (Mac)
- Deschideți link-urile **Figma** direct în aplicația Figma. (Mac)
- Deschideți link-urile **Notion** direct în aplicația Notion. (Mac)

Puteți obține toate acestea ca reguli predefinite gratuit din secțiunea Bibliotecă din aplicație!

## Funcționalități {#features}

Redirect Web vine cu următoarele funcționalități:

- Creați reguli folosind tipurile de model **Expresie regulată** sau **Joker**
- Utilizați **referințe inverse** pentru a specifica URL-uri țintă dinamice
- **Excludeți** URL-uri de la redirecționare
- **Procesați textele capturate** folosind codificarea/decodificarea URL, codificarea/decodificarea base64 sau înlocuirea aparițiilor

În plus, puteți debloca următoarele funcționalități printr-un abonament sau o achiziție unică:

- Activarea unui număr **nelimitat** de reguli
- **Sincronizarea regulilor** cu celelalte dispozitive Mac, iPhone și iPad **prin iCloud**

(Achiziționarea aplicației ne sprijină, de asemenea, în menținerea Redirect Web. 🙏)

Achiziția sau abonamentul dvs. este partajat pe Mac, iPhone, iPad și Apple Vision Pro prin Universal Purchase. Suportul dvs. ajută la dezvoltarea continuă.

# Bibliotecă

Bun venit în biblioteca noastră, unde puteți găsi gratuit reguli utile gata de utilizare! Le puteți accesa prin aplicația Redirect Web.

![Captură de ecran Bibliotecă](/img/library-screenshot.webp)

Pentru a copia regulile în Regulile Mele, puteți pur și simplu să selectați butonul Obțineți:

![Buton Obțineți - iOS](/img/library-get-button-ios.gif)

## Contribuiți cu regula dvs. la bibliotecă

Biblioteca actuală are doar câteva reguli, așa că apreciem contribuția dvs.!

Există 2 opțiuni pentru a contribui cu regulile dvs. la bibliotecă:

1. **Postați regula dvs. pe Forum**: Partajați regula dvs. de redirecționare cu alți utilizatori pe [forum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Apoi, vă putem solicita dacă am putea include și regula dvs. în bibliotecă. Acest lucru ar putea fi mai ușor decât opțiunea 2, deoarece ne puteți lăsa nouă adăugarea.
2. **Trimiteți-ne o cerere de extragere**: Ne puteți trimite o cerere de extragere. Vă rugăm să verificați [instrucțiunile](https://github.com/mshibanami/redirect-web/tree/main/library) pentru a afla mai multe.

## Alte informații

- Puteți vedea tot conținutul în [depozitul de pe GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- Găzduim serverul nostru API privat pe Cloudflare pentru conținutul bibliotecii, permițându-ne să oferim filtrare și alte servicii. Aplicația preia conținutul bibliotecii de pe acest server. Așa cum este menționat în [politica noastră de confidențialitate](/privacy-policy.mdx), acest server nu colectează informațiile dvs. personale.

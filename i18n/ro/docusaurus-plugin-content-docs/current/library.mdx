# Bibliotecă

Bine ați venit în biblioteca noastră, unde puteți găsi reguli utile, gata de utilizare, gratuit! Le puteți accesa prin aplicația web Redirect.

![Library Screenshot](/img/library-screenshot.webp)

Pentru a copia regulile în Regulile mele, puteți selecta pur și simplu butonul Obțineți:

![Get Button - iOS](/img/library-get-button-ios.gif)

## Contribuiți cu regula dvs. la bibliotecă

Biblioteca actuală are doar câteva reguli, așa că apreciem contribuția dumneavoastră!

Există 2 opțiuni pentru a contribui cu regulile dvs. la bibliotecă:

1. **Postați-vă regula pe Forum**: Împărtășiți-vă regula de redirecționare cu alți utilizatori pe [forum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Apoi, am putea solicita să includem regula dumneavoastră și în bibliotecă. Aceasta ar putea fi mai ușor decât opțiunea 2, deoarece puteți lăsa adăugarea în sarcina noastră.
2. **Trimiteți-ne un pull request**: Ne puteți trimite un pull request. Vă rugăm să verificați [instrucțiunile](https://github.com/mshibanami/redirect-web/tree/main/library) pentru a afla mai multe.

## Alte informații

- Puteți vedea tot conținutul în [depozitul de pe GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- Găzduim serverul nostru API privat pe Cloudflare pentru conținutul bibliotecii, permițându-ne să oferim filtrare și alte servicii. Aplicația preia conținutul bibliotecii de pe acest server. Așa cum este menționat în [politica noastră de confidențialitate](privacy-policy.mdx), acest server nu colectează informațiile dumneavoastră personale.

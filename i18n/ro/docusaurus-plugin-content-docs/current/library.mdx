# Biblioteca

Bun venit în biblioteca noastră, unde puteți găsi gratuit reguli utile, gata de utilizare! Le puteți accesa prin aplicația web Redirect.

![Library Screenshot](/img/library-screenshot.webp)

Pentru a copia regulile în Regulile mele, puteți pur și simplu să selectați butonul Obțineți:

![Get Button - iOS](/img/library-get-button-ios.gif)

## Contribuiți cu regula dumneavoastră la bibliotecă

Biblioteca actuală are doar câteva reguli, așa că apreciem contribuția dumneavoastră!

Există 2 opțiuni pentru a contribui cu regulile dumneavoastră la bibliotecă:

1. **Postați regula dumneavoastră pe Forum**: Partajați regula dumneavoastră de redirecționare cu alți utilizatori pe [forum](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Apoi, am putea solicita dacă am putea include și regula dumneavoastră în bibliotecă. Acest lucru ar putea fi mai ușor decât opțiunea 2, deoarece ne puteți lăsa pe noi să facem adăugarea.
2. **Trimiteți-ne o cerere de pull**: Ne puteți trimite o cerere de pull. Vă rugăm să verificați [instrucțiunile](https://github.com/mshibanami/redirect-web/tree/main/library) pentru a afla mai multe.

## Alte informații

- Puteți vedea tot conținutul în [repository-ul de pe GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- Găzduim serverul nostru API privat pe Cloudflare pentru conținutul bibliotecii, permițându-ne să oferim filtrare și alte servicii. Aplicația preia conținutul bibliotecii de pe acest server. Așa cum este menționat în [politica noastră de confidențialitate](/privacy-policy), acest server nu colectează informațiile dumneavoastră personale.

# Экспорт/Импорт правил

## Экспорт правил {#export-rules}

### Экспорт на macOS {#export-on-macos}

Нажмите кнопку с многоточием или кнопку «Поделиться», как показано на этом снимке экрана:

![Поделиться на macOS](/img/share-on-macos.png)

Вы также можете выбрать несколько правил и экспортировать их, щелкнув правой кнопкой мыши (вторичный щелчок) по ним на боковой панели списка правил:

![Поделиться несколькими правилами на macOS](/img/multiselect-on-macos.png)

Кроме того, вы можете экспортировать все правила из строки меню > Файл > Экспортировать все правила:

![Строка меню > Файл](/img/menubar-file.png)

> [!NOTE]
> Если вы хотите экспортировать свои правила в формате, совместимом с [Redirector Эйнара Эгилссона](https://einaregilsson.com/redirector/), нажмите клавишу Option (`⌥`) в контекстном меню или строке меню, тогда вы увидите следующее:
> ![Строка меню > Файл (Redirector)](/img/menubar-file-redirector.png)
>
> ![Контекстное меню (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web не полностью совместим с Redirector, и наоборот. Некоторые настройки, такие как «Заменить вхождения» для обработки групп захвата, не экспортируются в формате Redirector.
> :::

## Импорт правил {#import-rules}

### Импорт на macOS {#import-on-macos}

- Просто перетащите экспортированные файлы правил в окно Redirect Web.
- Или вы также можете импортировать их из строки меню > Файл > Импортировать правила

### Импорт на iOS {#import-on-ios}

Предположим, ваш файл правил находится в приложении «Файлы». Откройте его в «Файлах», нажмите кнопку «Поделиться», а затем выберите Redirect Web:

| Шаг 1: Нажмите кнопку «Поделиться» на панели инструментов | Шаг 2: Выберите Redirect Web |
| --------------------------------------------------------- | --------------------------------------------------- |
| ![Нажмите кнопку «Поделиться» на панели инструментов](/img/files-share-ios.png) | ![Нажмите значок приложения в листе](/img/share-sheet-ios.png) |

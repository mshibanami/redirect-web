# Экспорт/Импорт правил

## Экспорт правил

### Экспорт на macOS

Нажмите кнопку с многоточием или кнопку «Поделиться», как показано на этом снимке экрана:

![Share on macOS](/img/share-on-macos.png)

Вы также можете выбрать несколько правил и экспортировать их, нажав вторичной кнопкой (правой кнопкой мыши) на них в боковой панели со списком правил:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Кроме того, вы можете экспортировать все правила из строки меню > Файл > Экспортировать все правила:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Если вы хотите экспортировать свои правила в формате, совместимом с [Redirector от Эйнара Эгилссона](https://einaregilsson.com/redirector/), нажмите клавишу Option (`⌥`) в контекстном меню или строке меню, после чего вы увидите следующее:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web не полностью совместим с Redirector, и наоборот. Некоторые настройки, такие как «Заменить вхождения» для обработки захваченных групп, не экспортируются в формате Redirector.
> :::

## Импорт правил

### Импорт на macOS

- Просто перетащите экспортированные файлы правил в окно Redirect Web.
- Или вы также можете импортировать их из строки меню > Файл > Импортировать правила.

### Импорт на iOS

Предположим, ваш файл правил находится в приложении «Файлы». Откройте его в «Файлах», нажмите кнопку «Поделиться», а затем выберите Redirect Web:

| Шаг 1: Нажмите кнопку «Поделиться» в «Файлах» | Шаг 2: Выберите Redirect Web |

# Библиотека

Добро пожаловать в нашу библиотеку, где вы можете бесплатно найти полезные готовые правила! Вы можете получить к ним доступ через веб-приложение Redirect.

![Get Button - iOS](/img/library-get-button-ios.gif)

## Добавьте свое правило в библиотеку

В текущей библиотеке всего несколько правил, поэтому мы будем рады вашему вкладу!

Есть 2 варианта добавить свои правила в библиотеку:

1.  **Опубликуйте свое правило на форуме**: Поделитесь своим правилом перенаправления с другими пользователями на [форуме](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Затем мы можем запросить разрешение на добавление вашего правила в библиотеку. Это может быть проще, чем вариант 2, так как вы можете оставить добавление нам.
2.  **Отправьте нам pull request**: Вы можете отправить нам pull request. Пожалуйста, ознакомьтесь с [инструкциями](https://github.com/mshibanami/redirect-web/tree/main/library), чтобы узнать больше.

## Другая информация

-   Вы можете увидеть всё содержимое в [репозитории на GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
-   Мы размещаем наш приватный API-сервер на Cloudflare для содержимого библиотеки, что позволяет нам предоставлять фильтрацию и другие услуги. Приложение получает содержимое библиотеки с этого сервера. Как указано в [нашей политике конфиденциальности](/privacy-policy), этот сервер не собирает вашу личную информацию.

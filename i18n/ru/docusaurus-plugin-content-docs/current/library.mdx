# Библиотека

Добро пожаловать в нашу библиотеку, где вы можете найти полезные готовые правила бесплатно! Вы можете получить к ним доступ через приложение Redirect Web.

![Кнопка «Получить» - iOS](/img/library-get-button-ios.gif)

## Добавьте свое правило в библиотеку

В текущей библиотеке пока мало правил, поэтому мы ценим ваш вклад!

Есть 2 варианта добавить свои правила в библиотеку:

1. **Опубликуйте свое правило на Форуме**: Поделитесь своим правилом перенаправления с другими пользователями на [форуме](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Затем мы можем спросить, можем ли мы добавить ваше правило и в библиотеку. Это может быть проще, чем вариант 2, так как вы можете оставить добавление нам.
2. **Отправьте нам pull request**: Вы можете отправить нам pull request. Пожалуйста, ознакомьтесь с [инструкциями](https://github.com/mshibanami/redirect-web/tree/main/library), чтобы узнать больше.

## Прочая информация

- Вы можете увидеть все содержимое в [репозитории на GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- Мы размещаем наш частный API-сервер на Cloudflare для содержимого библиотеки, что позволяет нам предоставлять фильтрацию и другие услуги. Приложение получает содержимое библиотеки с этого сервера. Как указано в [нашей политике конфиденциальности](privacy-policy.mdx), этот сервер не собирает вашу личную информацию.

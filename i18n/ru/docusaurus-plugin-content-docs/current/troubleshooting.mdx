## Правила перенаправления не работают

Если ваши правила не работают должным образом, проверьте следующее:

- **Расширение браузера установлено и включено для правильной страницы:** Убедитесь, что вы установили расширение браузера Redirect Web и что оно включено в вашем браузере для веб-страницы, с которой вы хотите перенаправить. Если вы можете открыть всплывающее окно расширения, как показано ниже, на исходной веб-странице, значит, оно установлено правильно:
    <Image src={ExtensionPopup} alt="Всплывающее окно расширения Redirect Web" width="260" hasShadow/>

- **Конфигурация правила верна:** Внимательно проверьте свои правила перенаправления в нативном приложении Redirect Web. Убедитесь, что шаблоны исходных URL-адресов, целевые URL-адреса и любые условия настроены правильно. Чтобы проверить, является ли ваше правило действительным, введите пример URL-адреса в разделе «Примеры» на экране редактирования правила. Если ваше правило действительно, вы увидите ожидаемый URL-адрес назначения под вашим вводом, как показано ниже:
    <Image src={RuleEditExampleResult} alt="Пример экрана редактирования правила, показывающий результат примера URL-адреса" width="260" hasShadow/>

## Всплывающее окно расширения показывает ошибку

<Image src={ErrorOverlay} alt="Наложение ошибки Redirect Web" width="260" hasShadow hasRoundedCorners/>

> Приложение Redirect Web не установлено или работает неправильно.
> Пожалуйста, загрузите приложение из App Store.

Если вы видите вышеуказанную ошибку при открытии всплывающего окна расширения, это означает, что нативное приложение Redirect Web не установлено или работает неправильно.

Пожалуйста, убедитесь, что вы установили последнюю версию Redirect Web из [App Store](https://apps.apple.com/app/id1571283503).

Вам также может потребоваться перезапустить браузер после установки нативного приложения.

> [!NOTE]
> * Вам необходимо установить нативное приложение Redirect Web, чтобы использовать расширение браузера. Само по себе расширение не работает.
> * Пользователям Safari не нужно устанавливать расширение отдельно, так как оно поставляется в комплекте с нативным приложением.

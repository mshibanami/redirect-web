# Knižnica

Vitajte v našej knižnici, kde nájdete užitočné, pripravené na použitie pravidlá zadarmo! Môžete k nim pristupovať prostredníctvom aplikácie Redirect Web.

![Library Screenshot](/img/library-screenshot.webp)

Ak chcete skopírovať pravidlá do Moje pravidlá, jednoducho vyberte tlačidlo Získať:

![Get Button - iOS](/img/library-get-button-ios.gif)

## Prispejte svojím pravidlom do knižnice

Aktuálna knižnica má len niekoľko pravidiel, preto si vážime váš príspevok!

Existujú 2 možnosti, ako prispieť svojimi pravidlami do knižnice:

1. **Uverejnite svoje pravidlo na Fóre**: Zdieľajte svoje pravidlo presmerovania s ostatnými používateľmi na [fóre](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Potom vás môžeme požiadať, či by sme mohli vaše pravidlo pridať aj do knižnice. Toto môže byť jednoduchšie ako možnosť 2, pretože pridanie môžete nechať na nás.
2. **Pošlite nám pull request**: Môžete nám poslať pull request. Pozrite si [pokyny](https://github.com/mshibanami/redirect-web/tree/main/library), aby ste sa dozvedeli viac.

## Ďalšie informácie

- Celý obsah si môžete pozrieť v [úložisku na GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- Náš súkromný API server pre obsah knižnice hostíme na Cloudflare, čo nám umožňuje poskytovať filtrovanie a ďalšie služby. Aplikácia načítava obsah knižnice z tohto servera. Ako je uvedené v [našich zásadách ochrany osobných údajov](privacy-policy.mdx), tento server nezhromažďuje vaše osobné údaje.

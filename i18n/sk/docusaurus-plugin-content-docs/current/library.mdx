# Knižnica

Vitajte v našej knižnici, kde nájdete užitočné, hotové pravidlá zadarmo! Môžete k nim pristupovať prostredníctvom aplikácie Redirect Web.

![Snímka obrazovky knižnice](/img/library-screenshot.webp)

Ak chcete skopírovať pravidlá do Mojich pravidiel, stačí vybrať tlačidlo Získať:

![Tlačidlo Získať - iOS](/img/library-get-button-ios.gif)

## Prispejte svojím pravidlom do knižnice

Súčasná knižnica má len niekoľko pravidiel, takže si vážime váš príspevok!

Existujú 2 možnosti, ako prispieť svojimi pravidlami do knižnice:

1. **Uverejnite svoje pravidlo na fóre**: Zdieľajte svoje pravidlo presmerovania s ostatnými používateľmi na [fóre](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Potom môžeme požiadať, či by sme mohli vaše pravidlo zaradiť aj do knižnice. To môže byť jednoduchšie ako možnosť 2, pretože pridanie môžete nechať na nás.
2. **Pošlite nám žiadosť o stiahnutie (pull request)**: Môžete nám poslať žiadosť o stiahnutie. Ak sa chcete dozvedieť viac, pozrite si [pokyny](https://github.com/mshibanami/redirect-web/tree/main/library).

## Ďalšie informácie

- Všetok obsah si môžete pozrieť v [úložisku na GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- Náš súkromný API server pre obsah knižnice hostíme na Cloudflare, čo nám umožňuje poskytovať filtrovanie a ďalšie služby. Aplikácia načítava obsah knižnice z tohto servera. Ako je uvedené v [našich zásadách ochrany osobných údajov](privacy-policy.mdx), tento server nezhromažďuje vaše osobné údaje.

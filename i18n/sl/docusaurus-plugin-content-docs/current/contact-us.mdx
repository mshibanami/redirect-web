# Kontaktirajte nas

Če imate kakršnakoli vprašanja, povratne informacije ali težave z Redirect Web, smo tukaj, da vam pomagamo. Tukaj so najboljši načini za stik z nami:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Če naletite na kakršne koli napake ali želite zahtevati nove funkcije, obiščite stran GitHub Issues. Lahko brskate po obstoječih težavah, oddate novo težavo in spremljate napredek svoje težave.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Če imate kakršnakoli vprašanja ali želite dobiti pomoč od razvijalca ali drugih uporabnikov, obiščite stran GitHub Discussions. Lahko začnete novo razpravo, se pridružite obstoječim razpravam in dobite pomoč od skupnosti.
- **Obrazec za povratne informacije**: Povratne informacije nam lahko pošljete preko obrazca za povratne informacije v aplikaciji Redirect Web. To bo poslalo e-pošto neposredno razvijalcu, tako da bodo vaše povratne informacije ostale zasebne pred drugimi uporabniki.

Hvala, ker uporabljate Redirect Web! Cenimo vašo podporo in povratne informacije.

# Kontaktirajte nas

Tukaj smo, da vam pomagamo, če imate kakršnakoli vprašanja, povratne informacije ali težave z Redirect Web. Tukaj so najboljši načini za stik z nami:

- **[GitHub Težave](https://github.com/mshibanami/RedirectWeb/issues)**: Če naletite na kakršnekoli napake ali želite zahtevati nove funkcije, obiščite stran GitHub Težave. Lahko brskate po obstoječih težavah, oddate novo težavo in spremljate napredek svoje težave.
- **[GitHub Razprave](https://github.com/mshibanami/RedirectWeb/discussions)**: Če imate kakršnakoli vprašanja ali želite dobiti pomoč od razvijalca ali drugih uporabnikov, obiščite stran GitHub Razprave. Lahko začnete novo razpravo, se pridružite obstoječim razpravam in dobite pomoč skupnosti.
- **Obrazec za povratne informacije**: Povratne informacije nam lahko pošljete preko obrazca za povratne informacije v aplikaciji Redirect Web. To bo poslalo e-pošto neposredno razvijalcu, tako da bodo vaše povratne informacije ostale zasebne in nevidne drugim uporabnikom.

Hvala, ker uporabljate Redirect Web! Cenimo vašo podporo in povratne informacije.

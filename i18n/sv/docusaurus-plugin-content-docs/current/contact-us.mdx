# Kontakta oss

Vi finns här för att hjälpa dig om du har några frågor, feedback eller problem med Redirect Web. Här är de bästa sätten att komma i kontakt:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Om du stöter på några buggar eller vill begära nya funktioner, besök GitHub Issues-sidan. Du kan bläddra bland befintliga problem, skicka in ett nytt problem och följa framstegen för ditt problem.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Om du har några frågor eller vill få hjälp från utvecklaren eller andra användare, besök GitHub Discussions-sidan. Du kan starta en ny diskussion, gå med i befintliga diskussioner och få hjälp från communityn.
- **Feedbackformulär**: Du kan skicka oss feedback via feedbackformuläret i Redirect Web-appen. Detta skickar ett e-postmeddelande direkt till utvecklaren, så din feedback hålls privat från andra användare.

Tack för att du använder Redirect Web! Vi uppskattar ditt stöd och din feedback.

# Kontakta oss

Vi är här för att hjälpa dig om du har några frågor, feedback eller problem med Redirect Web. Här är de bästa sätten att komma i kontakt:

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**: Om du stöter på några buggar eller vill begära nya funktioner, vänligen besök sidan för GitHub Issues. Du kan bläddra bland befintliga ärenden, skicka in ett nytt ärende och följa dess framsteg.
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**: Om du har några frågor eller vill få hjälp från utvecklaren eller andra användare, vänligen besök sidan för GitHub Discussions. Du kan starta en ny diskussion, delta i befintliga diskussioner och få hjälp från gemenskapen.
- **Feedbackformulär**: Du kan skicka oss feedback via feedbackformuläret i appen Redirect Web. Detta skickar ett e-postmeddelande direkt till utvecklaren, så din feedback hålls privat från andra användare.

Tack för att du använder Redirect Web! Vi uppskattar ditt stöd och din feedback.

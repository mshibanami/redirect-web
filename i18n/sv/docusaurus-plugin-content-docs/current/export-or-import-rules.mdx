# Exportera/Importera Regler

## Exportera Regler {#export-rules}

### Exportera på macOS {#export-on-macos}

Klicka antingen på ellips-knappen eller Dela-knappen som visas i denna skärmdump:

<Image src={ShareOnMacOS} alt="Dela-knapp på macOS" width="460" hasShadow hasRoundedCorners/>

Välj sedan "Exportera regel" eller "Dela regel..." från snabbmenyn:

Du kan också markera flera regler och exportera dem genom att sekundärklicka (högerklicka) på dem i sidofältet för regellistan:

<Image src={MultiSelectOnMacOS} alt="Dela flera regler på macOS" height="400" hasShadow hasRoundedCorners/>


Dessutom kan du exportera alla regler från menyraden > Arkiv > Exportera alla regler:

<Image src={MenubarFile} alt="Menyrad > Arkiv" width="400" hasShadow hasRoundedCorners/>

> [!NOTE]
> Om du vill exportera dina regler i ett format som är kompatibelt med [Einar Egilssons Redirector](https://einaregilsson.com/redirector/), tryck på Option-tangenten (`⌥`) i snabbmenyn eller menyraden, då ser du dessa:
> <Image src={MenubarFileRedirector} alt="Menyrad > Arkiv (Redirector)" width="400" hasShadow hasRoundedCorners/>
> <br/>
> <Image src={ContextMenuRedirector} alt="Snabbmeny (Redirector)" width="400" hasShadow hasRoundedCorners/>
>
> :::warning
> RedirectWeb är inte helt kompatibel med Redirector, och vice versa. Vissa inställningar, som "Ersätt förekomster" för bearbetning av fångstgrupper, exporteras inte med Redirectors format.
> :::

## Importera Regler {#import-rules}

### Importera på macOS {#import-on-macos}

- Dra och släpp helt enkelt de exporterade regelfilerna till RedirectWebs fönster.
- Eller så kan du också importera dem från menyraden > Arkiv > Importera regler

### Importera på iOS {#import-on-ios}

Anta att din regelfil finns i appen Filer. Öppna den i Filer, tryck på Dela-knappen och välj sedan RedirectWeb:

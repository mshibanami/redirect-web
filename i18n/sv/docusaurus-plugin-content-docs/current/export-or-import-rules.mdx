# Exportera/Importera regler

## Exportera regler

### Exportera på macOS

Klicka antingen på ellipsknappen eller delningsknappen som visas i denna skärmdump:

![Share on macOS](/img/share-on-macos.png)

Du kan också markera flera regler och exportera dem genom att högerklicka på dem i sidofältet med regler:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Dessutom kan du exportera alla regler från menyraden > Arkiv > Exportera alla regler:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Om du vill exportera dina regler i det format som är kompatibelt med [Einar Egilssons Redirector](https://einaregilsson.com/redirector/), tryck på tangenten Option (`⌥`) på snabbmenyn eller menyraden, då ser du följande:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web är inte helt kompatibel med Redirector, och vice versa. Vissa inställningar, såsom "Ersätt förekomster" för bearbetning av fångstgrupper, exporteras inte i Redirectors format.
> :::

## Importera regler

### Importera på macOS

- Dra och släpp helt enkelt de exporterade regelfilerna till Redirect Webs fönster.
- Eller så kan du importera dem från menyraden > Arkiv > Importera regler

### Importera på iOS

Anta att din regelfil finns i appen Filer. Öppna den i Filer, tryck på delningsknappen och välj sedan Redirect Web:

| Steg 1: Tryck på delningsknappen i Filer | Steg 2: Välj Redirect Web            |
| -------------------------------------- | ------------------------------------ |

# Exportera/Importera Regler

## Exportera Regler {#export-rules}

### Exportera på macOS {#export-on-macos}

Klicka antingen på ellips-knappen eller Dela-knappen som visas i denna skärmdump:

![Share on macOS](/img/share-on-macos.png)

Du kan också markera flera regler och exportera dem genom att sekundärklicka (högerklicka) på dem i sidofältet för regellistan:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Dessutom kan du exportera alla regler från menyraden > Arkiv > Exportera alla regler:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Om du vill exportera dina regler i ett format som är kompatibelt med [Einar Egilssons Redirector](https://einaregilsson.com/redirector/), tryck på Option-tangenten (`⌥`) i snabbmenyn eller menyraden, så ser du dessa:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web är inte helt kompatibel med Redirector, och vice versa. Vissa inställningar, som "Ersätt förekomster" för bearbetning av fångstgrupper, exporteras inte med Redirectors format.
> :::

## Importera Regler {#import-rules}

### Importera på macOS {#import-on-macos}

- Dra och släpp helt enkelt de exporterade regelfilerna till Redirect Webs fönster.
- Eller så kan du också importera dem från menyraden > Arkiv > Importera regler

### Importera på iOS {#import-on-ios}

Anta att din regelfil finns i appen Filer. Öppna den i Filer, tryck på Dela-knappen och välj sedan Redirect Web:

| Steg 1: Tryck på Dela-knappen i Filer | Steg 2: Välj Redirect Web |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Tap the share button in the toolbar](/img/files-share-ios.png) | ![Tap the app icon in the sheet](/img/share-sheet-ios.png) |

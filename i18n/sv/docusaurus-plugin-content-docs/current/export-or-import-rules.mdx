# Exportera/Importera regler

## Exportera regler {#export-rules}

### Exportera på macOS {#export-on-macos}

Klicka antingen på ellips-knappen eller dela-knappen som visas i denna skärmdump:

![Dela på macOS](/img/share-on-macos.png)

Du kan också markera flera regler och exportera dem genom att sekundärklicka (högerklicka) på dem i sidofältet för regellistan:

![Dela flera regler på macOS](/img/multiselect-on-macos.png)

Dessutom kan du exportera alla regler från menyraden > Arkiv > Exportera alla regler:

![Menyrad > Arkiv](/img/menubar-file.png)

> [!OBS]
> Om du vill exportera dina regler i ett format som är kompatibelt med [Einar Egilssons Redirector](https://einaregilsson.com/redirector/), tryck på Option-tangenten (`⌥`) i snabbmenyn eller menyraden, då kommer du att se följande:
> ![Menyrad > Arkiv (Redirector)](/img/menubar-file-redirector.png)
>
> ![Snabbmeny (Redirector)](/img/context-menu-redirector.png)
>
> :::varning
> Redirect Web är inte helt kompatibel med Redirector, och vice versa. Vissa inställningar, såsom "Ersätt förekomster" för hantering av infångningsgrupper, exporteras inte med Redirectors format.
> :::

## Importera regler {#import-rules}

### Importera på macOS {#import-on-macos}

- Dra och släpp helt enkelt de exporterade regelfilerna till Redirect Webs fönster.
- Eller så kan du importera dem från menyraden > Arkiv > Importera regler

### Importera på iOS {#import-on-ios}

Anta att din regelfil finns i appen Filer. Öppna den i Filer, tryck på Dela-knappen och välj sedan Redirect Web:

| Steg 1: Tryck på Dela-knappen i Filer                                     | Steg 2: Välj Redirect Web                                         |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Tryck på dela-knappen i verktygsfältet](/img/files-share-ios.png) | ![Tryck på appikonen i menyn](/img/share-sheet-ios.png) |

# Vanliga frågor

## Varför ska jag tillåta tillägget att läsa/ändra min information?

Du kan se en varning som denna när du konfigurerar Redirect Web:

<IfBrowserStore stores='appStore'>
    <Image src={PermissionAlertSafari} alt="An permission requiring alert" width="200" margin="0 0 16px 0" hasShadow={true} />

    > Detta tillägg kommer att kunna läsa och ändra webbsidor samt se din webbhistorik på denna webbplats. **Detta kan inkludera känslig information, inklusive lösenord, telefonnummer och kreditkort.**
</IfBrowserStore>

<IfBrowserStore stores='appStore' includes={false}>
    <Image src={PermissionAlertChrome} alt="An permission requiring alert" height="200" hasRoundedCorners={true} hasShadow={true} margin="0 0 16px 0" />

    > Det kan:  
    > Läsa och ändra **all din data** på **alla webbplatser**
</IfBrowserStore>

Detta tillägg behöver ditt tillstånd för att observera sidnavigeringen för att kunna omdirigera dig dit du vill hoppa, och din webbläsare visar detta när tillägget begär de relaterade behörigheterna. Därför, om du inte tillåter det, kan detta tillägg inte fungera.

Beviljade behörigheter kommer endast att användas för appfunktioner som omdirigering, i enlighet med **[integritetspolicyn](./privacy-policy)**.

## Ska jag fortfarande skaffa appen efter att ha installerat tillägget för Chrome, Firefox eller Edge? \{#app-for-non-safari-browsers}

Ja. Du måste ladda ner [Redirect Web-appen från App Store](https://apps.apple.com/app/id1571283503) oavsett vilken webbläsare du använder. Du kan installera tillägget från din webbläsares tilläggsbutik, men appen krävs för att hantera dina regler, köphistorik och andra inställningar över enheter via iCloud.

## Fungerar Redirect Web på Windows, Android, Linux eller andra plattformar? \{#non-apple-platforms}

Inte för närvarande. Redirect Web är nära kopplad till Apples ekosystem, inklusive iCloud och Universal Purchase, vilket gör det svårt att erbjuda på andra plattformar.

## Varför inte [Redirector](https://github.com/einaregilsson/Redirector)?

Redirector är fortfarande ett utmärkt omdirigeringstillägg. Det är helt gratis, öppen källkod och fungerar även på plattformar som inte är Apple. Men Redirect Web har några extra funktioner som du kanske gillar:

*   **Detaljerade regelinställningar:** Flera exkluderingsmönster, flera matchningar för processer och funktioner för strängersättning finns endast i Redirect Web.
*   **Öppna i inbyggda appar:** På Mac kan du välja var målets URL ska öppnas. (Den anropar `open`-kommandot direkt. Inte djuplänkar.)
*   **[DNR](https://blog.mozilla.org/addons/2023/05/17/declarativenetrequest-available-in-firefox/) stöd:** Ger blixtsnabb omdirigering.
*   **Centraliserad regelhantering:** Hantera regler för alla webbläsare på ett ställe via den inbyggda appen.
*   **Manifest V3 & V2-stöd:** Användbart medan V2 fasas ut i Chrome.

## Varför kräver detta tillägg en ny behörighet? \{#permission-alert-on-safari}

Du kan se denna varning i Safari efter att ha uppdaterat Redirect Web:

<img src={require("/img/safari-additional-permission-alert.webp").default} alt="An alert of disabling the app" width="280" />

> "Redirect Web" har uppdaterats och begär ytterligare behörigheter. För att skydda din integritet har detta tillägg stängts av.
>
> Du kan aktivera detta tillägg i avsnittet Tillägg i Safaris inställningar.

Detta beror på behörigheten [declarativeNetRequestWithHostAccess](https://developer.apple.com/documentation/safariservices/safari_web_extensions/blocking_content_with_your_safari_web_extension) som krävs av [DNR](./rule-settings#type)-typen som introducerades i version 7. Vänligen återaktivera tillägget. Vi ber om ursäkt för besväret.

## Engångsköp eller årsabonnemang: Vad är bäst?

Vi rekommenderar engångsköpet, då prisskillnaden endast är 2,00 US-dollar.
Huvudanledningen till att vi erbjuder prenumerationen är att du ska kunna prova alla funktioner via dess kostnadsfria provperiod. Här är det förväntade användningsfallet:

1.  Prenumerera på årsabonnemanget.
2.  Avbryt omedelbart prenumerationen genom att gå till appen Inställningar, söka efter "Prenumerationer", trycka på denna app och välja "Avbryt prenumeration". Detta låter dig njuta av den kostnadsfria provperioden utan några åtaganden.
3.  Njut av full tillgång till alla funktioner i 7 dagar.
4.  Om du gillar funktionerna, välj engångsköpet.

Naturligtvis är det inget fel med att fortsätta med årsabonnemanget om du föredrar det!

## Injicerar Redirect Web annonser eller annat orelaterat innehåll på webbplatser?

Nej, det injicerar inget sådant innehåll. Endast kod som är nödvändig för att tillhandahålla de funktioner som beskrivs i denna dokumentation och på appsidan i App Store injiceras på webbplatser i Safari.

## Finns det någon chans att Redirect Web skulle kunna säljas till någon annan?

100 % nej. Webbläsartillägg har djupgående kontroll över din webbläsning och data, och att lämna över Redirect Web till en annan ägare skulle öppna dörren för integritetskränkningar. Vi har sett tillägg byta ägare och sedan samla in användardata eller injicera annonser utan förvarning. Det kommer aldrig att hända här. För mer om dessa risker, se Matt Frisbies artikel ["The Ugly Business of Monetizing Browser"](https://mattfrisbie.substack.com/p/the-ugly-business-of-monetizing-browser) och Brian Krebs utredning ["Is Your Browser Extension a Botnet Backdoor?"](https://krebsonsecurity.com/2021/03/is-your-browser-extension-a-botnet-backdoor/).

Redirect Web bygger på förtroende och ansvar. Vi anser det vara vår plikt att hålla din information säker, och vi vägrar att överföra äganderätten under några omständigheter. Redirect Web kommer alltid att förbli i våra händer, så du kan surfa med fullständigt förtroende.

# Bibliotek

Välkommen till vårt bibliotek, där du kan hitta användbara färdiga regler gratis! Du kan komma åt dem via Redirect Web-appen.

![Biblioteksskärmdump](/img/library-screenshot.webp)

För att kopiera reglerna till Mina regler kan du helt enkelt välja knappen Hämta:

![Hämta-knapp - iOS](/img/library-get-button-ios.gif)

## Bidra med din regel till biblioteket

Det nuvarande biblioteket har bara ett fåtal regler, så vi uppskattar ditt bidrag!

Det finns 2 alternativ för att bidra med dina regler till biblioteket:

1. **Posta din regel på forumet**: Dela din omdirigeringsregel med andra användare på [forumet](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules). Då kan vi fråga om vi också kan lägga in din regel i biblioteket. Detta kan vara enklare än alternativ 2 eftersom du kan överlåta tillägget till oss.
2. **Skicka oss en pull request**: Du kan skicka oss en pull request. Vänligen kolla [instruktionerna](https://github.com/mshibanami/redirect-web/tree/main/library) för att lära dig mer.

## Annan information

- Du kan se allt innehåll i [arkivet på GitHub](https://github.com/mshibanami/redirect-web/tree/main/library).
- Vi hostar vår privata API-server på Cloudflare för biblioteksinnehållet, vilket gör att vi kan tillhandahålla filtrering och andra tjänster. Appen hämtar biblioteksinnehållet från denna server. Som anges i [vår integritetspolicy](/privacy-policy.mdx) samlar denna server inte in din personliga information.

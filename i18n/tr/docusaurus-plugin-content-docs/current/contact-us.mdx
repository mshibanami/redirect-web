# Bize Ulaşın

Redirect Web ile ilgili herhangi bir sorunuz, geri bildiriminiz veya sorununuz varsa size yardımcı olmak için buradayız. İşte iletişime geçmenin en iyi yolları:

- **[GitHub Sorunları](https://github.com/mshibanami/RedirectWeb/issues)**: Herhangi bir hata ile karşılaşırsanız veya yeni özellikler talep etmek isterseniz, lütfen GitHub Sorunları sayfasını ziyaret edin. Mevcut sorunlara göz atabilir, yeni bir sorun gönderebilir ve sorununuzun ilerlemesini takip edebilirsiniz.
- **[GitHub Tartışmaları](https://github.com/mshibanami/RedirectWeb/discussions)**: Herhangi bir sorunuz varsa veya geliştiriciden veya diğer kullanıcılardan yardım almak isterseniz, lütfen GitHub Tartışmaları sayfasını ziyaret edin. Yeni bir tartışma başlatabilir, mevcut tartışmalara katılabilir ve topluluktan yardım alabilirsiniz.
- **Geri Bildirim Formu**: Redirect Web uygulamasındaki geri bildirim formu aracılığıyla bize geri bildirim gönderebilirsiniz. Bu, doğrudan geliştiriciye bir e-posta gönderecektir, böylece geri bildiriminiz diğer kullanıcılardan gizli kalır.

Redirect Web kullandığınız için teşekkür ederiz! Desteğiniz ve geri bildiriminiz için minnettarız.

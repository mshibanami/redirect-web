# Kuralları Dışa/İçe Aktar

## Kuralları Dışa Aktar {#export-rules}

### macOS'te Dışa Aktarma {#export-on-macos}

Bu ekran görüntüsünde gösterildiği gibi üç nokta düğmesine veya Paylaş düğmesine tıklayın:

![Share on macOS](/img/share-on-macos.png)

Kurallarınızı çoklu seçebilir ve kural listeleme kenar çubuğunda ikincil tıklayarak (sağ tıklayarak) dışa aktarabilirsiniz:

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

Ek olarak, tüm kuralları menü çubuğundan > Dosya > Tüm Kuralları Dışa Aktar seçeneğiyle dışa aktarabilirsiniz:

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> Kurallarınızı [Einar Egilsson'ın Redirector'ü](https://einaregilsson.com/redirector/) ile uyumlu formatta dışa aktarmak isterseniz, bağlam menüsünde veya menü çubuğunda Option tuşuna (`⌥`) basın, ardından şunları göreceksiniz:
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web, Redirector ile tam uyumlu değildir ve bunun tersi de geçerlidir. Yakalama Grubu İşleme için "Geçenleri Değiştir" gibi bazı ayarlar, Redirector'ün formatıyla dışa aktarılmaz.
> :::

## Kuralları İçe Aktar {#import-rules}

### macOS'te İçe Aktarma {#import-on-macos}

- Dışa aktarılan kural dosyalarını Redirect Web penceresine sürükleyip bırakmanız yeterlidir.
- Veya, menü çubuğundan > Dosya > Kuralları İçe Aktar seçeneğiyle de içe aktarabilirsiniz.

### iOS'te İçe Aktarma {#import-on-ios}

Kural dosyanızın Dosyalar uygulamasında olduğunu varsayalım. Dosyalar uygulamasında açın, Paylaş düğmesine dokunun ve ardından Redirect Web'i seçin:

| Adım 1: Dosyalar uygulamasında Paylaş düğmesine dokunun                          | Adım 2: Redirect Web'i seçin                                        |
| ------------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Tap the share button in the toolbar](/img/files-share-ios.png) | ![Tap the app icon in the sheet](/img/share-sheet-ios.png) |

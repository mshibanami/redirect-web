# Kuralları Dışa/İçe Aktarma

## Kuralları Dışa Aktarma {#export-rules}

### macOS'ta Dışa Aktarma {#export-on-macos}

Bu ekran görüntüsünde gösterildiği gibi üç nokta düğmesine veya Paylaş düğmesine tıklayın:

![macOS'ta Paylaş](/img/share-on-macos.png)

Kurallarınızı çoklu seçebilir ve kural listeleme kenar çubuğunda ikincil tıklayarak (sağ tıklayarak) dışa aktarabilirsiniz:

![macOS'ta birden fazla kuralı paylaşma](/img/multiselect-on-macos.png)

Ek olarak, tüm kuralları menü çubuğundan > Dosya > Tüm Kuralları Dışa Aktar seçeneğinden dışa aktarabilirsiniz:

![Menü Çubuğu > Dosya](/img/menubar-file.png)

> [!NOTE]
> Kurallarınızı [Einar Egilsson'ın Redirector'ı](https://einaregilsson.com/redirector/) ile uyumlu biçimde dışa aktarmak isterseniz, bağlam menüsünde veya menü çubuğunda Option tuşuna (`⌥`) basın, ardından şunları göreceksiniz:
> ![Menü Çubuğu > Dosya (Redirector)](/img/menubar-file-redirector.png)
>
> ![Bağlam Menüsü (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web, Redirector ile tam uyumlu değildir ve bunun tersi de geçerlidir. Yakalama Grubu İşleme için "Oluşumları Değiştir" gibi bazı ayarlar, Redirector'ın formatıyla dışa aktarılmaz.
> :::

## Kuralları İçe Aktarma {#import-rules}

### macOS'ta İçe Aktarma {#import-on-macos}

- Dışa aktarılan kural dosyalarını Redirect Web penceresine sürükleyip bırakmanız yeterlidir.
- Veya, menü çubuğundan > Dosya > Kuralları İçe Aktar seçeneğinden de içe aktarabilirsiniz.

### iOS'ta İçe Aktarma {#import-on-ios}

Kural dosyanızın Dosyalar uygulamasında olduğunu varsayalım. Dosyalar'da açın, Paylaş düğmesine dokunun ve ardından Redirect Web'i seçin:

| Adım 1: Dosyalar'daki araç çubuğunda Paylaş düğmesine dokunun | Adım 2: Sayfada Redirect Web'i seçin |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Araç çubuğundaki paylaş düğmesine dokunun](/img/files-share-ios.png) | ![Sayfadaki uygulama simgesine dokunun](/img/share-sheet-ios.png) |

# Kütüphane

Kullanışlı, kullanıma hazır kuralları ücretsiz bulabileceğiniz kütüphanemize hoş geldiniz! Bu kurallara Redirect Web uygulaması aracılığıyla erişebilirsiniz.

![Library Screenshot](/img/library-screenshot.webp)

Kuralları Kurallarım'a kopyalamak için Get (Al) düğmesini seçmeniz yeterlidir:

![Get Button - iOS](/img/library-get-button-ios.gif)

## Kuralınızı kütüphaneye katkıda bulunun

Mevcut kütüphanede yalnızca birkaç kural bulunmaktadır, bu nedenle katkılarınız için minnettarız!

Kurallarınızı kütüphaneye katkıda bulunmak için 2 seçeneğiniz var:

1.  **Kuralınızı Forumda Paylaşın**: Yönlendirme kuralınızı [forumda](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules) diğer kullanıcılarla paylaşın. Daha sonra kuralınızı kütüphaneye ekleyip ekleyemeyeceğimizi sorabiliriz. Bu, 2. seçeneğe göre daha kolay olabilir, çünkü ekleme işlemini bize bırakabilirsiniz.
2.  **Bize bir çekme isteği gönderin**: Bize bir çekme isteği gönderebilirsiniz. Daha fazla bilgi edinmek için lütfen [talimatları](https://github.com/mshibanami/redirect-web/tree/main/library) inceleyin.

## Diğer Bilgiler

-   Tüm içeriği [GitHub'daki depoda](https://github.com/mshibanami/redirect-web/tree/main/library) görebilirsiniz.
-   Kütüphane içerikleri için özel API sunucumuzu Cloudflare üzerinde barındırıyoruz, bu da filtreleme ve diğer hizmetleri sunmamızı sağlıyor. Uygulama, kütüphane içeriklerini bu sunucudan alıyor. [Gizlilik politikamızda](privacy-policy.mdx) belirtildiği gibi, bu sunucu kişisel bilgilerinizi toplamaz.

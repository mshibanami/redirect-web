## Yönlendirme kuralları çalışmıyor

Kurallarınız beklendiği gibi çalışmıyorsa, lütfen aşağıdakileri kontrol edin:

- **Tarayıcı uzantısı doğru sayfa için yüklü ve etkin:** Redirect Web tarayıcı uzantısını yüklediğinizden ve yönlendirmek istediğiniz web sayfası için tarayıcınızda etkinleştirildiğinden emin olun. Kaynak web sayfasında uzantının açılır penceresini bu şekilde açabiliyorsanız, doğru şekilde yüklenmiştir:
    <Image src={ExtensionPopup} alt="Redirect Web uzantı açılır penceresi" width="260" hasShadow/>

- **Kural yapılandırması doğru:** Redirect Web yerel uygulamasındaki yönlendirme kurallarınızı tekrar kontrol edin. Kaynak URL desenlerinin, hedef URL'lerin ve tüm koşulların doğru şekilde ayarlandığından emin olun. Kuralınızın geçerli olup olmadığını kontrol etmek için, Kural Düzenleme ekranındaki Örnekler bölümüne bir örnek URL girin. Kuralınız geçerliyse, girdinizin altında beklenen hedef URL'yi şöyle göreceksiniz:
    <Image src={RuleEditExampleResult} alt="Örnek URL sonucunu gösteren kural düzenleme ekranı örneği" width="260" hasShadow/>

## Uzantının açılır penceresi bir hata gösteriyor

<Image src={ErrorOverlay} alt="Redirect Web hata katmanı" width="260" hasShadow hasRoundedCorners/>

> Redirect Web uygulaması yüklenmedi veya düzgün çalışmıyor.
> Lütfen uygulamayı App Store'dan edinin.

Uzantı açılır penceresini açarken yukarıdaki hatayı görüyorsanız, bu Redirect Web yerel uygulamasının yüklü olmadığı veya düzgün çalışmadığı anlamına gelir.

Lütfen Redirect Web'in en son sürümünü [App Store](https://apps.apple.com/app/id1571283503)'dan yüklediğinizden emin olun.

Yerel uygulamayı yükledikten sonra tarayıcınızı yeniden başlatmanız da gerekebilir.

> [!NOTE]
> * Tarayıcı uzantısını kullanmak için Redirect Web yerel uygulamasını yüklemeniz gerekir. Uzantı tek başına çalışmaz.
> * Safari kullanıcılarının uzantıyı ayrıca yüklemesine gerek yoktur, çünkü yerel uygulama ile birlikte gelir.

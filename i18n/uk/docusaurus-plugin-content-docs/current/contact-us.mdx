# Зв'яжіться з нами

Ми тут, щоб допомогти, якщо у вас є запитання, відгуки або проблеми з Redirect Web. Ось найкращі способи зв'язатися з нами:

- **[Проблеми GitHub (GitHub Issues)](https://github.com/mshibanami/RedirectWeb/issues)**: Якщо ви зіткнулися з будь-якими помилками або хочете запросити нові функції, відвідайте сторінку GitHub Issues. Ви можете переглядати існуючі проблеми, подавати нові та відстежувати їхній прогрес.
- **[Обговорення GitHub (GitHub Discussions)](https://github.com/mshibanami/RedirectWeb/discussions)**: Якщо у вас є запитання або ви хочете отримати допомогу від розробника чи інших користувачів, відвідайте сторінку GitHub Discussions. Ви можете розпочати нове обговорення, приєднатися до існуючих та отримати допомогу від спільноти.
- **Форма зворотного зв'язку**: Ви можете надіслати нам відгук через форму зворотного зв'язку в застосунку Redirect Web. Це надішле електронний лист безпосередньо розробнику, тому ваш відгук залишиться конфіденційним від інших користувачів.

Дякуємо, що користуєтеся Redirect Web! Ми цінуємо вашу підтримку та відгуки.

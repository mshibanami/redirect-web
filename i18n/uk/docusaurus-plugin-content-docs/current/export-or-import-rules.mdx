# Експорт/Імпорт правил

## Експорт правил {#export-rules}

### Експорт на macOS {#export-on-macos}

Натисніть кнопку з трьома крапками або кнопку «Поділитися», як показано на цьому знімку екрана:

<Image src={ShareOnMacOS} alt="Share button on macOS" width="460" hasShadow hasRoundedCorners/>

Потім виберіть «Експортувати правило» або «Поділитися правилом...» з контекстного меню:

Ви також можете вибрати кілька правил і експортувати їх, клацнувши правою кнопкою миші на бічній панелі списку правил:

<Image src={MultiSelectOnMacOS} alt="Share multiple rules on macOS" height="400" hasShadow hasRoundedCorners/>

Крім того, ви можете експортувати всі правила з рядка меню > Файл > Експортувати всі правила:

<Image src={MenubarFile} alt="Menubar > File" width="400" hasShadow hasRoundedCorners/>

> [!NOTE]
> Якщо ви хочете експортувати свої правила у форматі, сумісному з [Redirector Ейнара Егілссона](https://einaregilsson.com/redirector/), натисніть клавішу Option (`⌥`) у контекстному меню або в рядку меню, тоді ви побачите наступне:
> <Image src={MenubarFileRedirector} alt="Menubar > File (Redirector)" width="400" hasShadow hasRoundedCorners/>
> <br/>
> <Image src={ContextMenuRedirector} alt="Context Menu (Redirector)" width="400" hasShadow hasRoundedCorners/>
>
> :::warning
> Redirect Web не повністю сумісний з Redirector, і навпаки. Деякі налаштування, такі як «Замінити входження» для обробки груп захоплення, не експортуються у форматі Redirector.
> :::

## Імпорт правил {#import-rules}

### Імпорт на macOS {#import-on-macos}

- Просто перетягніть експортовані файли правил у вікно Redirect Web.
- Або ви також можете імпортувати їх з рядка меню > Файл > Імпортувати правила

### Імпорт на iOS {#import-on-ios}

Припустимо, ваш файл правил знаходиться в програмі «Файли». Відкрийте його у «Файлах», натисніть кнопку «Поділитися», а потім виберіть Redirect Web:

| Крок 1: Натисніть кнопку «Поділитися» у «Файлах»                                                                          | Крок 2: Виберіть Redirect Web                                                                              |

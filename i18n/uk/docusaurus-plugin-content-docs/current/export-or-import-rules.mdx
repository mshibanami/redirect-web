# Експорт/Імпорт правил

## Експорт правил {#export-rules}

### Експорт на macOS {#export-on-macos}

Натисніть кнопку з трьома крапками або кнопку «Поділитися», як показано на цьому знімку екрана:

![Поділитися на macOS](/img/share-on-macos.png)

Ви також можете вибрати кілька правил і експортувати їх, натиснувши вторинну кнопку (праву кнопку миші) на бічній панелі списку правил:

![Поділитися кількома правилами на macOS](/img/multiselect-on-macos.png)

Крім того, ви можете експортувати всі правила з рядка меню > Файл > Експортувати всі правила:

![Рядок меню > Файл](/img/menubar-file.png)

> [!NOTE]
> Якщо ви хочете експортувати свої правила у форматі, сумісному з [Redirector Ейнара Егілссона](https://einaregilsson.com/redirector/), натисніть клавішу Option (`⌥`) у контекстному меню або в рядку меню, тоді ви побачите це:
> ![Рядок меню > Файл (Redirector)](/img/menubar-file-redirector.png)
>
> ![Контекстне меню (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web не повністю сумісний з Redirector, і навпаки. Деякі налаштування, такі як "Замінити входження" для обробки груп захоплення, не експортуються у форматі Redirector.
> :::

## Імпорт правил {#import-rules}

### Імпорт на macOS {#import-on-macos}

- Просто перетягніть експортовані файли правил у вікно Redirect Web.
- Або ви також можете імпортувати їх з рядка меню > Файл > Імпортувати правила

### Імпорт на iOS {#import-on-ios}

Припустимо, ваш файл правил знаходиться в програмі "Файли". Відкрийте його у "Файлах", натисніть кнопку "Поділитися", а потім виберіть Redirect Web:

| Крок 1: Натисніть кнопку "Поділитися" на "Файлах" | Крок 2: Виберіть Redirect Web |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Натисніть кнопку "Поділитися" на панелі інструментів](/img/files-share-ios.png) | ![Натисніть піктограму програми на аркуші](/img/share-sheet-ios.png) |

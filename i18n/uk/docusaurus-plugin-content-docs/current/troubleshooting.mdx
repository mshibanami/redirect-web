# Усунення несправностей

На цій сторінці перелічено поширені проблеми та їх вирішення під час використання Redirect Web.

<TOCInline toc={toc} />

## Правила перенаправлення не працюють

Якщо ваші правила не працюють належним чином, перевірте наступне:

- **Розширення браузера встановлено та увімкнено для правильної сторінки:** Переконайтеся, що ви встановили розширення браузера Redirect Web і що воно увімкнено у вашому браузері для веб-сторінки, з якої ви хочете перенаправити. Якщо ви можете відкрити спливаюче вікно розширення, як показано нижче, на вихідній веб-сторінці, воно встановлено правильно:
    <Image src={ExtensionPopup} alt="Спливаюче вікно розширення Redirect Web" width="260" hasShadow/>

- **Конфігурація правила правильна:** Двічі перевірте правила перенаправлення в нативному додатку Redirect Web. Переконайтеся, що шаблони вихідних URL-адрес, цільові URL-адреси та будь-які умови налаштовані правильно. Щоб перевірити, чи є ваше правило дійсним, введіть приклад URL-адреси в розділі "Приклади" на екрані редагування правила. Якщо ваше правило дійсне, ви побачите очікувану URL-адресу призначення під вашим вводом, як показано нижче:
    <Image src={RuleEditExampleResult} alt="Приклад екрана редагування правила, що показує результат прикладу URL-адреси" width="260" hasShadow/>

## Спливаюче вікно розширення показує помилку

<Image src={ErrorOverlay} alt="Накладання помилки Redirect Web" width="260" hasShadow hasRoundedCorners/>

> Додаток Redirect Web не встановлено або він працює неправильно.
> Будь ласка, завантажте додаток з App Store.

Якщо ви бачите вищезазначену помилку під час відкриття спливаючого вікна розширення, це означає, що нативний додаток Redirect Web не встановлено або він працює неправильно.

Будь ласка, переконайтеся, що ви встановили останню версію Redirect Web з [App Store](https://apps.apple.com/app/id1571283503).

Можливо, вам також знадобиться перезапустити браузер після встановлення нативного додатка.

> [!NOTE]
> * Вам потрібно встановити нативний додаток Redirect Web, щоб використовувати розширення браузера. Саме розширення не працює.
> * Користувачам Safari не потрібно встановлювати розширення окремо, оскільки воно входить до комплекту нативного додатка.

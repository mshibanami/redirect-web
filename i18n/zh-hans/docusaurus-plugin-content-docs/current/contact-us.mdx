# 联系我们

如果您对 Redirect Web 有任何疑问、反馈或问题，我们随时乐意为您提供帮助。以下是联系我们的最佳方式：

- **[GitHub Issues](https://github.com/mshibanami/RedirectWeb/issues)**：如果您遇到任何错误或想请求新功能，请访问 GitHub Issues 页面。您可以浏览现有问题、提交新问题并跟踪您问题的进度。
- **[GitHub Discussions](https://github.com/mshibanami/RedirectWeb/discussions)**：如果您有任何疑问或想从开发者或其他用户那里获得帮助，请访问 GitHub Discussions 页面。您可以发起新讨论、加入现有讨论，并从社区获得帮助。
- **反馈表单**：您可以通过 Redirect Web 应用中的反馈表单向我们发送反馈。这会直接向开发者发送电子邮件，因此您的反馈对其他用户是保密的。

感谢您使用 Redirect Web！我们非常感谢您的支持和反馈。

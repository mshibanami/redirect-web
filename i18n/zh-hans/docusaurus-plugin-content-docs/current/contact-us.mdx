# 联系我们

如果您对 Redirect Web 有任何疑问、反馈或遇到问题，我们随时为您提供帮助。以下是联系我们的最佳方式：

- **[GitHub Issues (问题)](https://github.com/mshibanami/RedirectWeb/issues)**: 如果您遇到任何错误或希望请求新功能，请访问 GitHub Issues 页面。您可以在此处浏览现有问题、提交新问题并跟踪其进展。
- **[GitHub Discussions (讨论)](https://github.com/mshibanami/RedirectWeb/discussions)**: 如果您有任何疑问或希望从开发者或其他用户那里获得帮助，请访问 GitHub Discussions 页面。您可以在此处发起新讨论、加入现有讨论，并从社区获得帮助。
- **反馈表单**: 您可以通过 Redirect Web 应用中的反馈表单向我们发送反馈。这将直接向开发者发送电子邮件，因此您的反馈将对其他用户保密。

感谢您使用 Redirect Web！我们感谢您的支持和反馈。

# 导出/导入规则

## 导出规则

### 在 macOS 上导出

点击省略号按钮或分享按钮，如截图所示：

![在 macOS 上分享](/img/share-on-macos.png)

您也可以多选规则，然后通过在规则列表侧边栏上二次点击（右键点击）它们来导出：

![在 macOS 上分享多条规则](/img/multiselect-on-macos.png)

此外，您还可以从菜单栏 > 文件 > 导出所有规则中导出所有规则：

![菜单栏 > 文件](/img/menubar-file.png)

> [!NOTE]
> 如果您想以与 [Einar Egilsson 的 Redirector](https://einaregilsson.com/redirector/) 兼容的格式导出规则，请在上下文菜单或菜单栏上按住 Option 键（`⌥`），然后您将看到以下内容：
> ![菜单栏 > 文件 (Redirector)](/img/menubar-file-redirector.png)
>
> ![上下文菜单 (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web 与 Redirector 并非完全兼容，反之亦然。某些设置，例如用于捕获组处理的“替换出现次数”，不会以 Redirector 的格式导出。
> :::

## 导入规则

### 在 macOS 上导入

- 只需将导出的规则文件拖放到 Redirect Web 的窗口中即可。
- 或者，您也可以从菜单栏 > 文件 > 导入规则中导入它们。

### 在 iOS 上导入

假设您的规则文件在“文件”应用中。在“文件”中打开它，点击分享按钮，然后选择 Redirect Web：

| 步骤 1：在“文件”中点击分享按钮 | 步骤 2：选择 Redirect Web         |
| ---------------------------------- | ---------------------------------- |
| ![Alt text](/img/files-share-ios.png) | ![Alt text](/img/share-sheet-ios.png) |

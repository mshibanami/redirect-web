# 导出/导入规则

## 导出规则 {#export-rules}

### 在 macOS 上导出 {#export-on-macos}

点击省略号按钮或分享按钮，如截图所示：

<Image src={ShareOnMacOS} alt="Share button on macOS" width="460" hasShadow hasRoundedCorners/>

然后，从上下文菜单中选择“导出规则”或“分享规则...”：

您还可以多选规则，然后通过在规则列表侧边栏上二次点击（右键点击）它们来导出：

<Image src={MultiSelectOnMacOS} alt="Share multiple rules on macOS" height="400" hasShadow hasRoundedCorners/>


此外，您可以从菜单栏 > 文件 > 导出所有规则中导出所有规则：

<Image src={MenubarFile} alt="Menubar > File" width="400" hasShadow hasRoundedCorners/>

> [!NOTE]
> 如果您想以与 [Einar Egilsson 的 Redirector](https://einaregilsson.com/redirector/) 兼容的格式导出规则，请在上下文菜单或菜单栏上按住 Option 键 (`⌥`)，然后您将看到：
> <Image src={MenubarFileRedirector} alt="Menubar > File (Redirector)" width="400" hasShadow hasRoundedCorners/>
> <br/>
> <Image src={ContextMenuRedirector} alt="Context Menu (Redirector)" width="400" hasShadow hasRoundedCorners/>
>
> :::warning
> RedirectWeb 与 Redirector 不完全兼容，反之亦然。某些设置，例如捕获组处理的“替换出现次数”，不会以 Redirector 的格式导出。
> :::

## 导入规则 {#import-rules}

### 在 macOS 上导入 {#import-on-macos}

- 只需将导出的规则文件拖放到 RedirectWeb 的窗口中。
- 或者，您也可以从菜单栏 > 文件 > 导入规则中导入它们。

### 在 iOS 上导入 {#import-on-ios}

假设您的规则文件在“文件”应用中。在“文件”中打开它，点击“分享”按钮，然后选择 RedirectWeb：

| 步骤 1：点击“文件”工具栏中的分享按钮                                                                          | 步骤 2：选择 RedirectWeb                                                                              |

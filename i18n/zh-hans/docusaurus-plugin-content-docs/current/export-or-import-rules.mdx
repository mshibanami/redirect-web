# 导出/导入规则

## 导出规则 {#export-rules}

### 在 macOS 上导出 {#export-on-macos}

点击省略号按钮或分享按钮，如下图所示：

![在 macOS 上分享](/img/share-on-macos.png)

您也可以多选规则，然后通过在规则列表侧边栏中进行辅助点击（右键点击）来导出它们：

![在 macOS 上分享多条规则](/img/multiselect-on-macos.png)

此外，您还可以通过菜单栏 > 文件 > 导出所有规则来导出所有规则：

![菜单栏 > 文件](/img/menubar-file.png)

> [!NOTE]
> 如果您想以兼容 [Einar Egilsson 的 Redirector](https://einaregilsson.com/redirector/) 的格式导出规则，请在上下文菜单或菜单栏中按下 Option 键（`⌥`），然后您将看到以下选项：
> ![菜单栏 > 文件 (Redirector)](/img/menubar-file-redirector.png)
>
> ![上下文菜单 (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web 与 Redirector 不完全兼容，反之亦然。某些设置，例如捕获组处理的“替换出现次数”，不会以 Redirector 的格式导出。
> :::

## 导入规则 {#import-rules}

### 在 macOS 上导入 {#import-on-macos}

- 只需将导出的规则文件拖放到 Redirect Web 的窗口中。
- 或者，您也可以通过菜单栏 > 文件 > 导入规则来导入它们。

### 在 iOS 上导入 {#import-on-ios}

假设您的规则文件在“文件”App 中。打开它，点击分享按钮，然后选择 Redirect Web：

| 步骤 1：点击“文件”App 工具栏中的分享按钮                         | 步骤 2：选择 Redirect Web                                       |
| ----------------------------------------------------------------- | --------------------------------------------------------------- |
| ![点击工具栏中的分享按钮](/img/files-share-ios.png) | ![点击面板中的 App 图标](/img/share-sheet-ios.png) |

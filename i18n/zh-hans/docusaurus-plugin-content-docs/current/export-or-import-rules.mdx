# 导出/导入规则

## 导出规则 {#export-rules}

### 在 macOS 上导出 {#export-on-macos}

单击省略号按钮或共享按钮，如下图所示：

![Share on macOS](/img/share-on-macos.png)

您还可以多选规则，然后通过辅助点击（右键单击）规则列表侧边栏中的规则来导出它们：

![Share multiple rules on macOS](/img/multiselect-on-macos.png)

此外，您可以从菜单栏 > 文件 > 导出所有规则中导出所有规则：

![Menubar > File](/img/menubar-file.png)

> [!NOTE]
> 如果您想以与 [Einar Egilsson 的 Redirector](https://einaregilsson.com/redirector/) 兼容的格式导出规则，请在上下文菜单或菜单栏上按 Option 键 (`⌥`)，然后您将看到以下内容：
> ![Menubar > File (Redirector)](/img/menubar-file-redirector.png)
>
> ![Context Menu (Redirector)](/img/context-menu-redirector.png)
>
> :::warning
> Redirect Web 与 Redirector 不完全兼容，反之亦然。某些设置，例如捕获组处理的“替换出现次数”，不会以 Redirector 的格式导出。
> :::

## 导入规则 {#import-rules}

### 在 macOS 上导入 {#import-on-macos}

- 只需将导出的规则文件拖放到 Redirect Web 的窗口中。
- 或者，您也可以从菜单栏 > 文件 > 导入规则中导入它们。

### 在 iOS 上导入 {#import-on-ios}

假设您的规则文件在“文件”应用中。在“文件”中打开它，点击“共享”按钮，然后选择 Redirect Web：

| 步骤 1：在“文件”中点击共享按钮                                     | 步骤 2：选择 Redirect Web                                         |
| ------------------------------------------------------------------------- | ------------------------------------------------------------------- |
| ![Tap the share button in the toolbar](/img/files-share-ios.png) | ![Tap the app icon in the sheet](/img/share-sheet-ios.png) |

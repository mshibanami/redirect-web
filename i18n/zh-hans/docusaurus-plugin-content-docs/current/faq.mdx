# 常见问题

## 为什么我的重定向规则不起作用？ {/* #redirection-not-working */}

如果您创建的规则没有按预期触发，请检查以下内容：

- **扩展是否已启用？** 检查浏览器的扩展设置，确保 RedirectWeb 已开启。
- **规则是否已开启？** 打开 RedirectWeb 应用，确保特定规则的开关处于“开启”状态。
- **规则是否正确？** 在“编辑规则”屏幕的“示例”字段中输入 URL 来测试规则。
- **[Safari] 您是否允许扩展访问源网站？**：确保您已允许扩展访问您想要重定向的网站。如果您可以打开扩展的弹出窗口，说明扩展可以访问源网站。
  > [!TIP]
  > 为方便起见，我们建议允许访问所有网站。（只是个建议！）

授予的权限将仅用于重定向等应用功能，并遵守**[隐私政策](/privacy-policy)**。

## 即使点击了“恢复购买”，我的购买仍未恢复 {/* #purchase-not-restored */}

如果“恢复购买”按钮未按预期工作，请按照以下故障排除步骤操作：

1. 登录 [https://reportaproblem.apple.com/](https://reportaproblem.apple.com/) 以确认交易已成功完成，并已列在您的 Apple ID 历史记录中。
2. 确保当前登录 **App Store 应用** 的 Apple ID 与最初购买时使用的 Apple ID 相同。
   > [!NOTE]
   >
   > 您设备的“设置”应用中登录的账号有时可能与 App Store 应用中使用的账号不同。必须是 App Store 应用中的账号与您的购买历史记录相匹配，恢复功能才能正常运作。
3. 如果您有多个 Apple ID，请退出 App Store 并使用正确的账号重新登录，然后再次尝试“恢复购买”按钮。

如果账号匹配但恢复仍然失败，请[联系我们](/contact-us)并附上您的收据或购买历史记录截图。我们将为您进一步调查此问题。

## 在安装了 Chrome、Firefox 或 Edge 的扩展后，我还需要下载该应用吗？ {/* #app-for-non-safari-browsers */}

是的。无论您使用哪种浏览器，您都必须[从 App Store 下载 RedirectWeb 应用](https://apps.apple.com/app/id1571283503)。您可以从浏览器的扩展商店安装扩展，但需要该应用来通过 iCloud 在设备之间管理您的规则、购买历史记录和其他设置。

## 如果我使用 Pro 版应用，是否需要为 Chrome、Firefox 或 Edge 安装不同的扩展？ {/* #different-extension-for-pro */}

不。无论您在 Mac 上使用标准版 RedirectWeb 应用还是 RedirectWeb Pro，您都应该从浏览器的相应扩展商店安装完全相同的扩展。该扩展将自动检测并与您安装的任何版本的应用进行通信。

## RedirectWeb 是否可以在 Windows、Android、Linux 或任何其他平台上运行？ {/* #non-apple-platforms */}

目前还不支持。RedirectWeb 与 Apple 生态系统紧密结合，包括 iCloud 和通用购买。由此带来的不便，我们深表歉意。

## RedirectWeb 和 RedirectWeb Pro 有什么区别？ {/* #difference-between-standard-and-pro */}

两个版本提供完全相同的功能，并同时接收更新。唯一的区别是付费模式：

- **RedirectWeb**：一款具有基础功能的免费应用（限制为 **1 条激活规则**），提供可选的应用内购买（订阅或一次性付费）以解锁所有功能，包括无限规则和 iCloud 同步。
- **RedirectWeb Pro**：一款预付费的完全解锁应用。此版本专为符合 **Apple 商务管理** 和 **Apple 校园教务管理 (VPP)** 资格而创建，允许组织批量购买应用，而这通过应用内购买是无法实现的。当然，任何喜欢预付费购买的人都可以购买它！

## 一次性付费、年度订阅还是 Pro 版本：哪种最好？ {/* #one-time-or-subscription-or-pro */}

我们建议在标准 RedirectWeb 应用内进行一次性付费购买，或者购买 RedirectWeb Pro，因为它们提供相同的价值。
我们提供订阅的主要原因是允许您通过其免费试用来尝试所有功能。以下是预期的使用场景：

1. 在标准 RedirectWeb 应用中订阅年度订阅。
2. 立即取消订阅：前往“设置”应用，搜索“订阅”，点击此应用，然后选择“取消订阅”。这让您可以享受免费试用而无需任何承诺。
3. 享受 7 天对所有功能的完整访问权。
4. 如果您喜欢这些功能，请选择一次性付费购买或购买 Pro 版本。

当然，如果您愿意，继续年度订阅也没有问题！

## RedirectWeb 是否会在网站中插入广告或其他无关内容？ {/* #does-it-inject-ads */}

不，它不会插入任何此类内容。只有为了提供本工作文档和 App Store 应用页面中描述的功能所必需的代码，才会被注入 Safari 上的网站。

## RedirectWeb 有没有可能被卖给别人？ {/* #could-it-be-sold */}

100% 不会。浏览器扩展对您的浏览和数据拥有深度控制权，将 RedirectWeb 转交给另一个所有者将为侵犯隐私打开大门。我们见过一些扩展在转手后就开始搜集用户数据或在不经警告的情况下插入广告。这里永远不会发生这种事。欲了解更多关于这些风险的信息，请参阅 Matt Frisbie 的文章 [“The Ugly Business of Monetizing Browser”](https://mattfrisbie.substack.com/p/the-ugly-business-of-monetizing-browser) 和 Brian Krebs 的调查 [“Is Your Browser Extension a Botnet Backdoor?”](https://krebsonsecurity.com/2021/03/is-your-browser-extension-a-botnet-backdoor/)。

RedirectWeb 建立在信任和责任的基础上。我们认为保护您的信息安全是我们的职责，并且我们在任何情况下都拒绝转让所有权。RedirectWeb 将永远留在我们手中，因此您可以完全放心地浏览。

欢迎来到 RedirectWeb 的支持网站。RedirectWeb 是一款浏览器扩展，可重定向浏览器中的所有页面加载，确保您始终到达想去的位置。

（原名 _Redirect Web for Safari_ 或简称 _Redirect Web_。）

## ⬇️ 下载链接 {/* #download-links */}

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh) (Chrome 网上应用店)
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/) (Firefox 附加组件)
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga) (Microsoft Edge 附加组件)
- [Safari](https://apps.apple.com/app/id1571283503) (该扩展已随原生应用一同捆绑。)

RedirectWeb 可用于：

- **Mac** (Safari, Chrome, Firefox, Edge 等)
- **iPhone**, **iPad**, 和 **Apple Vision Pro** (Safari)

:::note

- 无论您使用哪个浏览器，都必须从 [Mac App Store 安装 RedirectWeb 应用](https://apps.apple.com/app/id1571283503)，因为扩展需要依赖该应用来访问您的规则、设置等。
- 多浏览器支持仅在 Mac 上可用。

:::

## 常用用例 {/* #popular-use-cases */}

例如，您可以创建以下规则：

- 以经典设计 (old.reddit.com) 查看 **Reddit**。
- 每当出现 **Google 地图** 链接时，使用 **Apple 地图** 打开。
- 通过在每次搜索中添加 **?udm=14**，从 **Google 搜索结果** 中隐藏 Google 的 AI 概览（AI Overviews）。
- 从 Fandom 页面跳转到官方的 **Terraria** 或 **Minecraft** 维基。
- 将 **Kagi**、**Startpage**、**ChatGPT** 及任何其他搜索引擎作为您的 **默认搜索引擎**。
- 在 Google Chrome 中打开 **Google Meet** 链接。（Mac）
- 直接在 Figma 应用中打开 **Figma** 链接。（Mac）
- 直接在 Notion 应用中打开 **Notion** 链接。（Mac）

您可以在应用的“库”部分免费获取所有这些预定义规则！

## 功能 {/* #features */}

RedirectWeb 具备以下功能：

- 使用 **正则表达式** 或 **通配符** 模式类型创建规则
- 使用 **反向引用** 指定动态目标 URL
- **排除** 特定 URL 不进行重定向
- 使用 URL 编码/解码、Base64 编码/解码或替换匹配项来 **处理捕获的文本**

此外，您还可以通过订阅或一次性购买解锁以下功能：

- 启用 **无限** 条规则
- 通过 **iCloud** 在您的其他 Mac、iPhone 和 iPad 之间 **同步规则**

（购买应用也是对我们维护 RedirectWeb 的支持。🙏）

您的购买或订阅通过通用购买（Universal Purchase）在 Mac、iPhone、iPad 和 Apple Vision Pro 之间共享。您的支持有助于持续开发。

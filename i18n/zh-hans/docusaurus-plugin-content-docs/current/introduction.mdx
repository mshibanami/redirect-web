欢迎来到 Redirect Web 的支持网站。Redirect Web 是一款浏览器扩展，可重定向浏览器中的所有页面加载，确保您始终到达想去的位置。

（原名：Redirect Web for Safari。）

## ⬇️ 下载链接 {#download-links}

- [Chrome, Opera, Brave, Vivaldi](https://chromewebstore.google.com/detail/ffglckbhfbfmdkefdmjbhpnffkcmlhdh)（Chrome 网上应用店）
- [Firefox](https://addons.mozilla.org/firefox/addon/redirect-web/)（Firefox 附加组件）
- [Edge](https://microsoftedge.microsoft.com/addons/detail/hmfkakfdccgdpgemaicfgngnmflefpga)（Microsoft Edge 附加组件）
- [Safari](https://apps.apple.com/app/id1571283503)（该扩展已随原生应用一同捆绑）

Redirect Web 可用于：

- **Mac**（支持 Safari、Chrome、Firefox、Edge 等）
- **iPhone**、**iPad** 和 **Apple Vision Pro**（Safari）

:::note

- 无论您使用哪个浏览器，都必须从 [Mac App Store 安装 Redirect Web 应用](https://apps.apple.com/app/id1571283503)，因为扩展需要依赖该应用来访问您的规则、设置等。
- 多浏览器支持仅在 Mac 上可用。

:::

## 常见用例 {#popular-use-cases}

例如，您可以创建这些规则：

- 在经典界面查看 **Reddit**（old.reddit.com）。
- 每当出现 **Google Maps** 链接时，打开 **Apple Maps**。
- 通过在每次搜索中添加 **?udm=14**，在 **Google 搜索结果** 中隐藏 Google 的 AI 概览。
- 从 Fandom 页面跳转到官方 **Terraria** 或 **Minecraft** 维基。
- 将 **Kagi**、**Startpage**、**ChatGPT** 或任何其他搜索引擎设为您的 **默认搜索引擎**。
- 在 Mac 上将 **Google Meet** 链接在 Google Chrome 中打开。
- 在 Mac 上直接在 Figma 应用中打开 **Figma** 链接。
- 在 Mac 上直接在 Notion 应用中打开 **Notion** 链接。

您可以在应用的 Library 部分免费获取所有这些预定义规则！

## 功能 {#features}

Redirect Web 具有以下功能：

- 使用 **正则表达式** 或 **通配符** 模式类型创建规则
- 使用 **反向引用** 指定动态目标 URL
- **排除** URL 不被重定向
- 对捕获的文本进行处理（URL 编码/解码、base64 编码/解码，或替换匹配项）

此外，您可以通过订阅或一次性购买解锁以下功能：

- 启用 **无限** 条规则
- 通过 **iCloud** 在您的其他 Mac、iPhone 和 iPad 之间 **同步规则**

（购买应用也能支持我们维护 Redirect Web。🙏）

您的购买或订阅通过通用购买（Universal Purchase）在 Mac、iPhone、iPad 和 Apple Vision Pro 之间共享。您的支持有助于我们持续开发。

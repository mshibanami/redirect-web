# 库

欢迎来到我们的库，在这里您可以免费找到有用的即用型规则！您可以通过 Redirect Web 应用访问它们。

![Library Screenshot](/img/library-screenshot.webp)

要将规则复制到“我的规则”，您只需选择“获取”按钮：

![Get Button - iOS](/img/library-get-button-ios.gif)

## 将您的规则贡献给库

当前的库只有少量规则，因此我们感谢您的贡献！

有两种选择可以将您的规则贡献给库：

1. **在论坛上发布您的规则**：在[论坛](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules)上与其他用户分享您的重定向规则。然后我们可能会请求是否可以将您的规则也放入库中。这可能比选项 2 更容易，因为您可以将添加工作留给我们。
2. **向我们发送拉取请求**：您可以向我们发送拉取请求。请查看[说明](https://github.com/mshibanami/redirect-web/tree/main/library)以了解更多信息。

## 其他信息

- 您可以在 [GitHub 上的仓库](https://github.com/mshibanami/redirect-web/tree/main/library)中查看所有内容。
- 我们在 Cloudflare 上托管我们的私有 API 服务器，用于库内容，使我们能够提供过滤和其他服务。该应用从该服务器获取库内容。如[我们的隐私政策](/privacy-policy.mdx)所述，此服务器不收集您的个人信息。

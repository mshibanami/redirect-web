# 规则库

欢迎来到我们的规则库，在这里您可以免费找到即用型实用规则！您可以通过 Redirect Web 应用程序访问它们。

![Library Screenshot](/img/library-screenshot.webp)

要将规则复制到“我的规则”，只需选择“获取”按钮即可：

![Get Button - iOS](/img/library-get-button-ios.gif)

## 为规则库贡献您的规则

当前规则库中只有少量规则，因此我们非常感谢您的贡献！

您可以通过以下 2 种方式为规则库贡献您的规则：

1. **在论坛上发布您的规则**：在[论坛](https://github.com/mshibanami/redirect-web/discussions/categories/redirect-rules)上与其他用户分享您的重定向规则。然后我们可能会询问是否可以将您的规则也添加到规则库中。这可能比选项 2 更简单，因为您可以将添加工作留给我们。
2. **向我们发送拉取请求**：您可以向我们发送拉取请求。请查看[说明](https://github.com/mshibanami/redirect-web/tree/main/library)以了解更多信息。

## 其他信息

- 您可以在 [GitHub 上的存储库](https://github.com/mshibanami/redirect-web/tree/main/library)中查看所有内容。
- 我们将私有 API 服务器托管在 Cloudflare 上，用于提供规则库内容，使我们能够提供筛选和其他服务。应用程序从该服务器获取规则库内容。如[我们的隐私政策](/privacy-policy)所述，此服务器不会收集您的个人信息。

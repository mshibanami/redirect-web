# 与其他工具的比较

Redirect Web 有许多功能相似的工具。Redirect Web 并非要取代它们，而是要补充它们。以下是一些比较。

> [!NOTE]
> 这些比较基于 2025 年 10 月的状态。如果您发现任何过时信息或错误，请[联系我们](/contact-us)。谢谢！🙏

## [Redirector](https://github.com/einaregilsson/Redirector) {#redirector}

Redirector 是一个很棒的浏览器重定向扩展。它完全免费且开源，并且也适用于非 Apple 平台。但 Redirect Web 还有一些您可能会喜欢的功能：

* **详细的规则设置：** 支持多个排除模式、多个进程匹配以及仅在 Redirect Web 中可用的字符串替换功能。
* **在原生应用中打开：** 在 Mac 上，您可以选择在哪里打开目标 URL。它直接调用 `open` 命令（而不是深度链接）。
* **[DNR](https://blog.mozilla.org/addons/2023/05/17/declarativenetrequest-available-in-firefox/) 支持：** 提供闪电般的重定向速度。
* **集中式规则管理：** 通过原生应用在一个地方管理所有浏览器的规则。
* **Manifest V3 和 V2 支持：** Redirect Web 在基于 Chromium 的浏览器上支持 V3，在 Firefox 上支持 MV2，允许使用与 Redirector 相同的 API，以实现灵活性和效率。
* **Safari 支持：** Redirect Web 支持 iOS 和 macOS 上的 Safari；Redirector 不支持。

我们非常尊重 Redirector，并提供它也能处理的重定向规则的导入/导出功能，因此您可以灵活地在它们之间切换。有关更多详细信息，请参阅[此处](/export-or-import-rules)。

## 像 [Tampermonkey](https://www.tampermonkey.net/) 或 [Greasemonkey](https://www.greasespot.net/) 这样的用户脚本管理器 {#userscript-managers}

用户脚本管理器非常强大和灵活。您可以用它们做很多事情，包括重定向。但是，与 Redirect Web 相比，它们有一些缺点：

* **复杂性：** 编写和维护脚本可能很复杂且耗时，特别是对于不熟悉编程的用户。Redirect Web 提供了一个用户友好的界面，无需编码即可创建和管理规则。
* **安全性：** 用户脚本管理器可能会引入性能开销，特别是如果脚本复杂或未优化。如果编写不当，它们还可能导致无限重定向循环。Redirect Web 旨在高效且安全。
* **更新的 API 支持：** 某些用户脚本管理器可能无法访问 Declarative Net Request (DNR) API，该 API 可以提供更快、更高效的重定向。Redirect Web 支持 DNR 以获得更好的性能。

## Apple 的 [快捷指令](https://support.apple.com/guide/shortcuts/welcome/ios) {#why-not-shortcuts}

快捷指令是 Apple 在 iOS 和 macOS 上提供的强大自动化工具。您可以创建快捷指令来执行各种任务，包括打开 URL。但是，它需要手动触发，并且并非为无缝网页浏览而设计。

## [OpenIn](https://loshadki.app/openin4/) 或 [Velja](https://sindresorhus.com/velja) {#openin-or-velja}

OpenIn 和 Velja 都是浏览器选择器应用，它们也有浏览器扩展，可让您从浏览器中在另一个应用中打开链接。Redirect Web 也具有浏览器到应用的重定向功能。但是，它们之间存在一些差异：

* **自动重定向：** 它们需要您手动选择其工具栏图标才能在另一个应用中打开链接。Redirect Web 根据您的规则自动重定向 URL，无需额外步骤。这是处理链接的两种不同方法。

* **跨平台支持：** 虽然它们是为 Mac 设计的，但 Redirect Web 支持 Mac 上的所有主流浏览器以及 iPhone、iPad 和 Apple Vision Pro 上的 Safari。

## [StopTheMadness](https://www.stopthemadness.com/) {#stopthemadness}

StopTheMadness 是一个浏览器扩展，提供各种功能来增强网页浏览，包括重定向。但是，与 Redirect Web 相比，它有一些限制：

* **规则灵活性：** StopTheMadness 的规则配置选项有限。Redirect Web 提供更详细的规则设置，包括多个排除模式、捕获组处理等。
* **功能的复杂性：** StopTheMadness 包含许多功能，对于只需要重定向的用户来说可能没有必要。Redirect Web 专注于重定向。
* **价格：** StopTheMadness 售价 12-15 美元，没有免费试用或免费计划。Redirect Web 提供免费计划，包含基本功能，以及包含所有功能的免费试用订阅，以及一杯咖啡价格的一次性购买选项。

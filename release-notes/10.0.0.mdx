## Introducing RedirectWeb Pro

We launched a new app called **[RedirectWeb Pro](https://apps.apple.com/app/id6760209215)**, a separate paid version of RedirectWeb. It offers the same features as the fully unlocked version of RedirectWeb.

RedirectWeb Pro is especially suitable for users who want to purchase it in bulk (Volume Purchase) via Apple Business Manager or Apple School Manager. (Apple doesn't support Volume Purchase for in-app purchases, so we created a separate app.)

## Other Changes

- Minor performance improvements and bug fixes.
- Browser Extension

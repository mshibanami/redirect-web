This version is for RedirectWeb Pro on macOS only.

It includes a fix for the issue where the app bundle name was incorrect. It used to be `RedirectWeb.app` but now it's `RedirectWeb Pro.app`.

This is a follow-up to version 10.2.0 for RedirectWeb Pro on macOS.

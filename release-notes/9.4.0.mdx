![Supported Platforms: iOS](https://img.shields.io/badge/Platforms-iOS-white)

This version is only for iOS, but the changes will eventually be rolled out to Mac as well.

* iOS App
  * **ADDED** Support for MDM settings. (Details: [MDM Settings for Admins](/mdm-settings))
* Browser Extension
  * **ADDED** You can now create a DNR rule that opens an app using deep linking. This is intended to be handled automatically, but if it doesn't work, please explicitly select `Via Intermediate Page` for the Main Frame Redirection option on the Edit Rule screen.
